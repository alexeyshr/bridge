# Козырное эхо

Источник: [[sources/k7-standard-full-system|К7-стандарт — полная система]]

## Когда используется

- с 3 козырями и реальной возможностью убитки

## Сигнал

- старшая → младшая
- этот сигнал приоритетнее Лавинталя

## Что должен понять партнер

- нужно взять первый возможный ход
- после этого искать масть убитки

## Дополнительная договоренность

Козырное эхо может быть и сигналом в конкретную масть:
- мелкая → крупная = младшая масть
- крупная → мелкая = точно 3 козыря + старшая масть
