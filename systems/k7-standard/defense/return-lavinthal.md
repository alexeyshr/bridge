# Возвратный Лавинталь

Источник: [[sources/k7-standard-full-system|К7-стандарт — полная система]]

## Содержание раздела

- Старшая карта = в старшую из оставшихся
- Младшая = в младшую из оставшихся
- В том числе на убитку. Сигнал может быть запретом.
- При ограниченном наборе фосок: поощрение/запрет имеет приоритет над переключением.
- Сигнал запрета можно попытаться дать переключением через четность (московский лавинталь).
