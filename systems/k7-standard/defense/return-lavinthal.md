# Возвратный Лавинталь

Источник: [[sources/k7-standard-full-system|К7-стандарт — полная система]]

## Базовые значения

- старшая карта = в старшую из оставшихся мастей
- младшая = в младшую из оставшихся мастей

## Дополнительные правила

- работает в том числе на убитку
- сигнал может быть запретом
- при ограниченном наборе фосок поощрение / запрет важнее переключения
- запрет иногда можно показать переключением через четность или московский лавинталь

## Практический смысл

Возвратный Лавинталь — это язык возврата, а не только язык переключения.
