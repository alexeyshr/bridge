# Московский Лавинталь (на сносах)

Источник: [[sources/k7-standard-full-system|К7-стандарт — полная система]]

## Значения карт

- младшая карта = ничего или эта масть
- старшая четная фоска = старшая из оставшихся мастей
- старшая нечетная фоска = младшая из оставшихся мастей

## Главный принцип

- младшая карта может не быть сигналом
- старшая карта всегда сигнал

## Практический смысл

Этот метод особенно важен на свободных сносах, когда надо быстро показать переключение, не теряя ясность для партнера.
