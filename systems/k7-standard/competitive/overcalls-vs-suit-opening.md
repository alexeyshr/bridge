# Входы на мастевое открытие оппонента

Источник: [[sources/k7-standard-full-system|К7-стандарт — полная система]]

## Содержание раздела

- 1/2 в мажор = 10-16, 5+ карт
- 2 в минор = 12-16 (если 12-14, то 6 карт или 5-4 с мажором)
- Масть со скачком = до 11, от 6 карт. Блок, как открытие
- 1БК = 15-17, держка в масти открытия
- 2БК (на 2 масть) = 15-18, держка
- Контра = 12-15 без длинных мастей (±4441); 15-17 без держки; 17+ с длинной мастью; 18+ равномер
- Ответы на мастевой вход партнёра
- Фит на 2 уровне = НФ, 7-11
- Фит на 3 уровне = ИНВ, 10-11
- Новая масть на 1 уровне = Ф1, 8+, 4+
- Новая масть на 2 уровне = НФ, 8-11, без фита, 5+
- Новая масть на 3 уровне = ФГ, 12+, 5+
- 1БК = НФ, 8-10, без фита, держка
- 2БК = ИНВ, 11-12, без фита, держка
- Масть оппонентов = ФГ, 12+
- После входа партнёра на минорное/мажорное открытие
- (1 нат)-1М-(пас): 2♣ = Друри (инвит с 3-карт фитом)
- 2БК = ИНВ+ с 4-карт фитом
- (1♥)-2♦-(пас): 2♥ = ИНВ+ с фитом
- 2БК = ИНВ без фита с держкой
- 3♣/♦ = ФГ, 4+
- Ответы на контру партнёра
- Масть без скачка = НФ, 0-9, 4+ (мажоры в приоритете)
- Мажор со скачком = ИНВ, 10-11, 4+
- Минор со скачком = ИНВ, 10-11, 5+
- 1БК = НФ, 6-9, без мажоров, держка
- 2БК = ИНВ, 10-11, без мажоров, держка
- 3БК = 12-15, без мажоров, держка
- Масть оппонентов = ИНВ+, ФГ или инвит с двумя мастями
