# Battle Sheet: 1БК

Короткая боевая шпаргалка по 1БК в K7.

## База

- 1БК = 15-17
- расклады: 4333 / 4432 / 5332 с минором

## Самый короткий выбор

1. Есть 4-картный мажор?
   - да → сначала думаем про Stayman
   - нет → смотрим на трансфер или натуральный маршрут
2. Есть 5-картный мажор?
   - слабая / инвитная рука → обычно трансфер
   - ФГ → часто Stayman + Smolen
3. Нет 4М и длинный минор?
   - 2♠ = трансфер в ♣
   - 2БК = трансфер в ♦

## Ключевые заявки

- 2♣ = Stayman
- 2♦ = трансфер в ♥
- 2♥ = трансфер в ♠
- 2♠ = трансфер в ♣, без 4М
- 2БК = трансфер в ♦, без 4М
- 4♣ = ровно ФГ, мажоры 5-5
- 4♦ / 4♥ = большие трансферы, не про шлем
- 4БК = инвит, 16-17, без 4М

## Что помнить обязательно

- минорные трансферы отрицают 4-картный мажор
- с 5-4 в мажорах на ФГ — Stayman, потом Smolen
- суперприемы возможны после 2♦ с 4-картным фитом
- после силовой контры: слабая рука с мастью → называем масть, 8-9 → реконтра

## Если совсем забыли

- 4М ищем через Stayman / трансферы
- длинный минор без 4М → минорные трансферы
- шлемовый интерес после принятого трансфера часто идет через кюбиды / автосплинтер
