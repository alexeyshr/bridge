# Battle Sheet: мажорные открытия

Короткая боевая шпаргалка по 1♥ и 1♠.

## 1♥

- (11)12-21
- 5+♥
- нет 5♠

## 1♠

- (11)12-21
- 5+♠

## Что помнить первым делом

- 1БК = широкий полуфорсирующий ответ
- простое поднятие = слабое поднятие
- 2БК = инвитное поднятие
- 2/1 = сильная рука, много пространства для описания
- сплинтер = короткость + фит + высокий интерес

## Быстрый алгоритм для открывающего

1. Партнер уже ограничил силу?
2. Есть ли фит?
3. Это инвит к гейму или уже шлемовый интерес?
4. Нужен ли показ второй масти / ценностей / сплинтера?

## Если совсем забыли

- 1БК = не паникуем, это широкий “контейнер”
- простое поднятие = слабость
- 2БК = инвит
- после 2/1 не спешим закрывать пространство без причины
