# Шпаргалка по 1БК

Короткая рабочая страница по открытию 1БК в системе K7.

## База

- 1БК = 15-17
- расклад: 4333 / 4432 / 5332 с минором

## Самый короткий маршрут выбора

### Есть 4-картный мажор?

- Да → сначала думаем про Stayman
- Нет → смотрим, не лучше ли трансфер в 5-картную масть или минорный трансфер

### Есть 5-картный мажор?

- Да, слабая / инвитная рука → обычно трансфер
- Да, ФГ → часто Stayman + Смолен

## Ключевые ответы

- 2♣ = Stayman
- 2♦ = трансфер в ♥
- 2♥ = трансфер в ♠
- 2♠ = трансфер в ♣, без 4М
- 2БК = трансфер в ♦, без 4М
- 3БК = натурально, без 4М
- 4♣ = ровно ФГ, мажоры 5-5
- 4БК = инвит, 16-17, без 4М

## Что помнить обязательно

- минорные трансферы отрицают 4-картный мажор
- с 5-4 в мажорах на ФГ используем Смолен
- большой трансфер 4♦/4♥ не про шлем, а просто про игру в 4♥/4♠
- после силовой контры: слабая рука с мастью — называем масть, 8-9 — реконтра
