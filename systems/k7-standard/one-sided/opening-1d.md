# Открытие 1♦

Источник: [[sources/k7-standard-full-system|К7-стандарт — полная система]]

## Смысл открытия

- 12-21 HCP
- 4+♦
- при 6♦ и 4♣ возможно открытие 1♦
- отрицает 5-картный мажор

Это натуральное минорное открытие, где в системе K7 важна преференция силы на 1♦. После него торговля обычно строится вокруг:
- поиска мажорного фита
- уточнения длины бубны
- показов второй масти
- инвертированных миноров
- сильных реверсов и сплинтерных продолжений

## Первые ответы

### Натуральные и базовые ответы

- 1♥ = Ф1, 6+ очков, 4+♥, нет 5♠. С 4-4 в мажорах начинаем с 1♥.
- 1♠ = Ф1, 6+ очков, 4+♠. С 5-5 в мажорах начинаем с 1♠.
- 1БК = НФ, 6-10 очков, нет 4-картного мажора.
- 2♦ = Ф1, 10+, 4+♦, нет 4-картного мажора.
- 2БК = инвит, 11-12, нет 4-картного мажора.
- 3♦ = блок, 5-8(9), 6+♦.
- 3БК = НФ, 13-16, нет 4-картного мажора.

### Сильные ответы

- 2♣ = ФГ, 12+, 5+♣, нет 4-картного мажора.
- 2♥ = ФГ, 12+, хорошая 6+♥.
- 2♠ = ФГ, 12+, хорошая 6+♠.
- 3♣ = инвит к 3БК, 9-11, хорошая 6♣.

## После 1♦ – 1♥ – ?

### Базовые продолжения

- 1♠ = НФ, 11-17, 4♠, нет 4♥ → дальше возможен 2WCB
- 1БК = НФ, 11-14, нет 4♥/♠ → дальше возможен 2WCB
- 2♣ = НФ, 11-17, 4+♣ / 5+♦
- 2♦ = НФ, 11-14, 6+♦ → реле с 6-картом
- 2♥ = НФ, 11-14, 4♥
- 2БК = НФ, 18-19, равномер, нет 4♥

### Сильные продолжения

- 2♠ = ФГ, 18+, 4♠ и 5+♦, реверс без выхода
- 3♣ = ФГ, 18+, 4+♣ / 5+♦
- 3♦ = инвит, 15-17, 6♦+
- 3♥ = инвит, 15-17, 4♥
- 3♠ = ФГ, сплинтер, краткость ♠, 4♥
- 3БК = НФ, 18+, 6+♦ → новые масти ФГ, кюбиды на ♦, 4♥ = играть
- 4♣ = ФГ, 18-21, сплинтер, краткость ♣, 4♥

## После 1♦ – 1♠ – ?

### Базовые продолжения

- 1БК = НФ, 11-14, нет 4♠ → дальше возможен 2WCB
- 2♣ = НФ, 11-17, 4+♣ / 5+♦
- 2♦ = НФ, 11-14, 6+♦ → реле с 6-картом
- 2♠ = НФ, 11-14, 4♠
- 2БК = НФ, 18-19, равномер, нет 4♠

### Сильные продолжения

- 2♥ = Ф1, реверс 16+, 4♥ и 5+♦ → выход через 2БК, затем 3♣
- 3♣ = ФГ, 18+, 4+♣ / 5+♦
- 3♦ = инвит, 15-17, 6♦+
- 3♠ = инвит, 15-17, 4♠
- 3БК = НФ, 18+, 6+♦ → новые масти ФГ, кюбиды на ♦, 4♠ = играть
- 4♣ / 4♥ = ФГ, 18-21, сплинтер, краткость, 4♠

## После 1♦ – 1БК – ?

- 2♣ = НФ, 11-17, 4+♣ / 5+♦
- 2♦ = НФ, 11-14, 6+♦
- 2♥ = Ф1, реверс 16+, 4♥ и 5+♦
- 2♠ = Ф1, реверс 16+, 4♠ и 5+♦
- 2БК = НФ, 18-19
- 3♣ = ФГ, 18+, 4+♣ и 5+♦
- 3♦ = НФ, 15-17, 6+♦
- 3♥ / 3♠ = ФГ, 17+, 6+♦ и краткость, автосплинтер
- 3БК = НФ, 18+, 6+♦ → новые масти ФГ, кюбиды на ♦

## Инвертированные миноры

### После 1♦ – 2♦

- 2♥ / 2♠ = ценности, нет держки во втором мажоре
- на это 2БК = держка и минимум
- 2БК = держки в обоих мажорах, не минимум
- 3♦ = НФ, минимум или нет держек в мажорах

### После 1♦ – 2БК

- 3♣ = НФ, 12-13, 4+♣ / 4+♦, не хочу играть БК
- 3♦ = НФ, 12-13, 6+♦, не хочу играть БК
- 3♥ = ФГ, 17+, 4♥ и 5+♦, реверс; или держка при синглете
- 3♠ = ФГ, 17+, 4♠ и 5+♦, реверс; или держка при синглете

## Как этим пользоваться за столом

### Быстрые ориентиры

- после 1♦ система часто сначала ищет мажорный фит
- 2♣ после ответа в мажор — очень широкий натуральный показ второй масти / длины
- повтор бубны на низком уровне обычно ограничивает руку
- реверсы быстро отделяют сильные руки от минимальных
- прыжки после ответа в мажор обычно уже несут инвит / ФГ / краткость

### На что смотреть в первую очередь

1. Нашли ли мы мажорный фит?
2. Открывающий минимум или уже показал дополнительную силу?
3. Это просто повтор ♦ или уже реверс?
4. Есть ли вторая масть и насколько она натуральна?
5. Есть ли минорный фит, который переводит аукцион в ценностный режим?
