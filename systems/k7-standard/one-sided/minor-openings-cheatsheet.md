# Шпаргалка по минорным открытиям

Короткая рабочая страница по открытиям 1♣ и 1♦ в системе K7.

## 1♣

- 12-21
- 2+♣
- без 5-картного мажора
- часто длинный описательный аукцион

## 1♦

- 12-21
- 4+♦
- без 5-картного мажора
- при 6♦ и 4♣ возможно все равно 1♦

## Что означают базовые ответы

### После 1♣

- 1♦ = Ф1, 4+♦
- 1♥ / 1♠ = Ф1, мажорные ответы
- 1БК = НФ, без 4М
- 2♣ = Ф1, 10+, фит треф
- 2БК = инвит без 4М

### После 1♦

- 1♥ / 1♠ = Ф1, поиск мажорного фита
- 1БК = НФ, без 4М
- 2♦ = Ф1, 10+, фит бубен
- 2БК = инвит без 4М
- 2♣ = сильный ФГ ответ с трефой

## Что помнить обязательно

- после минорного открытия сначала часто ищем мажорный фит
- повтор миноры на низком уровне часто ограничивает руку
- реверс = сильная рука
- инвертированные миноры — важный источник точного ценностного аукциона
- прыжки после ответа в мажор часто уже показывают инвит / ФГ / краткость
