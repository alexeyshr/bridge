# Открытие 1♣

Источник: [[sources/k7-standard-full-system|К7-стандарт — полная система]]

## Смысл открытия

- 12-21 HCP
- 2+♣
- отрицает 4♦, если трефа не хуже хорошей шестерки
- отрицает 5-картный мажор

Это младшее натуральное открытие для рук без 5-картного мажора. В системе K7 после 1♣ важно быстро различать:
- слабые нефорсирующие руки
- форсинг-1 ответы
- инвитные руки
- форсинг-гейм последовательности
- минорные фитовые конструкции

## Первые ответы

### Натуральные и базовые ответы

- 1♦ = Ф1, от 5 очков, 4+♦. Если ровно 4♦, то нет 4♥/♠. С ФГ и 5+♦ + 4М начинаем с 1♦.
- 1♥ = Ф1, 6+ очков, 4+♥, нет 5♠. С 4-4 в мажорах начинаем с 1♥.
- 1♠ = Ф1, 6+ очков, 4+♠. С 5-5 в мажорах начинаем с 1♠.
- 1БК = НФ, 6-10 очков, нет 4-картного мажора.
- 2♣ = Ф1, 10+, 4+♣, нет 4-картного мажора.
- 2БК = инвит, 11-12, нет 4-картного мажора.
- 3♣ = блок, 5-8(9), 6♣.
- 3БК = НФ, 13-16, нет 4-картного мажора.

### Сильные мастевые ответы

- 2♦ = ФГ, 12+, хорошая 6+♦, минимум две фигуры, без четверок сбоку.
- 2♥ = ФГ, 12+, хорошая 6+♥, 2-3 фигуры, готовность играть на синглет.
- 2♠ = ФГ, 12+, хорошая 6+♠, 2-3 фигуры, готовность играть на синглет.
- 3♦ = инвит к 3БК, 9-11, хорошая 6♦.

Практический принцип для инвитов на 3 в минор:
- если у партнера есть фигура в нашей масти, мы обещаем 6 взяток
- если у него есть быстрые взятки сбоку, малоочковые 3БК часто становятся хорошим контрактом
- любая заявка после 3♦ уже форсирует к гейму

## После 1♣ – 1♥ – ?

### Базовые продолжения

- 1♠ = НФ, 11-17, 4♠, нет 4♥ → дальше возможен 2WCB
- 1БК = НФ, 11-14, нет 4♥/♠ → дальше возможен 2WCB
- 2♣ = НФ, 11-14, 6+♣ → реле с 6-картом
- 2♦ = Ф1, реверс 16+, 4♦ и 5+♣ → выход через 2БК, затем 3♣
- 2♥ = НФ, 11-14, 4♥
- 2БК = НФ, 18-19, равномер, нет 4♥; дальше новые масти натуральны и ФГ

### Сильные продолжения

- 2♠ = ФГ, 18+, 4♠ и 5+♣, реверс+ без удобного выхода
- 3♣ = инвит, 15-17, 6+♣; дальше натурально, через ценности, ФГ
- 3♦ = ФГ, 18+, 4♥, нет краткости → 3БК / 4♥ негатив, иначе кюбиды
- 3♥ = инвит, 15-17, 4♥
- 3♠ = ФГ, сплинтер, краткость ♠, 4♥
- 3БК = НФ, 18+, 6+♣ → новые масти ФГ, кюбиды на ♣, 4♥ = играть
- 4♦ = ФГ, 18-21, сплинтер, краткость ♦, 4♥

## После 1♣ – 1♠ – ?

### Базовые продолжения

- 1БК = НФ, 11-14, нет 4♠, возможен даже с синглетом → дальше 2WCB
- 2♣ = НФ, 11-14, 6+♣ → реле с 6-картом
- 2♦ = Ф1, реверс 16+, 4♦ и 5+♣ → выход через 2БК, затем 3♣
- 2♥ = Ф1, реверс 16+, 4♥ и 5+♣ → выход через 2БК, затем 3♣
- 2♠ = НФ, 11-14, 4♠
- 2БК = НФ, 18-19, равномер, нет 4♠

### Сильные продолжения

- 3♣ = инвит, 15-17, 6+♣
- 3♦ = ФГ, 18+, 4♠, нет краткости → 3БК / 4♠ негатив, иначе кюбиды
- 3♠ = инвит, 15-17, 4♠
- 3БК = НФ, 18+, 6+♣ → новые масти ФГ, кюбиды на ♣, 4♠ = играть
- 4♦ / 4♥ = ФГ, 18-21, сплинтер, краткость, 4♠

## После 1♣ – 1БК – ?

- 2♣ = НФ, 11-14, 6+♣
- 2♦ = Ф1, реверс 16+, 4♦ и 5+♣ → выход через 2БК, затем 3♣
- 2♥ = Ф1, реверс 16+, 4♥ и 5+♣ → выход через 2БК, затем 3♣
- 2♠ = Ф1, реверс 16+, 4♠ и 5+♣ → выход через 2БК, затем 3♣
- 2БК = НФ, 18-19
- 3♣ = НФ, 15-17, 6+♣; дальше натурально, через ценности, ФГ
- 3♦ / 3♥ / 3♠ = ФГ, 17+, 6+♣ и краткость, автосплинтер
- 3БК = НФ, 18+, 6+♣ → новые масти ФГ, кюбиды на ♣

## Инвертированные миноры

### После 1♣ – 2♣

- 2♦ / 2♥ / 2♠ = ценности, нет держки во втором мажоре
- на это 2БК = держка, но минимум
- 2БК = держки в обоих мажорах, не минимум

После 2БК:
- 3♣ = единственная слабая заявка, НФ, 10-11, инвит
- с настоящей силой ФГ (14+) открывающий продолжает описание руки

### После 1♣ – 2БК

- 3♣ = НФ, 12-13, 6+♣, не хочу играть БК
- 3♦ = ФГ, 17+, 4♦ и 5+♣, реверс
- 3♥ = ФГ, 17+, 4♥ и 5+♣, реверс; или держка при синглете в другом мажоре
- 3♠ = ФГ, 17+, 4♠ и 5+♣, реверс; или держка при синглете в другом мажоре

## Как этим пользоваться за столом

### Быстрые ориентиры

- 1♣ часто запускает длинные описательные аукционы
- 1♦ / 1♥ / 1♠ от отвечающего — это прежде всего про форму и форсинг-1
- 1БК и простые повторения трефы часто ограничивают руку
- реверс сразу показывает силу и второй раскладный источник
- прыжки после мажорного ответа обычно уже про инвит / ФГ / краткость

### На что смотреть в первую очередь

1. Есть ли у отвечающего мажор?
2. Торговля уже ограничена или еще Ф1 / ФГ?
3. Повтор трефы — минимум или сильная длинная масть?
4. Есть ли реверс или сплинтер?
5. Есть ли минорный фит, который превращает аукцион в ценностный?
