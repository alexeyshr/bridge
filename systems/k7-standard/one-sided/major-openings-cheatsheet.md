# Шпаргалка по мажорным открытиям

Короткая страница для быстрого ориентирования после 1♥ и 1♠.

## Открытие 1♥

- (11)12-21
- 5+♥
- нет 5♠

## Открытие 1♠

- (11)12-21
- 5+♠

## Что означают основные ответы

### Общая логика

- 1БК = полуфорсирующий широкий ответ
- простое поднятие в мажор = слабое поднятие
- 2БК = инвитное поднятие
- 2/1 = сильная рука, торговля не умирает быстро
- сплинтеры = короткость + фит + интерес к высокому контракту

### После 1♥

- 1♠ = Ф1, 4+♠
- 2♥ = слабое поднятие
- 2БК = инвит с фитом
- 2♣ / 2♦ = ФГ
- 2♠ = ФГ, 6+♠

### После 1♠

- 2♠ = слабое поднятие
- 2БК = инвит с фитом
- 2♣ / 2♦ / 2♥ = ФГ

## Что спрашивать себя открывающему

1. Партнер уже ограничил силу?
2. Есть ли фит?
3. Это просто инвит к гейму или уже шлемовый интерес?
4. Нужна ли мне заявка, которая показывает вторую масть или сплинтер?
