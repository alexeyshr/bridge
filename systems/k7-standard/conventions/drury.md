# Друри (после открытия на 3-4 руке)

Источник: [[sources/k7-standard-full-system|К7-стандарт — полная система]]

## Содержание раздела

- После пас-(пас)-1♥/♠ играем конвенцию Друри.
- После 1♥ на 3-4 руке – 2♣ Друри
- 2♣ = Ф1, ИНВ, 4+♥
- 2♥ = нет гейма, пасуй
- 2♦ = Ф1, ИНВ, 4+♦. Если принимаем и есть 4♦ и 3♠, говорим 3♠
- 2БК = ИНВ, 13-14, 5332
- 3♣ = Ф1, ИНВ, 4+♣
- 3♥ = ИНВ, 6+♥
- 3♠ = ФГ, Сплинтер, краткость ♠
- 3БК = НФ, на выбор, 5332
- 4♣ = ФГ, Сплинтер, краткость ♣
- После 1♠ на 3-4 руке – 2♣ Друри
- 2♣ = Ф1, ИНВ, 4+♣
- 2♦ = Ф1, ИНВ, 4+♦. ВНИМАНИЕ: если принимаем и есть 4♥ и 3♠, говорим 3♠
- 2♠ = нет гейма, пасуй
- 2БК = ИНВ, 13-14, 5332
- 3♣ = Ф1, ИНВ, 4+♣
- 3♦ = ФГ, 55♥♦
- 3♠ = ИНВ, 6+♠
- 3БК = НФ, на выбор, 5332
- 4♥ = ФГ, Сплинтер, краткость ♥
