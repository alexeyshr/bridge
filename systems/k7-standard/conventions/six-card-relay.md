# Реле с 6-картом

Источник: [[sources/k7-standard-full-system|К7-стандарт — полная система]]

## Содержание раздела

- Работает в позициях: 1♦-1♠-2♦, 1♦-1♥-2♦, 1♣-1♥-2♣, 1♣-1♠-2♣, 1♥-1♠-2♥
- Пример: 1♣ – 1♥ – 2♣ – ?
- 2♦ = Ф1, квази-натурально, не хочу занимать БК
- 2♥ = к игре
- 2♠ = Ф1, квази-натурально
- 2БК = ФГ, Реле
- 3♣ = 3 карты в ♥
- На 2БК Реле, ответы открывающего:
- 3♦ = фит в ♣, 9СМ♥ (→ 3БК = 6322, 3♠ = краткости)
- 3♥ = фит в ♥, 9СМ♥ (→ 3БК = 6322, 3♠/4♠ = краткости)
- 3♠ = синглет в ♦ (7+♣). 3БК контракт, другие масти – козырь
- 3БК = нет синглетов
- Остальные синглеты аналогично. На любой синглет: 3БК = контракт, другие масти = ищем козырной контракт.
- 3♣ от отвечающего = инвит с трефой
- 3♥ от отвечающего = инвит с червой
