# Ответы после мастевого входа на наше открытие

Источник: [[sources/k7-standard-full-system|К7-стандарт — полная система]]

## Базовые ответы

- пас = менее 8 очков
- фит мажора на 2 уровне = НФ, 7-9
- новая масть на 1 уровне = Ф1, 8+, 4+
- новая масть на 2 без скачка = НФ, 7-11, 5+
- новая масть на 2 со скачком = НФ, 5-7, 6+ в мажоре
- новая масть на 3 без скачка = ФГ, 12+, 5+
- минор на 3 со скачком = инвит к 3БК, 6+ карт
- 1БК = НФ, 8-9, без фита, с держкой
- 2БК на минор = инвит, 10-12, с держкой
- 2БК на мажор = инвит, 10-11, фит 3-4
- масть оппонента = ФГ, 12+, фит
- контра = 8+, две оставшиеся масти 4-4

## Практический смысл

После интервенции главное — быстро разделить слабые руки, фитовые руки, инвиты и ФГ.
