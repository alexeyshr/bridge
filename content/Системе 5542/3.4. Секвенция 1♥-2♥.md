> [!info]+
> NF, 7-10hcp, 3+♥

## Примеры рук отвечающего

```
♠Кхх ♥ДВх ♦хххх ♣Вхх (8 очков, 3♥)
♠хх ♥Кххх ♦Дхх ♣Вххх (7 очков, 4♥)
♠Дх ♥Вххх ♦Кхх ♣хххх (9 очков, 4♥)
♠хх ♥Тххх ♦ДВхх ♣Кхх (10 очков, 4♥ - верхняя граница)
```

## Заявки открывающего (1♥-2♥-?)

### Пас = 11-14hcp, минимум без дополнительных ценностей

```
Рука открывающего: ♠хх ♥КДххх ♦Тхх ♣Вхх (11 очков)
```

**К игре:** 2♥

### 2♠ = F1, 16+hcp, 5♥4♠ (реверс)

```
Рука открывающего: ♠ТКхх ♥ТДххх ♦Кх ♣Дх (17 очков)
```

**Развитие отвечающего:**

- **2NT** = слабая рука (7-8hcp), релей
- **3♣♦** = натурально, задержка (поиск 3NT)
- **3♥** = минимум с фитом 3♥
- **3♠** = фит 4♠ (редко, только с максимумом)
- **3NT** = G (с максимумом и держками)
- **4♥** = G, фит 4♥

### 2NT = F1, 16+hcp, поиск лучшего контракта

```
Рука открывающего: ♠Кхх ♥ТКДхх ♦ТДх ♣Кх (18 очков)
```

**Цель:** Изучить руку отвечающего для выбора между 3NT, 4♥ или даже шлемом.

**Развитие отвечающего:**

- **3♣♦** = натурально, краткость в масти или задержка
- **3♥** = минимум, 3♥, 7-8hcp
- **3♠** = краткость в ♠ с фитом 4♥
- **3NT** = G (максимум 9-10hcp с держками)
- **4♥** = G, 4♥, не максимум

### 3♣ = F1, 16+hcp, 5♥4+♣

```
Рука открывающего: ♠хх ♥ТКххх ♦Кх ♣ТДхх (16 очков)
```

**Развитие отвечающего:**

- **3♦** = задержка (поиск 3NT)
- **3♥** = минимум с фитом 3♥
- **3♠** = краткость в ♠ с фитом ♥
- **3NT** = G (с держками)
- **4♣** = фит в ♣ (редко)
- **4♦** = кюбид (согласован ♥)
- **4♥** = G, фит 4♥

### 3♦ = F1, 16+hcp, 5♥5+♦

```
Рука открывающего: ♠хх ♥ТКххх ♦ТДВхх ♣Кх (16 очков)
```

**Развитие:**

- Аналогично 3♣

### 3♥ = INV, 15-17hcp, 6+♥

```
Рука открывающего: ♠хх ♥ТКДВхх ♦Тхх ♣Кх (16 очков)
```

**Развитие отвечающего:**

- **Пас** = слабая рука (7hcp)
- **3♠** = кюбид, максимум (9-10hcp)
- **4♣♦** = кюбид, максимум
- **4♥** = G, принимаем инвит

### 3♠ = F1, 16+hcp, краткость ♠, 5♥4+m

```
Рука открывающего: ♠х ♥ТКххх ♦ТДхх ♣Кхх (15 очков, хорошая рука)
```

**Развитие отвечающего:**

- **3NT** = G, без шлемовых амбиций
- **4♣♦** = кюбид (согласован ♥)
- **4♥** = G, минимум

### 4♣/4♦ = GF, 18+hcp, Сплинтер, краткость ♣/♦, 6+♥

```
Рука открывающего (4♣): ♠Кх ♥ТКДххх ♦Тхх ♣х (15 очков)
```

**Развитие отвечающего:**

- **4♥** = негатив, нет шлемовых амбиций
- **Кюбиды** = хорошая рука, идем в шлем
- **4NT** = Блэквуд

### 4♥ = G, 15-17hcp, хорошая рука

```
Рука открывающего: ♠Кх ♥ТКДххх ♦Тхх ♣Кх (17 очков)
```

**К игре:** 4♥

### 4NT = Блэквуд, вопрос о тузах

```
Рука открывающего: ♠Кх ♥ТКДВхх ♦ТКх ♣Кх (21 очко, интерес к шлему)
```

## Примеры торговли

**Пример 1: Пас на минимуме**

```
Север: ♠хх ♥КДххх ♦Тхх ♣Вхх (11 очков)
Юг: ♠Кхх ♥Вхх ♦Дхх ♣Кххх (9 очков)

Торговля:
Север: 1♥
Юг: 2♥ (7-10hcp, 3♥)
Север: Пас (минимум)
```

**Пример 2: Инвит принят**

```
Север: ♠хх ♥ТКДВхх ♦Тхх ♣Кх (16 очков)
Юг: ♠Кхх ♥Дххх ♦Вхх ♣Дхх (10 очков, максимум)

Торговля:
Север: 1♥
Юг: 2♥ (7-10hcp, 4♥)
Север: 3♥ (инвит, 15-17hcp, 6♥)
Юг: 4♥ (принимаем с максимумом 10hcp)
```

**Пример 3: Реверс в ♠**

```
Север: ♠ТКхх ♥ТДххх ♦Кх ♣Дх (17 очков)
Юг: ♠Дхх ♥Кхх ♦Тхх ♣Вххх (9 очков)

Торговля:
Север: 1♥
Юг: 2♥ (7-10hcp, 3♥)
Север: 2♠ (реверс, 16+hcp, 4♠)
Юг: 3♠ (фит в ♠, максимум)
Север: 4♠ (гейм)
```

**Пример 4: Сплинтер**

```
Север: ♠Кх ♥ТКДххх ♦Тхх ♣х (15 очков, синглет ♣)
Юг: ♠Дхх ♥Дххх ♦Кхх ♣Вхх (10 очков)

Торговля:
Север: 1♥
Юг: 2♥ (7-10hcp, 4♥)
Север: 4♣ (сплинтер, краткость ♣)
Юг: 4♦ (кюбид, хорошая рука с контролем)
Север: 4♥ (нет всех контролей для шлема)
```

**Пример 5: Поиск 3NT**

```
Север: ♠Кхх ♥ТКДхх ♦ТДх ♣Кх (18 очков)
Юг: ♠Дхх ♥Вхх ♦Кхх ♣Дххх (9 очков, 3♥)

Торговля:
Север: 1♥
Юг: 2♥ (7-10hcp, 3♥)
Север: 2NT (16+hcp, поиск контракта)
Юг: 3NT (хороший равномер, держки везде)
```
