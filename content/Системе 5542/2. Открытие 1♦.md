> [!info]+
> 12-21hcp, 4+♦ (<font color="#9bbb59">м.б 5♣ при силе 11-15 hcp</font>)

> [!tip]- Пограничные руки 10-11 очков?
> Не уверен, открываться ли с 10-11 очками?  
> → См. [[Пограничные руки 10-11 очков]] (подробный разбор по зональности и позиции)

## Первые ответы
> [!important]- 5=hcp (пас или заявка?)
> - Если ровно 5 очков и есть Т в масти, в которой хотим сделать заявку - <font color="#9bbb59">делаем заявку</font>
> - Если раскладные руки 55+ или 64+ и очки в мастях и масти плотные - <font color="#9bbb59">делаем заявку</font>
> - <font color="#9bbb59">допустимо</font>: ДВxxxx Д10xx xx x

- [[2.1. Секвенция 1♦-1♥|1♥]] = F1, 6+hcp, 4+♥ (<font color="#d83931">нет 5+♠</font>, если 44М - говорим 1♥)
- [[2.2. Секвенция 1♦-1♠|1♠]] = F1, 6+hcp, 4+♠ (<font color="#9bbb59">м.б 5+♥</font>, если 55М - говорим 1♠)
- [[2.3. Секвенция 1♦-1NT|1NT]] = NF, 6-10hcp, BAL, <font color="#d83931">нет 4М</font> (<font color="#9bbb59">м.б 4М333</font>)
- [[2.4. Секвенция 1♦-2♣|2♣]] = GF, 12+hcp, 5+♣, <font color="#d83931">нет 4М</font>
- [[2.5. Секвенция 1♦-2♦|2♦]] = F1, 10+hcp, 4+♦, <font color="#d83931">нет 4М</font> (инвертированные миноры)
- 2♥! = GF, 12+hcp, 6+♥ (2-3 фигуры минимум), готов играть на синглет
- 2♠! = GF, 12+hcp, 6+♠ (2-3 фигуры минимум), готов играть на синглет
- 2NT = INV, 11-12hcp, BAL, <font color="#d83931">нет 4М</font> (<font color="#9bbb59">м.б 4М333</font>)
- 3♣ = INV к 3NT, 9-11hcp, 6♣ (6 взяток при второй фигуре)
- 3♦ = BLOCK, 5-9hcp, 6+♦
- 3NT = G, 13-16hcp, BAL, <font color="#d83931">нет 4М</font>

## Примеры рук для первых ответов

**1♥ (F1, 6+hcp, 4+♥):**
```
♠Кхх ♥ДВ10х ♦хх ♣Кхх (9 очков, 4♥)
♠хх ♥ТКхх ♦Вхх ♣хххх (8 очков, 4♥, не 5♠)
```

**1♠ (F1, 6+hcp, 4+♠):**
```
♠ДВ10х ♥Кхх ♦хх ♣Кхх (9 очков, 4♠)
♠ТКхх ♥ДВххх ♦х ♣ххх (9 очков, 55М - говорим 1♠)
```

**1NT (NF, 6-10hcp, без 4М):**
```
♠Кхх ♥Дхх ♦хх ♣Вхххх (7 очков, нет 4М)
♠Вхх ♥Кхх ♦Дхх ♣хххх (6 очков, 4333 с ♥)
```

**2♣ (GF, 12+hcp, 5+♣):**
```
♠Кх ♥Тхх ♦Дх ♣ДВххxx (13 очков, 6♣, нет 4М)
♠Дх ♥Кхх ♦Вх ♣ТКДхх (14 очков, 5♣, нет 4М)
```

**2♦ (F1, 10+hcp, 4+♦, инвертированные):**
```
♠Кхх ♥Дхх ♦ТВхх ♣хх (10 очков, 4♦, нет 4М)
♠Вх ♥Кхх ♦ДВххх ♣Дхх (11 очков, 5♦, нет 4М)
```

**2NT (INV, 11-12hcp, без 4М):**
```
♠Кхх ♥Дхх ♦Вхх ♣ТДх (12 очков, равномер)
♠Дхх ♥Кхх ♦Тхх ♣Вхх (11 очков, равномер)
```

> [!note]- Инвертированные миноры
> После 1♦-2♦ (инвертированные миноры) открывающий показывает:
> - **2♥/2♠** = Ценности, нет держки во втором мажоре
> - **2NT** = Держки в обоих мажорах, не минимум
> - **3♦** = НФ, минимум или нет держек в мажорах
> - **3m** = Фит в миноре партнера, хорошая рука

## Изменения при интервенции. Трансферы

> [!tip]- После 1♦-(контра)
> Начиная с 1NT до 2♠ играем трансферы:
> - **Пас** = натурально, негатив
> - **Реконтра** = 8+ равномерная рука
> - **1♥** = 4+♠ (трансфер)
> - **1♠** = трансфер БК/♦ (нет мажорных 4)
> - **1NT** = натурально
> - **2♣** = трансфер ♥, 7+
> - **2♦** = трансфер ♠, 6+
> - **2♥** = трансфер ♦, 6+
> - **2♠** = фит ♦, 8-10 очков
