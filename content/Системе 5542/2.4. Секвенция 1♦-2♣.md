## Примеры рук отвечающего

```
♠Кх ♥Тхх ♦Дх ♣ДВххxx (13 очков, 6♣, нет 4М)
♠Дх ♥Кхх ♦Вх ♣ТКДхх (14 очков, 5♣, нет 4М)
♠Вх ♥Дхх ♦Кх ♣ТКВххх (13 очков, 6♣, 4♥333)
```

## Заявки открывающего (1♦-2♣-?)

### 2♦ = NF, 11-15hcp, 5+♦

```
Рука открывающего: ♠Кх ♥хх ♦ТДВххх ♣Вхх (11 очков)
```

**Развитие отвечающего:**

- **2♥/2♠** = GF, задержка (поиск 3NT)
- **2NT** = GF, поиск фита/контракта
- **3♣** = GF, 6+♣
- **3♦** = GF, фит в ♦
- **3♥/3♠** = GF, краткость в масти
- **3NT** = G

### 2♥/2♠ = F1, 16+hcp, 5♦4М (реверс)

```
Рука открывающего (2♥): ♠Кх ♥ТКхх ♦ТДВхх ♣Дх (17 очков)
```

**Развитие отвечающего:**

- **2♠** (после 2♥) = слабая рука из GF, релей
- **2NT** = GF, поиск фита/контракта
- **3♣♦** = GF, натурально
- **3♥/♠** = GF, держка/фит в масти
- **3NT** = G

### 2NT = GF, 15-17hcp, BAL

```
Рука открывающего: ♠ТКх ♥Дхх ♦ТВхх ♣Кхх (15 очков)
```

**Развитие отвечающего:**

- **3♣** = натурально, 6+♣
- **3♦** = фит в ♦
- **3♥/3♠** = краткость в масти, фит ♣
- **3NT** = G, нет фита
- **4♣** = фит, шлемовые амбиции

### 3♣ = NF, 11-14hcp, 4+♣

```
Рука открывающего: ♠хх ♥Кх ♦ТДххх ♣КДхх (12 очков)
```

**Развитие отвечающего:**

- **3♦** = натурально, фит в ♦
- **3♥/3♠** = GF, задержка (поиск 3NT)
- **3NT** = G
- **4♣** = фит, шлемовые амбиции
- **5♣** = фит, без шлемовых амбиций

### 3♦ = INV, 15-17hcp, 6+♦

```
Рука открывающего: ♠Кх ♥хх ♦ТКДВхх ♣Дхх (16 очков)
```

**Развитие отвечающего:**

- **3♥/3♠** = кюбид, фит ♦
- **3NT** = G, нет фита
- **4♦** = фит, шлемовые амбиции
- **5♦** = фит, без шлемовых амбиций

### 3♥/3♠ = GF, 16+hcp, Сплинтер, краткость в М, 5+♦4+♣

```
Рука открывающего (3♥): ♠Кх ♥х ♦ТКВхх ♣ТДхх (16 очков)
```

**Развитие отвечающего:**

- **3♠** (после 3♥) = кюбид, выбор минора
- **3NT** = G, без шлемовых амбиций
- **4♣♦** = выбор минора, шлемовые амбиции
- **4NT** = Блэквуд (согласован минор)
