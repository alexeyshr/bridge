> [!info]+
> F1, 6+hcp, 4+♠ (<font color="#9bbb59">м.б 5+♥</font>, если 55М - говорим 1♠)

## Примеры рук отвечающего

```
♠ДВ10х ♥Кхх ♦хх ♣Кхх (9 очков, 4♠)
♠ТКхх ♥ДВххх ♦х ♣ххх (9 очков, 55М - говорим 1♠)
♠КДххх ♥Тх ♦хх ♣Вххх (9 очков, 5♠)
```

## Заявки открывающего (1♦-1♠-?)

> [!note]- Общая модель 1♦-1♠-1x-?
> После любого ответа открывающего на 1 уровне:
>
> - **1NT/2♣♦** = NF, натурально (к игре)
> - **2♥** = NF, 6-10hcp, 5+♥ (4-я масть)
> - **2♠** = NF, 6-10hcp, 5+♠
> - **2NT** = INV → 3NT, BAL
> - **3♣** = INV, 5♠4♣
> - **3♦** = INV, 5♠4♦
> - **3♥** = GF, натурально 5♠5♥
> - **3♠** = INV, 6+♠
> - **3NT** = G
> - **4♠** = G, 6+♠, нет шлемовых амбиций

^general

### 1NT = NF, 12-14hcp, BAL (м.б 4♥)

```
Рука открывающего: ♠Кхх ♥Дхх ♦ТВхх ♣Кхх (12 очков)
```

**Развитие:**

- **2♣♦** = NF, натурально, 5+♣/♦
- **2♥** = NF, 6-10hcp, 5+♥ (4-я масть)
- **2♠** = NF, 6-10hcp, 5+♠
- **2NT** = INV → 3NT
- **3♣♦** = GF, краткость в масти, 5♠4♦ (без 4♥)
- **3♥** = GF, натурально 5♠5♥
- **3♠** = INV, 6+♠
- Все остальное см. [[2.2. Секвенция 1♦-1♠#^general|общую модель]]

### 2♣ = NF, 11-14hcp, 5♦4♣

```
Рука открывающего: ♠хх ♥Вх ♦ТДххх ♣КДхх (12 очков)
```

**Развитие:**

- **2♦♠** = NF, натурально (к игре)
- **2♥** = GF, натурально 5♠5♥
- **2NT** = INV → 3NT
- **3♣♦** = INV, натурально
- **3♥** = GF, краткость в ♥ с фитом ♣ или ♦
- **3♠** = INV, 6+♠
- **3NT** = G

### 2♦ = NF, 11-14hcp, 6+♦

```
Рука открывающего: ♠Кх ♥хх ♦ТКДххх ♣Вхх (12 очков)
```

**Развитие:**

- **2♥** = NF, 5+♥ (4-я масть)
- **2♠** = NF, 5+♠ (к игре)
- **2NT** = INV → 3NT
- **3♦** = INV, фит в ♦
- **3♥** = GF, натурально 5♠5♥
- **3♠** = INV, 6+♠
- **3NT** = G

### 2♥ = NF, 11-14hcp, 5♦4♥

```
Рука открывающего: ♠хх ♥КДхх ♦ТВххх ♣Кх (13 очков)
```

**Развитие:**

- **2♠** = NF, 6-10hcp, нет фита ♥
- **2NT** = INV → 3NT, 9-10hcp
- **3♣** = GF, задержка (поиск 3NT)
- **3♦** = INV, фит в ♦
- **3♥** = INV, фит 4♥, 9-10hcp
- **3♠** = GF, натурально 5♠5♥
- **4♥** = G, фит 5♥

### 2♠ = NF, 11-14hcp, 4♠, м.б 5♦

```
Рука открывающего: ♠ТКхх ♥хх ♦ТВххх ♣Кх (13 очков)
```

**Развитие:**

- **Пас** = 6-8hcp, фит 4♠
- **2NT** = INV → 3NT, 9-10hcp
- **3♣♦** = GF, краткость в миноре с фитом ♠
- **3♥** = GF, натурально 5♠5♥
- **3♠** = INV, 9-10hcp, фит 4♠
- **4♠** = G, 11+hcp, фит 4♠

### 2NT = NF, 18-19hcp, BAL (м.б трехцвет 4♠41♥4)

```
Рука открывающего: ♠ТКхх ♥Дх ♦ТДхх ♣Кхх (18 очков)
```

**Развитие:**

- **3♣** = опиши расклад →
  - **3♦** = краткость в ♦
  - **3♥** = BAL, 4♠4♦
  - **3♠** = BAL, 4♠4♥
  - **3NT** = BAL, 5♦332
- **3♦** = NF, 5-6hcp, 6+♦ (слабая масть, signoff)
- **3♥** = ST, натурально 5♠5♥
- **3♠** = ST, натурально, 5♠4♦
- **3NT** = G
- **4♣♦** = ST на миноре

### 3♣ = INV, 15-17hcp, 5♦4♣

```
Рука открывающего: ♠Кх ♥хх ♦ТКххх ♣ТДхх (16 очков)
```

**Развитие:**

- Все натурально, GF
- **3♦** = фит в ♦ или ♣
- **3♥** = натурально, 5♠5♥
- **3♠** = натурально, 5+♠
- **3NT** = G, нет фита
- **4♣♦** = фит, шлемовые амбиции

### 3♦ = INV, 15-17hcp, 6+♦

```
Рука открывающего: ♠Кх ♥хх ♦ТКДВхх ♣Дхх (16 очков)
```

**Развитие:**

- **3♥** = натурально, 5♠5♥
- **3♠** = натурально, 5+♠
- **3NT** = G (без фита)
- **4♦** = фит, шлемовые амбиции
- **5♦** = фит, без шлемовых амбиций

### 3♥ = INV, 15-17hcp, 5♦4♥

```
Рука открывающего: ♠хх ♥ТКхх ♦ТДххх ♣Дх (16 очков)
```

**Развитие:**

- **3♠** = натурально, 5+♠, нет фита ♥
- **3NT** = G (без фита ♥)
- **4♣♦** = кюбид (согласован ♥ с 5♠5♥)
- **4♥** = G (фит ♥)

### 3♠ = INV, 15-17hcp, 4♠, м.б 5♦

```
Рука открывающего: ♠ТКхх ♥хх ♦ТДххх ♣Дх (16 очков)
```

**Развитие:**

- **3NT** = G (без интереса к шлему)
- **4♣♦** = кюбид (согласован ♠)
- **4♥** = кюбид (согласован ♠)
- **4♠** = G (без шлемовых амбиций)
