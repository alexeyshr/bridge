> [!info]+
> F1, 6+hcp, 4+♥ (<font color="#d83931">нет 5+♠</font>, если 44М - говорим 1♥)

## Примеры рук отвечающего
```
♠Кхх ♥ДВ10х ♦хх ♣Кхх (9 очков, 4♥, не 5♠)
♠хх ♥ТКхх ♦Вхх ♣хххх (8 очков, 4♥)
♠Вх ♥ТДхх ♦Кх ♣Дхххх (11 очков, 4♥, 44 в мажорах - называем ♥)
```

## Заявки открывающего (1♦-1♥-?)

> [!note]- Общая модель 1♦-1♥-1x-?
> После любого ответа открывающего на 1 уровне:
> - **1NT/2♣♦** = NF, натурально (к игре)
> - **2♥** = NF, 6-10hcp, 5+♥
> - **2NT** = INV → 3NT, BAL
> - **3♣** = INV, 5♥4♣
> - **3♦** = INV, 5♥4♦
> - **3♥** = INV, 6+♥
> - **3NT** = G
> - **4♣** = ST, натурально с 4♣, Semi-BAL 5422
> - **4♦** = ST, натурально с 5♦, BAL 5332
> - **4NT** = INV → 6NT

^general

### 1♠ = NF, 11-17hcp, 4♠, м.б 5♦
```
Рука открывающего: ♠ТВхх ♥хх ♦КДххх ♣Кх (14 очков)
```
**Развитие:**
- **2♣♦** = NF, натурально
- **2♥** = NF, 5+♥, 6-10hcp
- **2♠** = NF, фит в ♠, 6-10hcp
- **2NT** = INV → 3NT
- **3♥** = GF, натурально 5♥4♠
- **3♠** = GF, фит 4♠, шлемовые амбиции
- Все остальное см. [[2.1. Секвенция 1♦-1♥#^general|общую модель]]

### 1NT = NF, 12-14hcp, BAL (м.б 4М)
```
Рука открывающего: ♠Кхх ♥Дхх ♦ТВхх ♣Кхх (12 очков)
```
**Развитие:**
- **2♣♦** = NF, натурально, 5+♣/♦
- **2♥** = NF, 6-10hcp, 5+♥
- **2♠** = GF, натурально 5♥4♠
- **2NT** = INV → 3NT
- **3♣♦** = GF, краткость в масти, 5♥4♦ (без 4♠)
- **3♥** = INV, 6+♥
- Все остальное см. [[2.1. Секвенция 1♦-1♥#^general|общую модель]]

### 2♣ = NF, 11-14hcp, 5♦4♣
```
Рука открывающего: ♠хх ♥Вх ♦ТДххх ♣КДхх (12 очков)
```
**Развитие:**
- **2♦♥** = NF, натурально (к игре)
- **2♠** = GF, задержка (поиск 3NT/5m)
- **2NT** = INV → 3NT
- **3♣♦** = INV, натурально
- **3♥** = INV, 6+♥
- **3♠** = GF, краткость в ♠ с фитом ♣ или ♦
- **3NT** = G

### 2♦ = NF, 11-14hcp, 6+♦
```
Рука открывающего: ♠Кх ♥хх ♦ТКДххх ♣Вхх (12 очков)
```
**Развитие:**
- **2♥** = NF, 5+♥ (к игре)
- **2♠** = GF, задержка, поиск 3NT
- **2NT** = INV → 3NT
- **3♦** = INV, фит в ♦
- **3♥** = INV, 6+♥
- **3NT** = G

### 2♥ = NF, 11-14hcp, 4♥, м.б 5♦
```
Рука открывающего: ♠хх ♥КДхх ♦ТВххх ♣Кх (13 очков)
```
**Развитие:**
- **Пас** = 6-8hcp, фит 4♥
- **2♠** = GF, натурально 5♥4♠
- **2NT** = INV → 3NT, 9-10hcp
- **3♣♦** = GF, краткость в миноре с фитом ♥
- **3♥** = INV, 9-10hcp, фит 4♥
- **4♥** = G, 11+hcp, фит 4♥

### 2♠ = F1, 16+hcp, реверс, 5+♦4♠
```
Рука открывающего: ♠ТКхх ♥хх ♦ТДВхх ♣Кх (17 очков)
```
**Развитие:**
- **2NT** = слабая рука (6-8hcp), выход через 3♣
- **3♣♦♥** = GF, натурально
- **3♠** = GF, фит в ♠
- **3NT** = G
- Все остальное = GF

### 2NT = NF, 18-19hcp, BAL (м.б трехцвет 441♥4)
```
Рука открывающего: ♠ТКх ♥Дхх ♦ТДхх ♣Кхх (18 очков)
```
**Развитие:**
- **3♣** = опиши расклад →
  - **3♦** = краткость в ♦
  - **3♥** = BAL, 4♥4♦
  - **3NT** = BAL, 5♦332
- **3♦** = NF, 5-6hcp, 6+♦ (слабая масть, signoff)
- **3♥** = ST, натурально, 5♥4♦
- **3♠** = ST, краткость в ♠, 5♥4♦
- **3NT** = G
- **4♣♦** = ST на миноре

### 3♣ = INV, 15-17hcp, 5♦4♣
```
Рука открывающего: ♠Кх ♥хх ♦ТКххх ♣ТДхх (16 очков)
```
**Развитие:**
- Все натурально, GF
- **3♦** = фит в ♦ или ♣
- **3♥** = натурально, 5+♥
- **3NT** = G, нет фита
- **4♣♦** = фит, шлемовые амбиции

### 3♦ = INV, 15-17hcp, 6+♦
```
Рука открывающего: ♠Кх ♥хх ♦ТКДВхх ♣Дхх (16 очков)
```
**Развитие:**
- **3♥** = натурально, 5+♥
- **3NT** = G (без фита)
- **4♦** = фит, шлемовые амбиции
- **5♦** = фит, без шлемовых амбиций

### 3♥ = INV, 15-17hcp, 4♥, м.б 5♦
```
Рука открывающего: ♠Кх ♥ТКхх ♦ТДххх ♣Дх (16 очков)
```
**Развитие:**
- **3♠** = кюбид (согласован ♥)
- **3NT** = G (без интереса к шлему)
- **4♣♦** = кюбид (согласован ♥)
- **4♥** = G (без шлемовых амбиций)

### 3♠ = GF, 18+hcp, Сплинтер, краткость ♠, 4♥5+♦
```
Рука открывающего: ♠х ♥ТКхх ♦ТДВхх ♣Кхх (17 очков)
```
**Развитие:**

- **4♣♦** = кюбиды, согласован ♥
