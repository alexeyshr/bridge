> [!info]+
> **2♠** = Трансфер в ♣, 0+hcp, 6+♣ ИЛИ инвит 8-9hcp
> **2NT** = Трансфер в ♦, 0+hcp, 6+♦ ИЛИ слабые миноры 55

> [!tip]- Зачем нужны трансферы в миноры?
> **Цель:** Показать длинную минорную масть или уйти из неудобного 1NT.
> 
> **Преимущества:**
> - Можем играть 3m вместо 1NT с длинным минором
> - Инвитовать с 8-9hcp (2NT прямо = инвит к 3NT без длинного минора)
> - Показать слабые миноры 55

## Примеры рук отвечающего

**Трансфер 2♠ (в ♣):**
```
♠хх ♥Кх ♦хх ♣ДВхххх (5 очков, 6♣ слабая рука)
♠Кхх ♥Дх ♦Вхх ♣КДххх (9 очков, 5♣ инвит)
♠хх ♥хх ♦Кх ♣ТДхххх (7 очков, 6♣)
```

**Трансфер 2NT (в ♦):**
```
♠хх ♥Кх ♦ДВхххх ♣хх (5 очков, 6♦ слабая рука)
♠Кх ♥хх ♦ТДхххх ♣Вхх (8 очков, 6♦)
♠хх ♥х ♦Вхххх ♣Кхххх (4 очка, слабые миноры 55)
```

## Развитие после 1NT-2♠ (трансфер в ♣)

### Открывающий принимает: 3♣
```
Рука открывающего: ♠Кхх ♥Дхх ♦ТВх ♣ТКх (16 очков)
```

### Продолжение отвечающего (1NT-2♠-3♣-?)

| Заявка | Сила | Значение |
|--------|------|----------|
| **Пас** | 0-7 | К игре, 6+♣ |
| **3♦** | 8-9 | ИНВ, 5+♣4+♦ |
| **3♥** | 8-9 | ИНВ, 5+♣4♥ |
| **3♠** | 8-9 | ИНВ, 5+♣4♠ |
| **3NT** | 8-9 | ИНВ → 3NT, ровно 5♣ |
| **4♣** | 0-7 | К игре, 7+♣ (редко) |

**Примеры:**

**Пример 1: Слабая рука, играем 3♣**
```
Север: ♠Кхх ♥Дхх ♦ТВх ♣ТКх (16 очков)
Юг: ♠хх ♥Кх ♦хх ♣ДВхххх (5 очков, 6♣)

Торговля:
Север: 1NT
Юг: 2♠ (трансфер в ♣)
Север: 3♣ (принимаю)
Юг: Пас (к игре)
```

**Пример 2: Инвит с 5♣4♦**
```
Север: ♠Кхх ♥Дхх ♦ТВх ♣ТКх (17 очков, максимум)
Юг: ♠хх ♥Кх ♦Тххх ♣КДххх (9 очков, 5♣4♦)

Торговля:
Север: 1NT
Юг: 2♠ (трансфер в ♣)
Север: 3♣ (принимаю)
Юг: 3♦ (инвит, 5♣4♦)
Север: 3NT (максимум, принимаю инвит)
```

**Пример 3: Инвит с 5♣4М**
```
Север: ♠ТКхх ♥Дхх ♦ТВ ♣Кхх (17 очков, 4♠)
Юг: ♠Дххх ♥Кх ♦хх ♣ДВххх (9 очков, 4♠5♣)

Торговля:
Север: 1NT
Юг: 2♠ (трансфер в ♣)
Север: 3♣ (принимаю)
Юг: 3♠ (инвит, 5♣4♠)
Север: 4♠ (максимум, есть фит 4♠!)
```

**Пример 4: Инвит с ровно 5♣**
```
Север: ♠Кхх ♥Дхх ♦ТВх ♣ТКх (17 очков)
Юг: ♠Дхх ♥Кх ♦хххх ♣ДВхх (9 очков, 5♣)

Торговля:
Север: 1NT
Юг: 2♠ (трансфер в ♣)
Север: 3♣ (принимаю)
Юг: 3NT (инвит, ровно 5♣)
Север: Пас (максимум, но 3NT лучше чем 5♣)
```

## Развитие после 1NT-2NT (трансфер в ♦)

### Открывающий принимает: 3♦
```
Рука открывающего: ♠Кхх ♥Дхх ♦ТВх ♣ТКх (16 очков)
```

### Продолжение отвечающего (1NT-2NT-3♦-?)

| Заявка | Сила | Значение |
|--------|------|----------|
| **Пас** | 0-7 | К игре, 6+♦ или слабые миноры 55 |
| **3♥** | 8-9 | ИНВ, 5+♦4♥ |
| **3♠** | 8-9 | ИНВ, 5+♦4♠ |
| **3NT** | 8-9 | ИНВ → 3NT, ровно 5♦ |
| **4♣** | 8-9 | ИНВ, 5+♦5+♣ (миноры) |
| **4♦** | 0-7 | К игре, 7+♦ (редко) |

**Примеры:**

**Пример 1: Слабая рука, играем 3♦**
```
Север: ♠Кхх ♥Дхх ♦ТВх ♣ТКх (16 очков)
Юг: ♠хх ♥Кх ♦ДВхххх ♣хх (5 очков, 6♦)

Торговля:
Север: 1NT
Юг: 2NT (трансфер в ♦)
Север: 3♦ (принимаю)
Юг: Пас (к игре)
```

**Пример 2: Слабые миноры 55**
```
Север: ♠Кхх ♥Дхх ♦ТВх ♣ТКх (16 очков)
Юг: ♠хх ♥х ♦Вхххх ♣Кхххх (4 очка, 55 миноры)

Торговля:
Север: 1NT
Юг: 2NT (трансфер в ♦, слабые миноры)
Север: 3♦ (принимаю)
Юг: Пас (играем где принял, у меня слабая рука)
```

**Пример 3: Инвит с 5♦4М**
```
Север: ♠ТКхх ♥Дхх ♦ТВ ♣Кхх (17 очков, 4♠)
Юг: ♠Дххх ♥Кх ♦ТДххх ♣Вх (10 очков, 4♠5♦)

Торговля:
Север: 1NT
Юг: 2NT (трансфер в ♦)
Север: 3♦ (принимаю)
Юг: 3♠ (инвит, 5♦4♠)
Север: 4♠ (максимум, есть фит 4♠!)
```

**Пример 4: Инвит с минорами 55**
```
Север: ♠Кх ♥Дхх ♦ТВх ♣ТКхх (17 очков, максимум)
Юг: ♠хх ♥Кх ♦КДххх ♣Дххх (9 очков, 55 миноры)

Торговля:
Север: 1NT
Юг: 2NT (трансфер в ♦)
Север: 3♦ (принимаю)
Юг: 4♣ (инвит, 5♦5♣)
Север: 5♣ (максимум, хороший фит в ♣, идем в гейм)
```

**Пример 5: Инвит с ровно 5♦**
```
Север: ♠Кхх ♥Дхх ♦ТВх ♣ТКх (17 очков)
Юг: ♠Дхх ♥Кх ♦ТДххх ♣Вхх (9 очков, 5♦)

Торговля:
Север: 1NT
Юг: 2NT (трансфер в ♦)
Север: 3♦ (принимаю)
Юг: 3NT (инвит, ровно 5♦)
Север: Пас (максимум, но 3NT лучше чем 5♦)
```

## Прямые заявки (без трансфера)

### 1NT-3♣ = 5-8hcp, 6+♣, инвит в 3NT с 2 фигурами
```
Отвечающий: ♠хх ♥Кх ♦Дх ♣ТВхххх (7 очков, 6♣, 2 фигуры)
```
**Развитие открывающего:**
- **Пас** = минимум, нет фита
- **3NT** = максимум или хороший фит

### 1NT-3♦ = 5-8hcp, 6+♦, инвит в 3NT с 2 фигурами
**Аналогично 3♣**

## Суммарная таблица

| Заявка | Значение |
|--------|----------|
| **2♠** | Трансфер в ♣, 0+hcp, 6+♣ ИЛИ инвит 8-9 с 5♣ |
| **2NT** | Трансфер в ♦, 0+hcp, 6+♦ ИЛИ слабые миноры 55 ИЛИ инвит 8-9 с 5♦ |
| **3♣** | 5-8hcp, 6+♣, инвит в 3NT с 2 фигурами (прямая заявка) |
| **3♦** | 5-8hcp, 6+♦, инвит в 3NT с 2 фигурами (прямая заявка) |

> [!warning]- Важные нюансы
> - **2♠/2NT** = всегда трансфер, открывающий ОБЯЗАН принять на 3m
> - **3♣/3♦** = прямые заявки, НЕ трансфер!
> - После приема трансфера и **паса** = слабая рука, к игре
> - После приема трансфера и продолжения = инвит 8-9hcp
