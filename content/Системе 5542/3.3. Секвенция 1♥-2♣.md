> [!info]+
> **НА 1-2 руке:** GF, 12+hcp, равномер или 5+♣ или фит в ♥
> **НА 3-4 руке (после паса):** Друри, максимум паса 6-10hcp с фитом 3+♥

> [!warning]- ВАЖНО: Две разные конвенции!
> - **1♥ (на 1-2 руке) - 2♣** = GF, сильная рука
> - **пас-пас-1♥ - 2♣** = Друри, максимум паса с фитом
> 
> Не путай эти две ситуации!

## Вариант А: Открытие на 1-2 руке (GF заявка)

### Примеры рук отвечающего (GF)
```
♠Кх ♥Тхх ♦Кхх ♣ДВххx (13 очков, 5♣)
♠Дхх ♥КДх ♦Тхх ♣Вххх (12 очков, равномер с фитом)
♠Кх ♥Дхх ♦Тхх ♣ТКххх (14 очков, 5♣, GF сила)
```

### Заявки открывающего (1♥-2♣-?)

#### 2♦ = NF, 11-15hcp, 5♥4+♦
```
Рука открывающего: ♠хх ♥ТКххх ♦КДхх ♣Кх (13 очков)
```
**Развитие:**
- **2♥** = GF, фит 3♥
- **2♠** = GF, задержка (поиск 3NT)
- **2NT** = GF, поиск фита/контракта
- **3♣** = GF, 6+♣
- **3♦** = GF, фит в ♦
- **3♥** = GF, фит 3♥, хорошая рука
- **3♠** = GF, краткость в ♠
- **3NT** = G

#### 2♥ = NF, 11-14hcp, 6+♥
```
Рука открывающего: ♠хх ♥ТКДххх ♦Кхх ♣Вх (12 очков)
```
**Развитие:**
- **2♠** = GF, задержка (поиск 3NT)
- **2NT** = GF, поиск фита
- **3♣** = GF, 6+♣
- **3♦** = GF, задержка (поиск 3NT)
- **3♥** = GF, фит 3♥
- **3♠** = GF, краткость в ♠ с фитом ♥
- **3NT** = G

#### 2♠ = F1, 16+hcp, 5♥4♠ (реверс)
#### 2NT = GF, 15-17hcp, BAL
#### 3♣ = NF, 11-14hcp, 4+♣
#### 3♦ = INV, 15-17hcp, 5♥5+♦
#### 3♥ = INV, 15-17hcp, 6+♥
#### 3♠/4♣/4♦ = GF, 18+hcp, Сплинтер
#### 3NT = G, 18-21hcp

## Вариант Б: Друри (после паса-паса-1♥)

> [!note]- Конвенция Друри
> **Применение:** ТОЛЬКО после паса-паса-1♥-2♣
> 
> **Цель:** Показать максимум паса (6-10 очков) с фитом в черве (3-4 карты)
> 
> **Критерии:**
> - ✓ Открытие на 3-4 руке
> - ✓ У нас максимум паса (6-10 очков)
> - ✓ Есть 3-4 карты в ♥

### Примеры рук отвечающего (Друри)
```
♠хххх ♥Кххх ♦ххх ♣хх (7 очков, максимум паса, 3♥)
♠Кхх ♥ДВхх ♦хх ♣ххх (9 очков, максимум паса, хороший фит)
♠Дх ♥Тххх ♦хххх ♣Вхх (8 очков, максимум паса, 4♥)
```

### Заявки открывающего (пас-пас-1♥-2♣-?)

| Заявка | Значение |
|--------|----------|
| **2♦** | Нет гейма, пасуй на любую заявку |
| **2♥** | F1, ИНВ, 4+♥, есть интерес к гейму |
| **2♠** | F1, ИНВ, 4+♠ 5♥ |
| **2NT** | ИНВ, 13-14hcp, 5332 |
| **3♣** | F1, ИНВ, 4+♣ 5♥ |
| **3♦** | F1, ИНВ, 4+♦ 5♥ |
| **3♥** | ИНВ, 6+♥ |
| **3♠** | GF, Сплинтер, краткость ♠ |
| **3NT** | НФ, на выбор, 5332 |
| **4♣/4♦** | GF, Сплинтер, краткость ♣/♦ |
| **4♥** | G, к игре |

#### Пример: 2♦ (нет гейма)
```
Рука открывающего: ♠Кхх ♥ДВххх ♦Тхх ♣хх (10 очков, легкое открытие на 3-4 руке)
Рука отвечающего: ♠хх ♥Кххх ♦хххх ♣Вхх (7 очков, максимум паса)

Торговля:
пас - пас - 1♥ - 2♣ (Друри, максимум паса с фитом)
2♦ (нет гейма, играй где хочешь) - Пас
```

#### Пример: 2♥ (инвит)
```
Рука открывающего: ♠Кхх ♥ТКххх ♦Дхх ♣Вх (12 очков, нормальное открытие)
Рука отвечающего: ♠Дх ♥Кххх ♦ТВхх ♣Дхх (10 очков, максимум паса!)

Торговля:
пас - пас - 1♥ - 2♣ (Друри)
2♥ (инвит, 4+♥) - 4♥ (принимаем с максимумом)
```

#### Пример: 3♠ (сплинтер)
```
Рука открывающего: ♠х ♥ТКДхх ♦Кхх ♣ТДхх (16 очков)
Рука отвечающего: ♠Дхх ♥Кххх ♦Тхх ♣Вхх (10 очков, максимум паса)

Торговля:
пас - пас - 1♥ - 2♣ (Друри)
3♠ (сплинтер, краткость ♠, хорошая рука) - 4♥ (хорошая рука, идем в гейм)
```

## Примеры торговли (GF на 1-2 руке)

**Пример 1: Поиск фита**
```
Север: ♠хх ♥ТКххх ♦Кхх ♣Вх (12 очков)
Юг: ♠Кх ♥Дхх ♦Тхх ♣ТКххх (14 очков, 5♣, фит 3♥)

Торговля:
Север: 1♥
Юг: 2♣ (GF, 5+♣ или равномер с фитом)
Север: 2♥ (6♥, 11-14hcp)
Юг: 4♥ (гейм с фитом)
```

**Пример 2: Поиск 3NT**
```
Север: ♠Кх ♥ТКххх ♦Дхх ♣Вхх (13 очков)
Юг: ♠Дхх ♥хх ♦Тхх ♣ТКДхх (14 очков, 5♣, нет фита)

Торговля:
Север: 1♥
Юг: 2♣ (GF, 5+♣)
Север: 2♥ (6♥, 11-14hcp)
Юг: 2♠ (задержка, поиск 3NT)
Север: 2NT (есть держка в ♠)
Юг: 3NT (гейм)
```
