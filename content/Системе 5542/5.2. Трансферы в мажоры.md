> [!info]+
> **2♦** = Трансфер в ♥, 0+hcp, 5+♥
> **2♥** = Трансфер в ♠, 0+hcp, 5+♠

> [!tip]- Зачем нужны трансферы?
> **Цель:** Безопасно показать длинную масть (5+М) и перевести контракт на сильную руку (1NT).
> 
> **Преимущества:**
> - Ход идет в сильную руку → противник не видит карт
> - Можем описать силу после трансфера
> - Открывающий может показать суперфит

## Примеры рук отвечающего

**Трансфер 2♦ (в ♥):**
```
♠хх ♥ДВххх ♦Кхх ♣Вхх (7 очков, 5♥)
♠Кх ♥ТКхххх ♦Дх ♣Вхх (11 очков, 6♥)
♠хх ♥Кххxx ♦Тхх ♣Дхх (8 очков, 5♥)
♠х ♥ДВхххх ♦Кхх ♣Вхх (6 очков, 6♥)
```

**Трансфер 2♥ (в ♠):**
```
♠ДВххх ♥хх ♦Кхх ♣Вхх (7 очков, 5♠)
♠ТКхххх ♥Дх ♦Кх ♣Вхх (11 очков, 6♠)
♠Кххxx ♥хх ♦Тхх ♣Дхх (8 очков, 5♠)
```

## Ответы открывающего на трансфер

### Обычный прием трансфера (2♥/2♠)
**Требования:** 2 или 3 карты в масти трансфера

```
Рука открывающего (2♦→2♥): ♠Кхх ♥Дх ♦ТВхх ♣ТКх (16 очков, 2♥)
Рука открывающего (2♥→2♠): ♠Дхх ♥Кх ♦ТВхх ♣ТКх (16 очков, 3♠)
```

### Суперприем трансфера (прыжок)
**Требования:** 4-карточный фит + максимум (16-17hcp)

**Варианты суперприема после 2♦ (трансфер в ♥):**

| Заявка | Значение |
|--------|----------|
| **2NT** | 4♥, максимум (16-17), нет дуплетов |
| **3♣** | 4♥, максимум, дуплет в ♣ |
| **3♦** | 4♥, максимум, дуплет в ♦ |
| **3♥** | 4♥, минимум (15) |
| **3♠** | 4♥, максимум, дуплет в ♠ |

> [!note]- Что такое дуплет?
> **Дуплет** = ровно 2 карты в масти.
> 
> При суперприеме показываем дуплет, чтобы отвечающий понял раскладную силу руки.

**Примеры суперприема:**
```
2NT: ♠Кхх ♥ТДхх ♦ТКх ♣Дхх (17 очков, 4♥, нет дуплетов - 4333)
3♣: ♠Тхх ♥ТКхх ♦ТДхх ♣Кх (17 очков, 4♥, дуплет ♣)
3♦: ♠Тхх ♥ТКхх ♦Кх ♣ТДхх (17 очков, 4♥, дуплет ♦)
3♥: ♠Кхх ♥ТДхх ♦ТВх ♣Кхх (15 очков, 4♥, минимум)
3♠: ♠Кх ♥ТКхх ♦ТДхх ♣Дхх (17 очков, 4♥, дуплет ♠)
```

## Продолжение после обычного приема (1NT-2♦-2♥)

### Пас = 0-7hcp, 5♥, к игре
```
Отвечающий: ♠хх ♥Дхххх ♦Кхх ♣хх (5 очков)
```

### 2♠ = INV, 4♠5+♥, 8-9hcp
```
Отвечающий: ♠Кххх ♥ДВххх ♦Тх ♣хх (9 очков, 54 мажоры)
```
**Развитие открывающего:**
- **2NT** = нет 4♠, предлагаю 3NT
- **3♥** = нет 4♠, предпочтение ♥
- **3♠** = есть 4♠, принимаю инвит
- **4♥** = нет 4♠, максимум, принимаю инвит в ♥
- **4♠** = есть 4♠, максимум, гейм

### 2NT = INV, 8-9hcp, ровно 5♥
```
Отвечающий: ♠Кхх ♥ДВххх ♦Тх ♣Вхх (9 очков, 5♥)
```
**Развитие открывающего:**
- **Пас** = минимум (15hcp)
- **3♥** = максимум (16-17hcp), принимаю инвит
- **3NT** = максимум, предпочтение БК

### 3♣ = GF, 5+♥4+♣, 10+hcp
```
Отвечающий: ♠хх ♥ТКххх ♦Дх ♣КДхх (12 очков, 5♥4♣)
```
**Развитие открывающего:**
- **3♦** = задержка (поиск фита)
- **3♥** = фит 3♥, предпочтение
- **3♠** = задержка (поиск 3NT)
- **3NT** = G, нет фита
- **4♣** = фит в ♣
- **4♥** = фит 4♥ (редко)

### 3♦ = GF, 5+♥4+♦, 10+hcp
**Аналогично 3♣**

### 3♥ = INV, 6♥, 8-9hcp
```
Отвечающий: ♠хх ♥ДВхххх ♦Тх ♣Кхх (9 очков, 6♥)
```
**Развитие открывающего:**
- **Пас** = минимум (15hcp), 2♥
- **3NT** = максимум, предпочтение БК, 2♥
- **4♥** = принимаю инвит (фит 3-4♥ или максимум)

### 3♠ = Автосплинтер, INV к шлему, 6+♥, краткость ♠
```
Отвечающий: ♠х ♥ТКДххх ♦Кхх ♣Дхх (13 очков, 6♥, синглет ♠)
```
**Развитие открывающего:**
- **3NT** = нет интереса к шлему
- **4♣♦** = кюбид (согласован ♥)
- **4♥** = нет интереса к шлему
- **4NT** = Блэквуд

### 3NT = GF, ровно 5♥, 10-15hcp
```
Отвечающий: ♠Кхх ♥ТКххх ♦Дх ♣Вхх (12 очков, 5♥)
```
**К игре:** 3NT

### 4♣/4♦ = Автосплинтер, INV к шлему, 6+♥, краткость
```
Отвечающий (4♣): ♠Кх ♥ТКДххх ♦Тхх ♣х (13 очков, 6♥, синглет ♣)
```
**Развитие:**
- Аналогично 3♠ (автосплинтер)

### 4♥ = G, 6+♥, нет интереса к шлему
```
Отвечающий: ♠хх ♥ТКхххх ♦Дх ♣Вхх (9 очков, 6♥)
```

## Продолжение после 1NT-2♥-2♠ (трансфер в ♠)

**Развитие полностью аналогично 1NT-2♦-2♥**, только заменяем ♥↔♠:

- **Пас** = 0-7hcp, 5♠, к игре
- **2NT** = INV, 8-9hcp, ровно 5♠
- **3♣♦** = GF, 5+♠4+m, 10+hcp
- **3♥** = INV, 4♥5+♠, 8-9hcp (аналог 2♠ после трансфера в ♥)
- **3♠** = INV, 6♠, 8-9hcp
- **3NT** = GF, ровно 5♠, 10-15hcp
- **4♣♦♥** = Автосплинтер, INV к шлему, 6+♠, краткость
- **4♠** = G, 6+♠, нет интереса к шлему

## Примеры торговли с трансферами

**Пример 1: Слабая рука, играем 2М**
```
Север: ♠Кхх ♥Дх ♦ТВхх ♣ТКх (16 очков)
Юг: ♠хх ♥ДВххх ♦Кхх ♣хх (5 очков, 5♥)

Торговля:
Север: 1NT
Юг: 2♦ (трансфер в ♥)
Север: 2♥ (принимаю)
Юг: Пас (к игре)
```

**Пример 2: Инвит с 5М**
```
Север: ♠Кхх ♥Дхх ♦ТВх ♣ТКх (17 очков, максимум)
Юг: ♠хх ♥ТКххх ♦Дх ♣Вххх (9 очков, 5♥)

Торговля:
Север: 1NT
Юг: 2♦ (трансфер в ♥)
Север: 2♥ (принимаю)
Юг: 2NT (инвит, ровно 5♥)
Север: 3♥ (максимум, принимаю инвит в ♥)
```

**Пример 3: GF с двумя мастями**
```
Север: ♠Кх ♥Дхх ♦ТВхх ♣ТКх (16 очков)
Юг: ♠хх ♥ТКххх ♦Дх ♣КДхх (13 очков, 5♥4♣)

Торговля:
Север: 1NT
Юг: 2♦ (трансфер в ♥)
Север: 2♥ (принимаю)
Юг: 3♣ (GF, 5♥4♣)
Север: 3♥ (фит 3♥, предпочтение)
Юг: 4♥ (гейм)
```

**Пример 4: Суперприем с дуплетом**
```
Север: ♠Кх ♥ТКхх ♦ТДхх ♣Дхх (17 очков, 4♥, дуплет ♠)
Юг: ♠Дххх ♥ДВххх ♦Кх ♣хх (10 очков, 5♥)

Торговля:
Север: 1NT
Юг: 2♦ (трансфер в ♥)
Север: 3♠ (суперприем! 4♥, максимум, дуплет ♠)
Юг: 4♥ (отлично, у меня Дххх в ♠, идем в гейм)
```

**Пример 5: Автосплинтер (шлемовая попытка)**
```
Север: ♠Кхх ♥ТДхх ♦ТКх ♣Дхх (17 очков, 4♥)
Юг: ♠х ♥КВхххх ♦Дх ♣ТКхх (14 очков, 6♥, синглет ♠)

Торговля:
Север: 1NT
Юг: 2♦ (трансфер в ♥)
Север: 2♥ (принимаю)
Юг: 3♠ (автосплинтер, краткость ♠, инвит к шлему)
Север: 4♣ (кюбид, согласован ♥, интерес к шлему)
Юг: 4♦ (кюбид)
Север: 4NT (Блэквуд)
Юг: 5♣ (1 или 4 туза)
Север: 6♥ (шлем!)
```

**Пример 6: Инвит с 4♠5♥**
```
Север: ♠ТКхх ♥Дхх ♦ТВх ♣Кхх (16 очков, 4♠)
Юг: ♠Дххх ♥ТКххх ♦Кх ♣хх (10 очков, 4♠5♥)

Торговля:
Север: 1NT
Юг: 2♦ (трансфер в ♥)
Север: 2♥ (принимаю)
Юг: 2♠ (инвит, 4♠5♥)
Север: 3♠ (есть 4♠, принимаю инвит)
Юг: 4♠ (гейм с фитом 4-4)
```

**Пример 7: Без суперприема (минимум)**
```
Север: ♠Кхх ♥ТДхх ♦ТВх ♣Кхх (15 очков, 4♥, минимум)
Юг: ♠хх ♥Кхххх ♦Дх ♣Вххх (8 очков, 5♥)

Торговля:
Север: 1NT
Юг: 2♦ (трансфер в ♥)
Север: 3♥ (суперприем, но минимум - только 15hcp)
Юг: Пас (не хватает на гейм)
```

**Пример 8: GF с ровно 5М**
```
Север: ♠Кхх ♥Дхх ♦ТВх ♣ТКх (16 очков)
Юг: ♠хх ♥ТКххх ♦Дхх ♣Кхх (12 очков, 5♥)

Торговля:
Север: 1NT
Юг: 2♦ (трансфер в ♥)
Север: 2♥ (принимаю)
Юг: 3NT (GF, ровно 5♥, играем БК)
```

## Суммарная таблица продолжений после приема трансфера

| Заявка отвечающего | Сила | Значение |
|-------------------|------|----------|
| **Пас** | 0-7 | К игре, 5М |
| **2♠ (после 2♥)** | 8-9 | ИНВ, 4♠5+♥ |
| **2NT (после 2♥)** | 8-9 | ИНВ, ровно 5♥ |
| **3♣ (после 2♥)** | 8-9 | ИНВ, ровно 5♥ |
| **3♣** | 10+ | GF, 5+М4+♣ |
| **3♦** | 10+ | GF, 5+М4+♦ |
| **3♥ (после 2♥)** | 8-9 | ИНВ, 6♥ |
| **3♥ (после 2♠)** | 8-9 | ИНВ, 4♥5+♠ |
| **3♠ (после 2♥)** | 13+ | Автосплинтер, 6+♥, краткость ♠ |
| **3♠ (после 2♠)** | 8-9 | ИНВ, 6♠ |
| **3NT** | 10-15 | GF, ровно 5М |
| **4♣** | 13+ | Автосплинтер, 6+М, краткость ♣ |
| **4♦** | 13+ | Автосплинтер, 6+М, краткость ♦ |
| **4♥ (после 2♠)** | 13+ | Автосплинтер, 6+♠, краткость ♥ |
| **4М** | 8-12 | G, 6+М, нет шлемовых амбиций |

> [!warning]- Важные нюансы
> - **2NT после 2♥** и **3♣ после 2♥** = оба означают инвит с ровно 5♥, но 3♣ чуть сильнее
> - **Автосплинтеры (3♠/4♣/4♦/4♥)** = всегда INV к шлему, требуют 13+hcp
> - **4М** напрямую = нет интереса к шлему, просто хороший 6-карт
