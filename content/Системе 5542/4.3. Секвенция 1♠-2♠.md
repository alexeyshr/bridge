> [!info]+
> NF, 7-10hcp, 3+♠

## Примеры рук отвечающего
```
♠ДВх ♥Кхх ♦хххх ♣Вхх (8 очков, 3♠)
♠Кххх ♥хх ♦Дхх ♣Вххх (7 очков, 4♠)
♠Вххх ♥Дх ♦Кхх ♣хххх (9 очков, 4♠)
♠Тххх ♥хх ♦ДВхх ♣Кхх (10 очков, 4♠ - верхняя граница)
```

## Заявки открывающего (1♠-2♠-?)

### Пас = 11-14hcp, минимум без дополнительных ценностей
```
Рука открывающего: ♠КДххх ♥хх ♦Тхх ♣Вхх (11 очков)
```
**К игре:** 2♠

### 2NT = F1, 16+hcp, поиск лучшего контракта
```
Рука открывающего: ♠ТКДхх ♥Кхх ♦ТДх ♣Кх (18 очков)
```
**Цель:** Изучить руку отвечающего для выбора между 3NT, 4♠ или даже шлемом.

**Развитие отвечающего:**
- **3♣♦♥** = натурально, краткость в масти или задержка
- **3♠** = минимум, 3♠, 7-8hcp
- **3NT** = G (максимум 9-10hcp с держками)
- **4♣♦♥** = краткость с фитом 4♠
- **4♠** = G, 4♠, не максимум

### 3♣ = F1, 16+hcp, 5♠4+♣
```
Рука открывающего: ♠ТКххх ♥хх ♦Кх ♣ТДхх (16 очков)
```
**Развитие отвечающего:**
- **3♦♥** = задержка (поиск 3NT)
- **3♠** = минимум с фитом 3♠
- **3NT** = G (с держками)
- **4♣** = фит в ♣ (редко)
- **4♦♥** = кюбид (согласован ♠)
- **4♠** = G, фит 4♠

### 3♦ = F1, 16+hcp, 5♠5+♦
```
Рука открывающего: ♠ТКххх ♥хх ♦ТДВхх ♣Кх (16 очков)
```
**Развитие:**
- Аналогично 3♣

### 3♥ = F1, 16+hcp, 5♠5+♥
```
Рука открывающего: ♠ТКххх ♥ТДВхх ♦Кх ♣Дх (17 очков)
```
**Развитие:**
- Аналогично 3♣

### 3♠ = INV, 15-17hcp, 6+♠
```
Рука открывающего: ♠ТКДВхх ♥хх ♦Тхх ♣Кх (16 очков)
```
**Развитие отвечающего:**
- **Пас** = слабая рука (7hcp)
- **4♣♦♥** = кюбид, максимум (9-10hcp)
- **4♠** = G, принимаем инвит

### 4♣/4♦/4♥ = GF, 18+hcp, Сплинтер, краткость, 6+♠
```
Рука открывающего (4♥): ♠ТКДххх ♥х ♦Тхх ♣Кхх (15 очков)
```
**Развитие отвечающего:**
- **4♠** = негатив, нет шлемовых амбиций
- **Кюбиды** = хорошая рука, идем в шлем
- **4NT** = Блэквуд

### 4♠ = G, 15-17hcp, хорошая рука
```
Рука открывающего: ♠ТКДххх ♥Кх ♦Тхх ♣Кх (17 очков)
```
**К игре:** 4♠

### 4NT = Блэквуд, вопрос о тузах
```
Рука открывающего: ♠ТКДВхх ♥Кх ♦ТКх ♣Кх (21 очко, интерес к шлему)
```

## Вариант Б: Друри (после паса-паса-1♠)

> [!tip]- Когда применять Друри
> - ✓ Открытие было на 3-4 руке (после паса-паса)
> - ✓ У вас максимум паса (6-10 очков)
> - ✓ Есть фит 3-4 карты в ♠
> 
> Цель: Показать партнеру, что у вас не проходная рука, а хорошая поддержка с максимумом.

### Примеры рук отвечающего (Друри)
```
♠Кххх ♥хх ♦ххх ♣Вххх (7 очков, максимум паса, 3♠)
♠ДВхх ♥Кхх ♦хх ♣ххх (9 очков, максимум паса, хороший фит)
♠Тххх ♥Дх ♦хххх ♣Вхх (8 очков, максимум паса, 4♠)
```

### Заявки открывающего (пас-пас-1♠-2♣-?)

| Заявка | Значение |
|--------|----------|
| **2♦** | Нет гейма, пасуй на любую заявку |
| **2♥** | F1, ИНВ, 4+♥ 5♠ |
| **2♠** | F1, ИНВ, есть интерес к гейму |
| **2NT** | ИНВ, 13-14hcp, 5332 |
| **3♣** | F1, ИНВ, 4+♣ 5♠ |
| **3♦** | F1, ИНВ, 4+♦ 5♠ |
| **3♥** | GF, Сплинтер, краткость ♥ |
| **3♠** | ИНВ, 6+♠ |
| **3NT** | НФ, на выбор, 5332, есть гейм |
| **4♣/4♦** | GF, Сплинтер, краткость ♣/♦ |
| **4♠** | G, к игре |

## Примеры торговли (Друри)

**Пример 1: 2♦ (нет гейма)**
```
Север: ♠ДВххх ♥Кхх ♦Тхх ♣хх (10 очков)
Юг: ♠Кххх ♥хх ♦хххх ♣Вх (7 очков)

Торговля:
пас - пас - 1♠ (Север) - 2♣ (Друри, Юг)
2♦ (нет гейма) - Пас
```

**Пример 2: 2♠ (инвит принят)**
```
Север: ♠ТКххх ♥Дхх ♦Кхх ♣Вх (12 очков)
Юг: ♠Дххх ♥Тх ♦Вхх ♣Кхх (10 очков, максимум!)

Торговля:
пас - пас - 1♠ (Север) - 2♣ (Друри, Юг)
2♠ (инвит) - 4♠ (принимаем с максимумом)
```

**Пример 3: 3♥ (сплинтер)**
```
Север: ♠ТКДхх ♥х ♦Кхх ♣ТДхх (16 очков)
Юг: ♠Дххх ♥Кхх ♦Тхх ♣Вхх (10 очков)

Торговля:
пас - пас - 1♠ (Север) - 2♣ (Друри, Юг)
3♥ (сплинтер, хорошая рука) - 4♠ (идем в гейм)
```

**Пример 4: 2♥ (инвит с 4♥5♠)**
```
Север: ♠ТДххх ♥ТКхх ♦Кх ♣Дх (14 очков, 5♠4♥)
Юг: ♠Кхх ♥Дхх ♦Тхх ♣Вххх (9 очков)

Торговля:
пас - пас - 1♠ (Север) - 2♣ (Друри, Юг)
2♥ (инвит, 5♠4♥) - 3♥ (фит в ♥, хорошая рука)
Север: 4♥ (гейм)
```

## Примеры торговли (GF на 1-2 руке)

**Пример 1: Поиск фита**
```
Север: ♠ТКххх ♥хх ♦Кхх ♣Вх (12 очков)
Юг: ♠Дхх ♥Кх ♦Тхх ♣ТКххх (14 очков, 5♣, фит 3♠)

Торговля:
Север: 1♠
Юг: 2♣ (GF, 5+♣ или равномер с фитом)
Север: 2♠ (6♠, 11-14hcp)
Юг: 4♠ (гейм с фитом)
```
