> [!info]+
> Сигналы в защите - способ общения с партнером через сброс карт. Каждая сброшенная карта несет информацию.

## Три типа сигналов

### 1. Сигнал отношения (Attitude)

**Вопрос:** ХОЧУ ли я продолжения этой масти?

| Карта       | Значение                               |
| ----------- | -------------------------------------- |
| **Высокая** | ХОЧУ продолжения (есть фигура или фит) |
| **Мелкая**  | НЕ ХОЧУ продолжения (нет интереса)     |

### 2. Сигнал количества (Count)

**Вопрос:** СКОЛЬКО карт у меня в масти?

| Карта              | Значение                         |
| ------------------ | -------------------------------- |
| **Высокая→Мелкая** | ЧЕТНОЕ количество (2, 4, 6...)   |
| **Мелкая→Высокая** | НЕЧЕТНОЕ количество (1, 3, 5...) |

### 3. Сигнал преференса (Suit Preference)

**Вопрос:** КАКУЮ масть я предпочитаю?

| Карта       | Значение                          |
| ----------- | --------------------------------- |
| **Высокая** | Хочу СТАРШУЮ из оставшихся мастей |
| **Мелкая**  | Хочу МЛАДШУЮ из оставшихся мастей |

---

## Приоритет сигналов

> [!tip]- Какой сигнал важнее?
> **Приоритет (от важного к менее важному):**
>
> 1. **Отношение (Attitude)** - самый важный!
> 2. **Количество (Count)** - важен в определенных ситуациях
> 3. **Преференс (Suit Preference)** - когда другие неактуальны
>
> **ВАЖНО:** Не пытайся показать все сразу! Выбирай ОДИН сигнал в каждой ситуации.

## Сигнал отношения (Attitude)

### Когда применять?

**✓ ПРИМЕНЯЕМ отношение:**

- Партнер вышел мастью
- Мы сбрасываем на его ход
- Нужно сказать: "хочу продолжения" или "не хочу"

**✗ НЕ применяем:**

- Когда важнее количество (например, в их длинной масти)
- Когда нужен преференс (например, партнер отдает нам ход)

### Как показывать отношение

| Холдинг  | Сброс           | Значение                 |
| -------- | --------------- | ------------------------ |
| **Кхх**  | **х** (высокая) | ХОЧУ продолжения, есть К |
| **Дхх**  | **х** (высокая) | ХОЧУ продолжения, есть Д |
| **Вхх**  | **х** (высокая) | ХОЧУ продолжения, есть В |
| **хххх** | **х** (мелкая)  | НЕ ХОЧУ, нет фигур       |
| **хх**   | **х** (мелкая)  | НЕ ХОЧУ, нет фигур       |

> [!note]- Что значит "высокая" и "мелкая"?
> **Относительно холдинга:**
>
> - Из Д82 → "высокая" = 8, "мелкая" = 2
> - Из 974 → "высокая" = 9 или 7, "мелкая" = 4
> - Из К63 → "высокая" = 6, "мелкая" = 3
>
> **Правило:** Старшая из мелких = поощрение, младшая = отказ

### Примеры сигнала отношения

**Пример 1: Поощряем продолжение**

```
Партнер вышел К♠ (из КД или ТК)
Болван: ♠742
Вы: ♠Д108

Анализ:
- У партнера К♠ (и обещает Д или Т)
- У нас Д♠
- Хотим, чтобы он продолжил ♠

Сброс: 8♠ (высокая = ХОЧУ продолжения)
Результат: Партнер продолжит ♠, возьмем Д
```

**Пример 2: Отказываем от продолжения**

```
Партнер вышел К♠
Болван: ♠742
Вы: ♠1083

Анализ:
- У партнера К♠
- У нас нет фигур в ♠
- НЕ хотим продолжения

Сброс: 3♠ (мелкая = НЕ ХОЧУ продолжения)
Результат: Партнер переключится на другую масть
```

**Пример 3: Фит без фигур**

```
Партнер вышел К♥
Болван: ♥942
Вы: ♥10763

Анализ:
- У партнера К♥ (и Д)
- У нас 4 карты, но нет фигур
- Умеренное поощрение

Сброс: 7♥ (средняя = умеренное поощрение, есть фит)
```

## Сигнал количества (Count)

### Когда применять?

**✓ ПРИМЕНЯЕМ количество:**

- Разыгрывающий играет масть (нам нужно знать, когда брать)
- Противник в болване играет длинную масть
- Нужно понять распределение

**✗ НЕ применяем:**

- На первом ходе партнера (там отношение!)
- Когда важнее отношение или преференс

### Как показывать количество

> [!tip]- Правило "Высокая-Мелкая = Четное"
>
> - **Четное** (2, 4, 6): Высокая → Мелкая (ВМ)
> - **Нечетное** (1, 3, 5): Мелкая → Высокая (МВ)

| Холдинг   | 1-й сброс | 2-й сброс | Количество   |
| --------- | --------- | --------- | ------------ |
| **Д8**    | **8**     | **Д**     | 2 (четное)   |
| **К74**   | **4**     | **7**     | 3 (нечетное) |
| **В863**  | **8**     | **3**     | 4 (четное)   |
| **10752** | **2**     | **5**     | 5 (нечетное) |

### Примеры сигнала количества

**Пример 1: Четное количество (дуплет)**

```
Разыгрывающий играет из болвана: Т♦
Болван: ♦ТКД10
Вы: ♦96

Анализ:
- Нужно показать количество
- У нас 2♦ (четное)

Сброс: 9♦ (высокая из 96)
Результат: Партнер поймет, что у нас 2 карты (четное)
→ Значит у разыгрывающего 5 карт (нечетное)
```

**Пример 2: Нечетное количество (триплет)**

```
Разыгрывающий играет из болвана: Т♦
Болван: ♦ТКД10
Вы: ♦964

Анализ:
- Нужно показать количество
- У нас 3♦ (нечетное)

Сброс: 4♦ (мелкая из 964)
Результат: Партнер поймет, что у нас 3 карты (нечетное)
→ Значит у разыгрывающего 4 карты (четное)
```

**Пример 3: Определение распределения**

```
Болван играет: ♣ТКДВ
Вы: ♣1086
Партнер дает: ♣9 (высокая)

Анализ:
- Партнер показал четное (2 или 4♣)
- У нас 3♣
- Итого в защите: 5 или 7 карт
- У разыгрывающего: 8 или 6 карт

Логика:
- Если партнер дал 9♣ из 92 (дуплет)
  → У разыгрывающего 8 карт
- Если из 9742 (четверка)
  → У разыгрывающего 6 карт
```

## Сигнал преференса (Suit Preference)

### Когда применять?

**✓ ПРИМЕНЯЕМ преференс:**

- Партнер отдает нам ход (мы берем взятку)
- Отношение и количество неактуальны
- Нужно сказать: "какую масть я хочу"

**Типичные ситуации:**

- Перебитие карты партнера
- Партнер дает нам взятку тузом

### Как показывать преференс

> [!tip]- Правило "Высокая-Низкая"
> Исключаем козырную масть и масть, в которой сейчас играем.
>
> **Из оставшихся двух мастей:**
>
> - **Высокая карта** = хочу СТАРШУЮ масть
> - **Мелкая карта** = хочу МЛАДШУЮ масть

### Масти по старшинству

**Ранг мастей:**

1. ♠ (самая старшая)
2. ♥
3. ♦
4. ♣ (самая младшая)

### Примеры сигнала преференса

**Пример 1: Перебитие (хочу старшую масть)**

```
Контракт: 4♥ (козырь)
Партнер вышел: Т♦
Болван: ♦2
Вы перебиваете: ♥К (козырь)

Какую карту ♦ сбросить под своё перебитие?

Ваша рука: ♠Тхх ♥Кх ♦Дх ♣хххх

Анализ:
- Исключаем: ♥ (козырь), ♦ (текущая масть)
- Остаются: ♠ (старшая) и ♣ (младшая)
- У нас Т♠ - хотим ♠!

Сброс: Д♦ (высокая = хочу ♠!)
Результат: Партнер вернет ♠, мы возьмем Т♠
```

**Пример 2: Перебитие (хочу младшую масть)**

```
Контракт: 4♥ (козырь)
Партнер вышел: Т♦
Болван: ♦2
Вы перебиваете: ♥К

Ваша рука: ♠хх ♥Кх ♦Дх ♣ТКххх

Анализ:
- Исключаем: ♥ (козырь), ♦ (текущая масть)
- Остаются: ♠ (старшая) и ♣ (младшая)
- У нас ТК♣ - хотим ♣!

Сброс: 2♦ (мелкая = хочу ♣!)
Результат: Партнер вернет ♣, мы возьмем ТК♣
```

**Пример 3: Партнер дает тузом**

```
Контракт: 4♠ (козырь)
Партнер берет: Т♥
Вы даете: ?

Ваша рука: ♠хх ♥842 ♦Тхххх ♣Кх

Анализ:
- Партнер взял Т♥, даст нам ход
- Исключаем: ♠ (козырь), ♥ (текущая масть)
- Остаются: ♦ (старшая) и ♣ (младшая)
- У нас Т♦ - хотим ♦!

Сброс: 8♥ (высокая = хочу ♦!)
Результат: Партнер вернет ♦, мы возьмем Т♦
```

**Пример 4: Нет предпочтения**

```
Контракт: 4♠
Партнер берет: Т♥
Вы даете: ?

Ваша рука: ♠хх ♥842 ♦Вххх ♣Дхх

Анализ:
- Нет явного предпочтения (♦Вххх и ♣Дхх одинаково)
- Даем среднюю карту

Сброс: 4♥ (средняя = нет предпочтения)
```

## Приоритет и выбор сигнала

### Матрица: Какой сигнал давать?

| Ситуация                       | Сигнал             | Приоритет  |
| ------------------------------ | ------------------ | ---------- |
| **Партнер вышел мастью**       | Отношение          | 1 (высший) |
| **Разыгрывающий играет масть** | Количество         | 2          |
| **Партнер дает нам ход**       | Преференс          | 3          |
| **Сброс на другую масть**      | Отношение к сбросу | 2          |
| **Перебитие**                  | Преференс          | 3          |

### Конфликт сигналов

**Ситуация:** Партнер вышел К♠, у нас ♠Д82

**Проблема:**

- Хотим показать отношение: "хочу продолжения" (8♠)
- Хотим показать количество: "три карты" (2♠)

**Решение:**

- **1-й сброс:** 8♠ (ОТНОШЕНИЕ - приоритет 1!)
- **2-й сброс:** 2♠ (количество, если актуально)

> [!warning]- ВАЖНО: Один сигнал за раз!
> Не пытайся показать отношение И количество одновременно.
>
> **Приоритет:**
>
> 1. Сначала отношение (хочу/не хочу)
> 2. Потом количество (если нужно)

## Детальные примеры по ситуациям

### Ситуация 1: Партнер вышел тузом (против козыря)

> [!tip]- Т просит отношение!
> Когда партнер выходит тузом против козырного контракта, он ВСЕГДА просит отношение.

**Пример 1: Поощряем**

```
Контракт: 4♠
Партнер вышел: Т♥
Болван: ♥952
Вы: ♥К76

Сброс: 7♥ (высокая = ХОЧУ, есть К♥!)
Результат: Партнер продолжит ♥, возьмем К
```

**Пример 2: Отказываем**

```
Контракт: 4♠
Партнер вышел: Т♥
Болван: ♥К52
Вы: ♥1084

Сброс: 4♥ (мелкая = НЕ ХОЧУ, нет фигур)
Результат: Партнер переключится на другую масть
```

**Пример 3: Дуплет (особый случай)**

```
Контракт: 4♠
Партнер вышел: Т♥
Болван: ♥Д52
Вы: ♥К6

Сброс: К♥! (старшая из дуплета = дуплет + фигура!)
Результат: Партнер поймет: у нас К и еще одна (дуплет)
→ Продолжит ♥ на следующем ходу
```

---

### Ситуация 2: Партнер вышел королем

> [!tip]- К тоже просит отношение!
> К обещает Д (или Т). Партнер хочет знать: продолжать ли?

**Пример 1: Поощряем (есть фигура)**

```
Контракт: 3NT
Партнер вышел: К♠
Болван: ♠942
Вы: ♠Т76

Сброс: 7♠ (высокая = ХОЧУ, есть Т!)
Результат: Партнер продолжит ♠
```

**Пример 2: Отказываем (нет фигур)**

```
Контракт: 3NT
Партнер вышел: К♠
Болван: ♠Д942
Вы: ♠1076

Сброс: 6♠ (мелкая = НЕ ХОЧУ)
Результат: Партнер переключится
```

**Пример 3: Дуплет с фигурой**

```
Контракт: 3NT
Партнер вышел: К♠
Болван: ♠В942
Вы: ♠Т6

Сброс: Т♠! (дуплет с тузом)
Результат: Партнер продолжит ♠ на следующем ходу
```

### Ситуация 3: Разыгрывающий играет масть

> [!tip]- Показываем количество!
> Когда разыгрывающий играет масть, партнеру нужно знать распределение.

**Пример 1: Четное количество**

```
Контракт: 3NT
Болван играет: ♣ТКД из ♣ТКДх
Вы: ♣10842

Сброс:
- 1-й ход: 8♣ (высокая)
- 2-й ход: 2♣ (мелкая)

Сигнал: ВМ = ЧЕТНОЕ (4 карты)
Результат: Партнер поймет, что у разыгрывающего 3♣ (нечетное)
```

**Пример 2: Нечетное количество**

```
Контракт: 3NT
Болван играет: ♣ТКД из ♣ТКДх
Вы: ♣1084

Сброс:
- 1-й ход: 4♣ (мелкая)
- 2-й ход: 8♣ (высокая)

Сигнал: МВ = НЕЧЕТНОЕ (3 карты)
Результат: Партнер поймет, что у разыгрывающего 4♣ (четное)
```

### Ситуация 4: Сброс на посторонние масти

> [!tip]- Сброс = отношение к этой масти
> Когда сбрасываем на масть, в которой не можем снести, показываем отношение к МАСТИ СБРОСА.

**Пример:**

```
Контракт: 4♠ (козырь)
Болван играет ♦ТКДВ (длинная масть)
Мы должны сбросить 4 карты

Наша рука: ♠хх ♥КД10х ♦82 ♣Вхххх

Стратегия сбросов:
1. Сброс: ♥10 (высокая = ХОЧУ ♥!)
2. Сброс: ♣х (мелкая = НЕ ХОЧУ ♣)
3. Сброс: ♣х (мелкая = НЕ ХОЧУ ♣)
4. Сброс: ♣х (мелкая = НЕ ХОЧУ ♣)

Результат: Партнер поймет, что мы хотим ♥ (не ♣)
```

### Ситуация 5: Перебитие (рафф)

> [!tip]- Карта перебития = преференс
> Какую карту используем для перебития → показывает масть, которую хотим вернуть.

**Пример 1: Хотим старшую масть**

```
Контракт: 4♥ (козырь)
Партнер вышел: Т♦
Болван: ♦2
Вы перебиваете козырем

Ваша рука: ♠Тхх ♥1098 ♦х ♣хххх

Анализ:
- Исключаем: ♥ (козырь), ♦ (текущая)
- Остаются: ♠ (старшая), ♣ (младшая)
- У нас Т♠ - хотим ♠!

Перебиваем: 10♥ (высокая козырная = хочу ♠!)
Результат: Партнер вернет ♠
```

**Пример 2: Хотим младшую масть**

```
Контракт: 4♥ (козырь)
Партнер вышел: Т♦
Болван: ♦2
Вы перебиваете

Ваша рука: ♠хх ♥1098 ♦х ♣ТКххх

Анализ:
- Остаются: ♠ (старшая), ♣ (младшая)
- У нас ТК♣ - хотим ♣!

Перебиваем: 8♥ (мелкая козырная = хочу ♣!)
Результат: Партнер вернет ♣
```

**Пример 3: Нет предпочтения**

```
Контракт: 4♠
Партнер вышел: Т♣
Вы перебиваете: ♠7

Ваша рука: ♠743 ♥Дхх ♦Вхх ♣х

Анализ:
- Нет явного предпочтения (♥Дхх и ♦Вхх примерно одинаковы)

Перебиваем: 4♠ (средняя = нет предпочтения)
```

## Специальные сигналы

### 1. Сигнал развала (Распад масти)

**Ситуация:** Партнер вышел К, болван открылся с Тхх, у нас х/хх

**Проблема:** Нужно срочно переключиться!

**Сигнал:** Даем САМУЮ ВЫСОКУЮ карту (даже из хх!)

**Пример:**

```
Партнер вышел: К♠
Болван: ♠Т73
Вы: ♠92

Анализ:
- У болвана Т♠
- У партнера К♠ (и Д♠)
- Масть развалилась! Нужно переключаться!

Сброс: 9♠ (САМАЯ высокая = развал, переключайся!)
Результат: Партнер поймет и переключится на другую масть
```

### 2. МВМ (Мелкая-Высокая-Мелкая)

**Применение:** Показ дуплета или триплета через последовательность сбросов

**Из 3 карт (триплет):**

```
♥974 → даем: 4 (мелкая), затем 9 (высокая), затем 7
Партнер видит МВ → понимает: 3 карты (нечетное)
```

**Из 2 карт (дуплет):**

```
♥94 → даем: 9 (высокая), затем 4 (мелкая)
Партнер видит ВМ → понимает: 2 карты (четное)
```

### 3. Форсирующий сброс (Lavinthal)

**Применение:** Когда сбрасываем явно бесполезную карту, это ПРЕФЕРЕНС!

**Пример:**

```
Контракт: 4♠
Болван играет: ♥ТКДВ (мы все сбросили ♥)
Теперь играет из болвана: ♣ТКД

Ваша рука: ♠хх ♥- ♦Тхххх ♣Вхххх

Сброс на ♣ТКД:
- У нас Т♦ - хотим ♦
- Исключаем: ♠ (козырь), ♣ (текущая)
- Остаются: ♥ (старшая), ♦ (младшая)
- У нас ♥- и ♦Тхххх

1-й сброс на ♣Т: 2♦ (мелкая = хочу младшую = ♦!)
Результат: Партнер поймет, что нам нужен ♦ (не ♥)
```

## Типичные ошибки в сигналах

### ❌ Ошибка 1: Путаница в приоритетах

```
Партнер вышел: К♠
Болван: ♠742
Вы: ♠Д96

НЕПРАВИЛЬНО: Сброс 9♠ и мысль "это четное количество"
ПРАВИЛЬНО: Сброс 9♠ = "ХОЧУ продолжения, есть Д!"

Причина: На ход партнера ВСЕГДА отношение (не количество!)
```

### ❌ Ошибка 2: Сброс нужной карты

```
Контракт: 3NT
Болван играет: ♣ТКДВх
Вы: ♣1086

НЕПРАВИЛЬНО: Сброс 10♣ (думая "высокая = четное")
ПРАВИЛЬНО: Сброс 6♣ (сохраняем 10♣ для взятки!)

Причина: Не жертвуй нужными картами ради сигнала!
```

### ❌ Ошибка 3: Сигнал в очевидной ситуации

```
Партнер вышел: Т♠
Болван: ♠КД10хх
Вы: ♠хх

НЕПРАВИЛЬНО: Долго думать, какую сбросить
ПРАВИЛЬНО: Любую (сигнал бесполезен, масть развалилась)

Причина: Партнер видит болван, он и так всё понял!
```

## Практические примеры защиты с сигналами

### Пример 1: Полная раздача с отношением

```
Контракт: 4♠
         Болван
         ♠ТКД10
         ♥952
         ♦743
         ♣852
Партнер         Вы
♠хх            ♠хх
♥Т106          ♥КД87
♦ТВ109         ♦Д652
♣В1064         ♣Д73
         Разыгрывающий
         ♠Дхххх
         ♥Вх
         ♦Кх
         ♣ТКх

Защита:
1. Партнер выходит: Т♥
2. Болван: 2♥
3. Вы даете: 8♥ (высокая = ХОЧУ! есть КД♥)
4. Разыгрывающий: В♥

5. Партнер понял сигнал: продолжает 10♥
6. Вы берете: К♥
7. Вы продолжаете: Д♥
8. Партнер перебивает: ♠х (козырь)

Результат: Взяли 4 взятки (Т♥, К♥, Д♥, ♠ рафф), положили 4♠!
```

### Пример 2: Полная раздача с количеством

```
Контракт: 3NT
         Болван
         ♠742
         ♥К65
         ♦ТКД10
         ♣ТКД
Партнер         Вы
♠ДВ1096        ♠К83
♥Т102          ♥Д874
♦94            ♦8652
♣1085          ♣76
         Разыгрывающий
         ♠Тх
         ♥Вх
         ♦Вх
         ♣Вхххх

Защита:
1. Партнер выходит: Д♠ (из последовательности)
2. Болван: 2♠
3. Вы: 8♠ (высокая = ХОЧУ! есть К♠)
4. Разыгрывающий: Т♠ (взял)

5. Разыгрывающий играет из болвана: ♦ТКД
6. Вы даёте количество:
   - На Т♦: 8♦ (высокая)
   - На К♦: 2♦ (мелкая)
   - Сигнал: ВМ = ЧЕТНОЕ (4♦)

7. Партнер понял: у разыгрывающего 2♦ (четное)
   → После 2 кругов ♦ партнер не даст еще одну

Результат: Правильная защита, положили 3NT
```

### Пример 3: Полная раздача с преференсом

```
Контракт: 4♥
         Болван
         ♠К742
         ♥ТКД10
         ♦852
         ♣К5
Партнер         Вы
♠хх            ♠Д10
♥хх            ♥х
♦ТКД10         ♦Вхххх
♣Дхххх         ♣ТВхх
         Разыгрывающий
         ♠Тхххх
         ♥Вхххх
         ♦х
         ♣хх

Защита:
1. Партнер выходит: Т♦
2. Болван: 2♦
3. Вы: Д♦ (дуплет с Д)
4. Разыгрывающий перебивает: ♥В

5. Партнер продолжает: К♦
6. Болван: 5♦
7. Вы перебиваете: ♥х (козырь)

8. Какую ♦ сбросить под свое перебитие?
   - Исключаем: ♥ (козырь), ♦ (текущая)
   - Остаются: ♠ (старшая), ♣ (младшая)
   - У нас ТВ♣ - хотим ♣!

9. Сброс: х♦ (мелкая = хочу ♣!)

10. Партнер понял: возвращает ♣
11. Вы берете: Т♣ и В♣

Результат: Взяли 5 взяток (Т♦, К♦, ♥рафф, Т♣, В♣), положили 4♥!
```

## Чек-лист сигналов в защите

### ✅ Перед сбросом карты спроси себя:

**1. Какая ситуация?**

- [ ] Партнер вышел мастью? → **Отношение**
- [ ] Разыгрывающий играет масть? → **Количество**
- [ ] Я перебиваю или партнер дает ход? → **Преференс**
- [ ] Сброс на постороннюю масть? → **Отношение к сбросу**

**2. Какой сигнал показать?**

- [ ] Отношение: Хочу/Не хочу → Высокая/Мелкая
- [ ] Количество: Четное/Нечетное → ВМ/МВ
- [ ] Преференс: Какую масть? → Высокая=старшая/Мелкая=младшая

**3. Какую карту можно сбросить?**

- [ ] Не сбрасывай нужные карты (В, 10 у болвана Дх)!
- [ ] Выбирай из "мусорных" карт
- [ ] Сохраняй фигуры для взяток

**4. Думай про партнера:**

- [ ] Что ему важно знать?
- [ ] Поймет ли он мой сигнал?
- [ ] Не противоречу ли своим предыдущим сигналам?

---

## Золотые правила сигналов

1. **Отношение > Количество > Преференс** (приоритет!)
2. **На ход партнера** → ВСЕГДА отношение
3. **Высокая = хочу/четное/старшая масть**
4. **Мелкая = не хочу/нечетное/младшая масть**
5. **Дуплет с фигурой** → сброс фигуры!
6. **Не жертвуй нужные карты** ради сигнала
7. **МВМ** (Мелкая-Высокая-Мелкая) из 3 карт
8. **ВМ** (Высокая-Мелкая) из 2 карт
9. **Один сигнал за раз** - не пытайся показать всё
10. **Думай, что важно партнеру** в этой ситуации
