> [!info]+
> Полуфорсирующий, 4-6 с фитом ♠, или 5-10 без фита, или 10-12 равномер

## Примеры рук отвечающего
```
♠Дх ♥Кхх ♦хххх ♣Вххх (5 очков, 2♠ - слабая рука с фитом)
♠хх ♥Кхх ♦ТДхх ♣Вххх (9 очков, без фита ♠)
♠Дхх ♥Кхх ♦ТВхх ♣Кхх (11 очков, равномер)
```

## Заявки открывающего (1♠-1NT-?)

### Пас = 11-14hcp, BAL, 5♠332, минимум
```
Рука открывающего: ♠ТДххх ♥Кхх ♦Дхх ♣Вх (11 очков)
```
**К игре:** 1NT (может быть опасен, но лучший контракт)

### 2♣ = ГАЗИЛЛИ! Искусственная заявка
**Значения:**
- Любая рука 16+hcp (все силы)
- ИЛИ 11-15hcp с 5♠4♣ (полунатуральная)

> [!note]- Что такое Газилли?
> **Газилли** - конвенция, позволяющая эффективно разделить руки открывающего по силе через систему **зон**:
> - **Зона 1 (11-15):** Масти на 2 уровне
> - **Зона 2 (15-17):** БК и масти на 3 уровне (инвит)
> - **Зона 3 (18+):** Прямые GF заявки
> 
> После 1♠-1NT-2♣ отвечающий говорит **2♦** (waiting) = 8+hcp, и открывающий уточняет свою силу.

**Ответы отвечающего на 2♣:**

| Заявка | Значение |
|--------|----------|
| **2♦** | 8+hcp, waiting (жду уточнения силы) |
| **2♥** | <8hcp, задержка в ♥ (поиск лучшего контракта) |
| **2♠** | <8hcp, 2-3♠ (к игре) |
| **2NT** | <8hcp, 0-1♠, 4+4+ миноры |
| **3♣/3♦** | <8hcp, 6+♣/♦ |

**После 2♦ (waiting 8+hcp) открывающий уточняет:**

| Заявка | Зона | Значение |
|--------|------|----------|
| **2♥** | 3 (16+) | 5♠4♥ |
| **2♠** | 1 (11-15) | 6♠ или 5♠4♣ |
| **2NT** | 3 (16+) | 5♠4♦ |
| **3♣** | 3 (16+) | 5♠4+♣ |
| **3♦** | 3 (16+) | 5♠5+♦ |
| **3♥** | 3 (16+) | 5♠5+♥ |
| **3♠** | 2 (15-17) | 6+♠, инвит |
| **3NT** | 2 (15-17) | 5♠(332) |
| **4♣/4♦/4♥** | 3 (18+) | Сплинтер, краткость, 6+♠ |

### 2♦ = NF, 11-14hcp, 5♠4+♦
```
Рука открывающего: ♠ТКххх ♥хх ♦КДхх ♣Кх (13 очков)
```
**Развитие:**
- **Пас** = предпочтение ♦ (редко)
- **2♥** = GF, задержка в ♥ (поиск 3NT)
- **2♠** = NF, предпочтение, 2-3♠
- **2NT** = INV → 3NT, 10-11hcp
- **3♣** = GF, задержка в ♣ (поиск 3NT)
- **3♦** = INV, фит в ♦
- **3♠** = INV, фит 3♠
- **3NT** = G

### 2♥ = NF, 11-14hcp, 5♠4+♥
```
Рука открывающего: ♠ТКххх ♥ТДхх ♦Кх ♣хх (13 очков)
```
**Развитие:**
- **2♠** = NF, предпочтение, 2♠
- **2NT** = INV → 3NT, 10-11hcp
- **3♣♦** = GF, задержка (поиск 3NT)
- **3♥** = INV, фит в ♥ (редко)
- **3♠** = INV, фит 3♠
- **3NT** = G
- **4♠** = G, фит 3♠

### 2♠ = NF, 11-14hcp, 6+♠
```
Рука открывающего: ♠ТКДххх ♥хх ♦Кхх ♣Вх (12 очков)
```
**Развитие:**
- **Пас** = слабая рука, 2-3♠
- **2NT** = INV → 3NT, 10-11hcp
- **3♣♦♥** = GF, задержка (поиск 3NT)
- **3♠** = INV, фит 3♠, 8-9hcp
- **3NT** = G
- **4♠** = G, фит 3♠

### 2NT = INV, 18-19hcp, БК 5♠(332)
```
Рука открывающего: ♠ТКДхх ♥Кхх ♦ТДх ♣Кх (19 очков)
```
**Развитие:**
- **Пас** = 5-7hcp, минимум
- **3♣♦♥** = натурально, 5+ карт (поиск фита)
- **3♠** = фит 3♠, 8+hcp
- **3NT** = G, 8+hcp

### 3♣ = INV, 15-17hcp, 5♠4+♣
```
Рука открывающего: ♠ТКххх ♥хх ♦Кх ♣ТДхх (16 очков)
```
**Развитие:**
- Все GF
- **3♦♥** = задержка (поиск 3NT)
- **3♠** = фит 3♠
- **3NT** = G
- **4♣** = фит в ♣, шлем

### 3♦ = INV, 15-17hcp, 5♠5+♦
### 3♥ = INV, 15-17hcp, 5♠5+♥
### 3♠ = INV, 15-17hcp, 6+♠
### 3NT = G, 18-21hcp, остальные руки

## Примеры торговли с Газилли

**Пример 1: Зона 1 (11-15hcp)**
```
Север: ♠ТКххх ♥хх ♦Кх ♣КДхх (13 очков)
Юг: ♠Дхх ♥Вхх ♦Тхх ♣Вххх (8 очков)

Торговля:
Север: 1♠
Юг: 1NT
Север: 2♣ (Газилли)
Юг: 2♦ (waiting, 8hcp)
Север: 2♠ (зона 1, 11-15hcp, 6♠ или 5♠4♣)
Юг: 3♠ (инвит с 3♠)
Север: 4♠ (принимаем инвит, у нас максимум зоны 1)
```

**Пример 2: Зона 3 (16+hcp)**
```
Север: ♠ТКДхх ♥ТДВхх ♦Тх ♦Кх (19 очков, 5♠4♥)
Юг: ♠Дхх ♥Кх ♦Дхх ♣Вххх (10 очков)

Торговля:
Север: 1♠
Юг: 1NT
Север: 2♣ (Газилли)
Юг: 2♦ (waiting, 10hcp)
Север: 2♥ (зона 3, 16+hcp, 5♠4♥!)
Юг: 2NT (нет фита, есть держки)
Север: 3NT (гейм)
```

**Пример 3: Слабая рука отвечающего**
```
Север: ♠ТКххх ♥хх ♦Кх ♣ТДхх (15 очков)
Юг: ♠Дхх ♥Вххх ♦хххх ♣хх (5 очков)

Торговля:
Север: 1♠
Юг: 1NT
Север: 2♣ (Газилли)
Юг: 2♠ (<8hcp, 2-3♠, к игре)
Север: Пас
```
