> [!info]+
> Первый ход - самое важное решение в защите. Правильный выход может сразу положить контракт или помочь найти оптимальную защиту.

> [!tip]- Смотри также
> - [[Сигналы в защите]] - Как общаться с партнером после первого хода
> - [[Вист]] - Конкурентная торговля и защитные заявки

## Базовые принципы выбора хода

### 1. Против БК (NT) vs против козырных контрактов

| Контракт     | Стратегия                                       |
| ------------ | ----------------------------------------------- |
| **БК (NT)**  | Ходим **длинной мастью** (устанавливаем взятки) |
| **Козырной** | Ходим **безопасно** (не отдаем лишних взяток)   |

### 2. Слушай торговлю!

**Противники заявляли масть?**
- ❌ НЕ ходи в их масть (обычно)
- ✅ Ходи в незаявленную масть

**Партнер заявлял масть?**
- ✅ Обычно ходи в его масть
- ❌ Не ходи в его масть только с очень хорошей причиной

### 3. Типы выходов

- **Активный ход:** Атакуем, пытаемся взять быстрые взятки
- **Пассивный ход:** Безопасный ход, не отдаем ничего

## Выход из последовательностей

### Таблица выходов из последовательностей

> [!tip]- Основное правило
> Из последовательности выходим **ВЕРХНЕЙ** картой (top of sequence)

| Последовательность | Карта выхода | Что обещаем |
|-------------------|--------------|-------------|
| **ТКД** | Т | Обещаем К и Д |
| **ТКВ** | Т | Обещаем К, возможно В |
| **ТК** (без Д) | Т | Обещаем К, НЕТ Д |
| **КД10** | К | Обещаем Д и 10 |
| **КДВ** | К | Обещаем Д, возможно В |
| **КД** (без 10) | К | Обещаем Д, НЕТ 10 |
| **Д10х** | Д | Обещаем 10, НЕТ К |
| **Д10** | Д | Обещаем 10 |
| **ДВ9** | Д | Обещаем В и 9 |
| **В10х** | В | Обещаем 10, НЕТ Д |
| **В109** | В | Обещаем 10 и 9 |
| **109х** | 10 | Обещаем 9, НЕТ В |

### Примеры выходов из последовательностей

```
♠ТКДхх → ходим Т♠ (top of sequence)
♠КДВх → ходим К♠ (обещаем Д и В)
♠Д10хх → ходим Д♠ (обещаем 10, нет К)
♠В109х → ходим В♠ (обещаем 10 и 9)
♠109хх → ходим 10♠ (обещаем 9, нет В)
```

> [!note]- Почему это важно?
> Партнер видит нашу карту выхода и **ЗНАЕТ**, какие карты у нас есть!
> 
> **Пример:** Мы вышли К♠
> - Партнер знает: у нас Д♠ (гарантированно!)
> - Партнер знает: нет Т♠ (иначе вышли бы с Т)

## Выход от короля (специальный случай)

### Правило: К обещает Д (или Т)

**Варианты выхода королем:**
1. **ТКх** → К♠ (обещаем Т)
2. **ТКхх+** → К♠ (обещаем Т)
3. **КДх** → К♠ (обещаем Д, нет Т)
4. **КДхх+** → К♠ (обещаем Д, нет Т)

> [!warning]- НЕ ходи королем от Кх или Кхх!
> **Проблема:** Отдашь королю в руку разыгрывающего бесплатно!
> 
> **Из Кх/Кхх:** Ходи мелкой картой (4-я/2-я по старшинству)

### Примеры

```
✅ ХОДИМ К♥:
♠хх ♥КДхх ♦Тхх ♣Вхх → К♥ (обещаем Д♥)

✅ ХОДИМ К♠:
♠ТКхх ♥Дх ♦хххх ♣Вхх → К♠ (обещаем Т♠)

❌ НЕ ходим К♦ (ходим мелкой):
♠Тхх ♥Кхх ♦Кх ♣Дхххх → 2♦ (из Кх ходим мелкой!)

❌ НЕ ходим К♣:
♠Дххх ♥Кхх ♦Тх ♣Кхх → 2♣ (из Кхх ходим мелкой!)
```

## Выход от туза

### Правило: Т против козырного контракта

> [!tip]- Т просит отношения (attitude)
> Когда выходим тузом против козырного контракта, партнер должен показать **отношение** к этой масти:
> - **Высокая карта** = хочу продолжения (есть фигура)
> - **Мелкая карта** = не хочу продолжения (нет фигуры)

**Когда ходить тузом?**
1. **ТКх/ТКхх** → ходим К (не Т!)
2. **Тх/Тхх** → ходим Т (чтобы не потерять)
3. **Тххх+** → обычно НЕ ходим (можем сохранить для перебития)

### Примеры

```
✅ ХОДИМ Т♥ (дуплет):
♠Кхх ♥Тх ♦Дхххх ♣Вхх (контракт 4♠)
Причина: Тх может пропасть, если не возьмем сейчас

✅ ХОДИМ К♦ (из ТК):
♠Кхх ♥хх ♦ТКхх ♣Вххх (контракт 4♥)
Причина: Из ТК всегда ходим К

❌ НЕ ходим Т♣ (сохраняем):
♠Дхх ♥Кхх ♦Вхх ♣Тххх (контракт 4♠)
Причина: Тххх можем сохранить для перебития их онеров
```

## Выход против БК контрактов

### Стратегия: Длинная масть партнера или своя

**Приоритет выбора масти:**
1. **Масть партнера** (если заявлял)
2. **Своя длинная масть** (5+ карт)
3. **Безопасная масть** (из последовательности)

### Карта выхода против БК

> [!tip]- Правило "4-й по старшинству" (4th best)
> Против БК из длинной масти (4+ карты) выходим **4-й картой по старшинству**.
> 
> **Пример:** ♠КВ852 → ходим 5♠ (4-я карта)
> **Пример:** ♠Д10743 → ходим 4♠ (4-я карта)

**Исключения:**
- Из последовательности → верхняя карта
- Из 3 карт → верхняя из мелких (top of nothing)
- Туз → только с Тх/Тхх (дуплет/триплет)

### Примеры выхода против 3NT

```
Торговля: 1NT - 3NT

♠КВ852 ♥Дх ♦10хх ♣ххх
→ Ходим 5♠ (4-я по старшинству из длинной масти)

♠Кхх ♥Д10874 ♦Вх ♣ххх
→ Ходим 7♥ (4-я по старшинству)

♠ТКД10 ♥хххх ♦Вхх ♣хх
→ Ходим Т♠ (из последовательности - верхняя)

♠Кхх ♥Дхх ♦В1074 ♣хх
→ Ходим 10♦ (из последовательности В10)

♠Кх ♥Дхх ♦Вхххх ♣хх
→ Ходим В♦ (из 5 карт верхняя из мелких)
```

## Выход против козырных контрактов

### Стратегия: Безопасность или активность?

**Активный выход (когда наша сила):**
- У нас есть взятки (тузы, короли)
- Нужно взять быстрые взятки ДО того, как разыгрывающий установит козыри
- Ходим от фигур

**Пассивный выход (когда их сила):**
- У них большая сила
- Ходим безопасно, не отдаем лишних взяток
- Ходим козырем или из пустых мастей

### Таблица выбора масти против козырного

| Ситуация | Масть для выхода |
|----------|------------------|
| **Партнер заявлял** | Его масть (почти всегда) |
| **У нас синглет** | Синглет (на перебитие) |
| **У нас ТКх/КДх** | Эта масть (быстрые взятки) |
| **Их длинная масть** | НЕ ходим туда |
| **Ничего хорошего** | Козырь (пассивно) |

### Карта выхода против козырного

> [!tip]- Правило выбора карты
> - **Из онеров:** Верхняя из последовательности (Т, К, Д из КД)
> - **Из мелких:** Мелкая карта (4-я, 2-я или мелкая из 3)
> - **Синглет:** Синглет
> - **Дуплет:** Верхняя (doubleton top)

**Таблица:**

| Холдинг | Карта выхода | Значение |
|---------|--------------|----------|
| **Тх** | Т | Дуплет, прошу переключения или сигнала |
| **Кх** | Мелкая (х) | Из Кх ходим мелкой! |
| **Кхх** | Мелкая | Из Кхх ходим мелкой! |
| **КДх** | К | Из КД ходим К |
| **Дхх** | Мелкая | Из Дхх ходим мелкой! |
| **Вхх** | Мелкая | Из мелких ходим мелкой |
| **хх** | Верхняя | Дуплет - верхняя карта |
| **хххх** | 4-я или 2-я | Из 4+ мелких карт |
| **х** | Синглет | Синглет |

### Примеры выхода против 4♠

```
Торговля: 1♠ - 4♠

♠хх ♥КДхх ♦Тхх ♣хххх
→ Ходим К♥ (из КД - быстрые взятки)

♠хх ♥Кхх ♦Дххх ♣Тххх
→ Ходим Т♣ (безопасный выход из Т)

♠хх ♥Дхх ♦В10хх ♣хххх
→ Ходим В♦ (из последовательности В10)

♠х ♥Кххх ♦Дхххх ♣Вхх
→ Ходим х♠ (синглет козыря на перебитие!)

♠ххх ♥Кхх ♦Дхх ♣хххх
→ Ходим х♠ (козырь - пассивный выход)
```

## Детальный разбор выходов

### Выход из онеров (фигур)

#### 1. Туз (Т)

**Правило:**
- **Против козыря:** Тх/Тхх → ходим Т (иначе может пропасть)
- **Против БК:** НЕ ходим с Т (потеряем взятки в масти)
- **ТК+:** Ходим К (не Т!)

**Примеры:**

```
Против 4♥:
✅ ♠Тх ♥хх ♦Дххх ♣Кхххх → Т♠ (дуплет может пропасть)
✅ ♠Тхх ♥Кх ♦Дххх ♣хххх → Т♠ (триплет, боимся потерять)
❌ ♠Тхххх ♥Кх ♦Дхх ♣ххх → 4♠ (НЕ ходим с Т из 5 карт!)

Против 3NT:
❌ ♠Тхххх ♥Кх ♦Дхх ♣ххх → 4♠ (НЕ с Т против БК!)
✅ ♠ТКхх ♥Дх ♦хххх ♣хх → К♠ (из ТК ходим К)
```

#### 2. Король (К)

**Правило:**
- **К ОБЕЩАЕТ Д** (или Т)
- Из Кх/Кхх НЕ ходим королем!

**Примеры:**

```
✅ ♠КДхх ♥Тх ♦хххх ♣Вхх → К♠ (из КД)
✅ ♠ТКхх ♥Дх ♦хххх ♣Вхх → К♠ (из ТК)
✅ ♠КД10х ♥Тх ♦хххх ♣хх → К♠ (из КД10)

❌ ♠Кхх ♥Тх ♦Дхххх ♣Вхх → 2♠ (из Кхх мелкая!)
❌ ♠Кх ♥Дхх ♦Тхххх ♣хх → х♠ (из Кх мелкая!)
```

#### 3. Дама (Д)

**Правило:**
- **Д ОБЕЩАЕТ 10** (или К)
- Из Дх/Дхх НЕ ходим дамой!

**Примеры:**

```
✅ ♠Д10хх ♥Кх ♦Тхх ♣хххх → Д♠ (из Д10)
✅ ♠ДВ9х ♥Кх ♦Тхх ♣хххх → Д♠ (из ДВ9)
✅ ♠КДхх ♥Тх ♦хххх ♣Вх → К♠ (из КД ходим К!)

❌ ♠Дхх ♥Кх ♦Тхххх ♣Вхх → 2♠ (из Дхх мелкая!)
❌ ♠Дх ♥Кхх ♦Тхххх ♣Вх → х♠ (из Дх мелкая!)
```

#### 4. Валет (В)

**Правило:**
- **В ОБЕЩАЕТ 10**
- Из Вх/Вхх НЕ ходим валетом!

**Примеры:**

```
✅ ♠В10хх ♥Кх ♦Дххх ♣ххх → В♠ (из В10)
✅ ♠В109х ♥Кх ♦Дххх ♣хх → В♠ (из В109)

❌ ♠Вхх ♥Кх ♦Дхххх ♣Вхх → 2♠ (из Вхх мелкая!)
```

## Выход из мелких карт

### Правило "4-й по старшинству" (против БК)

**Применение:** Из 4+ карт без последовательности

**Как считать:** Считаем карты сверху вниз

```
♠К8652 → считаем: К(1), 8(2), 6(3), 5(4) → ходим 5
♠Д10743 → считаем: Д(1), 10(2), 7(3), 4(4) → ходим 4
♠98542 → считаем: 9(1), 8(2), 5(3), 4(4) → ходим 4
```

### Правило "2-й по старшинству" (против козыря)

**Применение:** Из 4+ карт без последовательности против козырного контракта

**Как считать:** Считаем карты сверху вниз, берем 2-ю

```
♠К852 → считаем: К(1), 8(2) → ходим 8
♠Д1074 → считаем: Д(1), 10(2) → ходим 10
♠9852 → считаем: 9(1), 8(2) → ходим 8
```

### Выход из 3 карт

**Правило:** Верхняя из мелких (МВМ - Мелкая-Высокая-Мелкая)

```
♠962 → ходим 9 (верхняя из мелких)
♠853 → ходим 8 (верхняя из мелких)
♠742 → ходим 7 (верхняя из мелких)

Исключение:
♠Кхх → ходим мелкая (не К!)
♠Дхх → ходим мелкая (не Д!)
```

### Выход из дуплета (2 карты)

**Правило:** Верхняя карта

```
♠К2 → ходим 2 (из Кх ходим мелкая!)
♠Д4 → ходим 4 (из Дх ходим мелкая!)
♠94 → ходим 9 (дуплет - верхняя)
♠52 → ходим 5 (дуплет - верхняя)
```

> [!note]- Как партнер узнает про дуплет?
> **МВМ (Мелкая-Высокая-Мелкая):**
> - 1-й ход: 9♠ (высокая)
> - 2-й ход: 4♠ (мелкая)
> - Партнер понимает: это был дуплет!

## Правило 11 (против БК)

> [!tip]- Правило 11
> **Применение:** Партнер вышел мелкой картой против БК (4-я по старшинству)
> 
> **Формула:** 11 - (карта выхода) = количество карт старше у других игроков
> 
> **Цель:** Определить, какую карту положить из болвана

### Как применять Правило 11

**Шаги:**
1. Партнер вышел мелкой (например, 5)
2. Считаем: 11 - 5 = 6 карт старше пятерки у других
3. Видим болван + свою руку
4. Определяем, сколько карт у разыгрывающего
5. Решаем, какую карту класть

### Примеры Правила 11

**Пример 1:**
```
Партнер вышел 5♠

Болван: ♠К82 (карты старше 5: К,8 - это 2 карты)
Мы: ♠Д104 (карты старше 5: Д,10,8 - это 3 карты)

Расчет:
11 - 5 = 6 карт старше пятерки у ВСЕХ
Видим: 2 (болван) + 3 (мы) = 5
Значит у разыгрывающего: 6 - 5 = 1 карта старше (это Т!)

Решение: Кладем 4♠ из болвана (экономим К и Д)
```

**Пример 2:**
```
Партнер вышел 7♦

Болван: ♦К93 (карты старше 7: К,9 - это 2 карты)
Мы: ♦Д1082 (карты старше 7: Д,10,8 - это 3 карты)

Расчет:
11 - 7 = 4 карты старше семерки
Видим: 2 (болван) + 3 (мы) = 5... это больше 4!

Значит: Партнер вышел НЕ 4-й по старшинству (возможно, дуплет или МВМ)

Решение: Кладем К♦ из болвана (берем взятку)
```

## Специальные ситуации

### 1. Выход синглетом (против козыря)

**Когда ходить синглетом?**
- ✅ У вас короткий козырь (2-3 карты) - можете перебить
- ✅ Партнер заявлял эту масть - у него фигуры
- ❌ У вас длинный козырь (4+) - не нужно перебивать
- ❌ Синглет в козырной масти противников - потеря темпа

**Примеры:**

```
Контракт 4♥, у вас:

✅ ♠х ♥Кхх ♦Дхххх ♣Вххх
→ х♠ (синглет, короткий козырь, можем перебить)

✅ ♠х ♥КД ♦хххх ♣Вхххх (партнер заявлял ♠)
→ х♠ (синглет в масти партнера)

❌ ♠х ♥КДхх ♦Дхххх ♣Вхх
→ НЕ ходим х♠ (длинный козырь ♥КДхх, перебивать не нужно)
→ Ходим К♥ или Д♦
```

### 2. Выход козырем

**Когда ходить козырем?**
- ✅ У вас плоская рука без длинных мастей
- ✅ Противники торговали много мастей (болван будет раскладный)
- ✅ Вы хотите уменьшить возможности перебиток у болвана
- ❌ У вас есть хорошая масть для атаки

**Примеры:**

```
Контракт 4♠, у вас:

✅ ♠Кхх ♥Дхх ♦Вхх ♣хххх (плоская рука)
→ ходим х♠ (козырь - пассивный выход)

✅ ♠Дхх ♥Кхх ♦Вхх ♣хххх (торговля: 1♠-2♣-2♦-4♠)
→ ходим х♠ (они показали все масти - болван раскладный)

❌ ♠Дхх ♥КДхх ♦Вхх ♣ххх (есть хорошая ♥)
→ НЕ ходим козырем, ходим К♥ (активно)
```

### 3. Выход от короля в масти партнера

**Ситуация:** Партнер заявлял масть, у нас Кх/Кхх

**Правило:**
- **Кх** → ходим мелкую (не К!)
- **Кхх** → ходим мелкую
- **Кхххх** → ходим 4-ю/2-ю (как обычно)
- **КД+** → ходим К

**Примеры:**

```
Торговля: 1♦ (партнер) - 1♠ - 2♠ - 4♠

♠хх ♥Кх ♦Дхххх ♣Вххх
→ ходим х♥ (из Кх в масти партнера - мелкая!)

♠хх ♥Кхх ♦Дхххх ♣Вхх
→ ходим 2♥ (из Кхх - мелкая!)

♠хх ♥КДхх ♦Тхх ♣хххх
→ ходим К♥ (из КД - всегда К)
```

## Ситуационный выбор хода

### Ситуация 1: Партнер заявлял масть

**Правило:** Почти ВСЕГДА ходим в масть партнера

**Исключения:**
- У вас синглет в другой масти и короткий козырь
- У вас ТК в своей масти (быстрые взятки)

**Примеры:**

```
Торговля: 1♥ (партнер) - 2♦ - 4♦

♠Кхх ♥Дхх ♦хх ♣Вхххх
→ ходим 2♥ (масть партнера, мелкая из Дхх)

♠х ♥Дхх ♦Кхх ♣Вхххх
→ ходим х♠ (синглет лучше, чем масть партнера)
```

### Ситуация 2: Они торговали одну масть (4М контракт)

**Правило:** НЕ ходим в их масть

**Стратегия:**
- Ходим безопасно в незаявленную масть
- Или козырем (пассивно)

**Примеры:**

```
Торговля: 1♠ - 2NT - 3♠ - 4♠ (только ♠ заявили)

♠хх ♥Кххх ♦Дхх ♣Вххх
→ ходим К♥ (незаявленная масть, активно)

♠Дхх ♥Вххх ♦10хх ♣Кхх
→ ходим х♠ (козырь - пассивно)
```

### Ситуация 3: Быстрый гейм (1♠ - 4♠)

**Правило:** Агрессивный выход! Они прячут слабость.

**Стратегия:**
- Ходим от фигур
- Пытаемся взять быстрые взятки
- НЕ ходим пассивно

**Примеры:**

```
Торговля: 1♠ - 4♠ (быстрый гейм!)

♠хх ♥КДхх ♦Тх ♣хххх
→ ходим К♥ (агрессивно, быстрые взятки!)

♠хх ♥Тх ♦КДхх ♣хххх
→ ходим К♦ (из КД)

❌ ♠хх ♥Кхх ♦Дххх ♣Вххх
→ НЕ ходим козырем! Ходим К♥ или Д♦
```

### Ситуация 4: Медленная торговля до гейма

**Правило:** Пассивный выход. У них сила, не отдаем лишнего.

**Стратегия:**
- Ходим безопасно
- Козырь - хороший вариант
- Избегаем ходов от фигур без последовательности

**Примеры:**

```
Торговля: 1♠ - 2♣ - 2♦ - 2♥ - 2♠ - 3♠ - 4♠ (долгая торговля)

♠хх ♥Кхх ♦Дххх ♣Вххх
→ ходим х♠ (козырь - безопасно)

♠Дхх ♥Вхх ♦10хх ♣Кхх
→ ходим х♠ (козырь - безопасно)
```

## Чек-лист первого хода

### ✅ Перед ходом ответь на вопросы:

**1. Какой контракт?**
- [ ] БК (NT) → длинная масть
- [ ] Козырной → безопасный ход

**2. Что говорила торговля?**
- [ ] Партнер заявлял масть? → его масть
- [ ] Противники заявляли масти? → не туда
- [ ] Быстрый гейм? → активно
- [ ] Медленная торговля? → пассивно

**3. Что у меня в руке?**
- [ ] Синглет + короткий козырь? → синглет
- [ ] Последовательность (КД, Д10, В10)? → верхняя карта
- [ ] Длинная масть (5+)? → 4-я (БК) или 2-я (козырь)
- [ ] Плоская рука? → козырь (пассивно)

**4. Какую конкретно карту?**
- [ ] Из последовательности → верхняя
- [ ] Из Кх/Дх → мелкая (не фигура!)
- [ ] Из 4+ карт → 4-я (БК) или 2-я (козырь)
- [ ] Из 3 карт → верхняя из мелких (МВМ)
- [ ] Из 2 карт → верхняя
- [ ] Синглет → синглет

## Памятка: Золотые правила первого хода

1. **Т обещает К** (или дуплет/триплет)
2. **К обещает Д** (или Т)
3. **Д обещает 10** (или К)
4. **В обещает 10**
5. **Из Кх/Дх/Вх** → ходим МЕЛКОЙ (не фигурой!)
6. **Против БК** → 4-я по старшинству из длинной
7. **Против козыря** → 2-я по старшинству или безопасно
8. **МВМ** (Мелкая-Высокая-Мелкая) из 3 карт
9. **Дуплет** → верхняя карта
10. **Синглет** → синглет (если есть смысл)
