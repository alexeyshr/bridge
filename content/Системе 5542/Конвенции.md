## Классификация конвенций

### По фазе игры:

1. **Односторонняя торговля** - мы торгуемся без помех
2. **Конкурентная торговля** - противники вмешиваются
3. **Шлемовая торговля** - исследование шлема
4. **Защита** - сигналы и первый ход

### По типу:

- **Натуральные** - показывают масть и силу
- **Искусственные** - условное значение (не про масть)
- **Вопросы** - спрашивают информацию у партнера
- **Ответы** - отвечают на вопрос партнера

# 1. Конвенции односторонней торговли

## 1.1. Стейман (Stayman)

**Заявка:** 1NT - **2♣**

**Тип:** Искусственная, вопрос

**Значение:** "У тебя есть 4-карточный мажор?"

**Когда применять:**

- ✓ У вас есть 4М (или 44 в мажорах)
- ✓ Хотите найти фит 4-4 в мажоре
- ✗ НЕ применяйте с 5М (используйте трансфер!)

**Ответы:**

- **2♦** = нет 4М
- **2♥** = есть 4♥ (может быть 4♠)
- **2♠** = есть 4♠, нет 4♥

**Примеры:**

```
✅ Применяем Стейман:
Север: 1NT (15-17hcp)
Юг: 2♣ (♠Кххх ♥Дххх ♦Тх ♣Вхх, 10 очков, 44 мажоры)
Север: 2♥ (есть 4♥)
Юг: 4♥ (гейм с фитом)

✅ Стейман, потом Смолен:
Север: 1NT
Юг: 2♣ (♠ТКххх ♥Дххх ♦хх ♣Дх, 11 очков, 5♠4♥)
Север: 2♦ (нет 4М)
Юг: 3♥ (Смолен: 4♥5+♠)
Север: 3♠ (выбираю ♠)
Юг: 4♠
```

**Детали:** → [[5.1. Стейман]]

## 1.2. Трансферы в мажоры

**Заявки:**

- 1NT - **2♦** = трансфер в ♥
- 1NT - **2♥** = трансфер в ♠

**Тип:** Искусственная, показ масти

**Значение:** "У меня 5+ карт в мажоре, переведи контракт на свою руку"

**Когда применять:**

- ✓ У вас 5+ карт в мажоре
- ✓ Любая сила (от 0 до 18+hcp)
- ✓ Хотите, чтобы сильная рука была разыгрывающей

**Ответы открывающего:**

- **Обычный прием** (2♥/2♠) = 2-3 карты в масти
- **Суперприем** (прыжок) = 4 карты + максимум (16-17hcp)

**Варианты суперприема (после 2♦):**

- **2NT** = 4♥, максимум, нет дуплетов (4333)
- **3♣/♦/♠** = 4♥, максимум, дуплет в масти
- **3♥** = 4♥, минимум (15hcp)

**Примеры:**

```
✅ Простой трансфер (слабая рука):
Север: 1NT
Юг: 2♦ (♠хх ♥Дхххх ♦Кхх ♣хх, 5 очков, 5♥)
Север: 2♥ (принимаю)
Юг: Пас (к игре)

✅ Трансфер + инвит:
Север: 1NT
Юг: 2♦ (♠хх ♥ТКххх ♦Дх ♣Вххх, 9 очков, 5♥)
Север: 2♥
Юг: 2NT (инвит, ровно 5♥)
Север: 3♥ (принимаю с максимумом)

✅ Суперприем с дуплетом:
Север: 1NT (♠Кх ♥ТКхх ♦ТДхх ♣Дхх, 17 очков, 4♥, дуплет ♠)
Юг: 2♦ (♠Дххх ♥ДВххх ♦Кх ♣хх, 10 очков, 5♥)
Север: 3♠ (суперприем! 4♥, максимум, дуплет ♠)
Юг: 4♥ (отлично, идем в гейм)
```

**Детали:** → [[5.2. Трансферы в мажоры]]

## 1.3. Трансферы в миноры

**Заявки:**

- 1NT - **2♠** = трансфер в ♣
- 1NT - **2NT** = трансфер в ♦

**Тип:** Искусственная, показ масти

**Значение:**

- "У меня 6+ карт в миноре" (слабая рука)
- ИЛИ "Инвит с 5m и 8-9hcp"
- ИЛИ "Слабые миноры 55" (только 2NT)

**Когда применять:**

- ✓ У вас 6+m (хотите уйти из 1NT)
- ✓ Инвит с 5m и 8-9hcp
- ✓ Слабые миноры 55 (убежать из 1NT)

**Ответы:** Открывающий ОБЯЗАН принять на 3m

**Примеры:**

```
✅ Слабая рука с длинным минором:
Север: 1NT
Юг: 2♠ (♠хх ♥Кх ♦хх ♣ДВхххх, 5 очков, 6♣)
Север: 3♣ (принимаю)
Юг: Пас (к игре)

✅ Инвит с 5♣4♠:
Север: 1NT
Юг: 2♠ (♠Дххх ♥Кх ♦хх ♣КДххх, 9 очков, 4♠5♣)
Север: 3♣ (принимаю)
Юг: 3♠ (инвит, 5♣4♠)
Север: 4♠ (есть фит, принимаю)

✅ Слабые миноры:
Север: 1NT
Юг: 2NT (♠хх ♥х ♦Вхххх ♣Кхххх, 4 очка, 55 миноры)
Север: 3♦ (принимаю)
Юг: Пас (играем где принял)
```

**Детали:** → [[5.3. Трансферы в миноры]]

## 1.4. Газилли (Gazilli)

**Заявки:**

- 1♥ - 1♠ - **2♣**
- 1♥ - 1NT - **2♣**
- 1♠ - 1NT - **2♣**

**Тип:** Искусственная (или полунатуральная)

**Значение:**

- Любая рука 16+hcp (все силы)
- ИЛИ 11-15hcp с 5М4♣ (натуральная)

**Цель:** Разделить руки по зонам силы

**Зоны силы:**

- **Зона 1 (11-15):** Масти на 2 уровне
- **Зона 2 (15-17):** Масти на 3 уровне (инвит)
- **Зона 3 (18+):** Прямые GF заявки или 2NT (10 карт)

**Схема после 1♥-1♠-2♣-2♦ (waiting):**

| Ребид   | Зона | Значение            |
| ------- | ---- | ------------------- |
| **2♥**  | 1    | 11-15hcp, 5♥4♣      |
| **2♠**  | 3    | 16+hcp, 3+♠         |
| **2NT** | 2    | 15-17hcp, 3♠5♥(32)  |
| **3♣**  | 3    | 16+hcp, 5♥4+♣       |
| **3♦**  | 3    | 16+hcp, 5♥5+♦       |
| **3♥**  | 2    | 15-17hcp, 6+♥       |
| **3♠**  | 2    | 15-17hcp, 4♠5♥22    |
| **3NT** | 2    | 15-17hcp, 2♠5♥(332) |

**Примеры:**

```
✅ Зона 1 (11-15hcp):
Север: 1♥
Юг: 1♠ (♠ДВ10хх ♥Дх ♦Тхх ♣Вхх, 11 очков)
Север: 2♣ (♠Кх ♥ТКххх ♦хх ♣КДхх, 13 очков, Газилли)
Юг: 2♦ (waiting, 8+hcp)
Север: 2♥ (зона 1, 11-15hcp, 5♥4♣)
Юг: 2♠ (предпочтение пике)
Север: Пас

✅ Зона 3 (16+hcp) с фитом:
Север: 1♥
Юг: 1♠ (♠ДВхх ♥Кх ♦Дхх ♣Вххх, 10 очков)
Север: 2♣ (♠ТКхх ♥ТДВхх ♦Тх ♣Кх, 18 очков, Газилли)
Юг: 2♦ (waiting)
Север: 2♠ (зона 3, 16+hcp, 3+♠!)
Юг: 3♠ (фит, хорошая рука)
Север: 4♠ (гейм)
```

**Детали:** → [[3.1. Секвенция 1♥-1♠]], [[3.2. Секвенция 1♥-1NT]], [[4.1. Секвенция 1♠-1NT]]

## 1.5. Друри (Drury)

**Заявка:** пас-пас - 1М - **2♣**

**Тип:** Искусственная, показ силы

**Значение:** "У меня максимум паса (6-10hcp) с фитом 3+М"

**Когда применять:**

- ✓ ТОЛЬКО на 3-4 руке (после паса)
- ✓ Партнер открыл 1М на 3-4 руке
- ✓ У вас 6-10hcp и фит 3-4М

> [!warning]- ВАЖНО!
> Если открытие было на 1-2 руке, 2♣ = обычная GF заявка (НЕ Друри)!

**Ответы открывающего:**

| Заявка  | Значение                         |
| ------- | -------------------------------- |
| **2♦**  | Нет гейма, пасуй на любую заявку |
| **2М**  | F1, инвит, есть интерес к гейму  |
| **2OM** | F1, инвит, 4+OM 5М               |
| **2NT** | Инвит, 13-14hcp, 5332            |
| **3m**  | F1, инвит, 4+m 5М                |
| **3М**  | Инвит, 6+М                       |
| **3OM** | GF, Сплинтер, краткость OM       |
| **3NT** | НФ, на выбор, 5332               |
| **4m**  | GF, Сплинтер, краткость m        |
| **4М**  | G, к игре                        |

**Примеры:**

```
✅ Друри, нет гейма:
Торговля: пас - пас - 1♥ (Север) - 2♣ (Юг, Друри)
Север: 2♦ (♠Кхх ♥ДВххх ♦Тхх ♣хх, 10 очков, легкое открытие, нет гейма)
Юг: Пас

✅ Друри, инвит принят:
Торговля: пас - пас - 1♥ (Север) - 2♣ (Юг, Друри)
Север: 2♥ (♠Кхх ♥ТКххх ♦Дхх ♣Вх, 12 очков, инвит)
Юг: 4♥ (♠Дх ♥Дххх ♦Кхх ♣Тхх, 10 очков, максимум паса!)

✅ Друри, сплинтер:
Торговля: пас - пас - 1♠ (Север) - 2♣ (Юг)
Север: 3♥ (♠ТКДхх ♥х ♦Кхх ♣ТДхх, 16 очков, сплинтер)
Юг: 4♠ (♠Дххх ♥Кхх ♦Тхх ♣Вхх, 10 очков, идем в гейм)
```

**Детали:** → [[3.3. Секвенция 1♥-2♣]], [[4.2. Секвенция 1♠-2♣]]

## 1.6. Инвертированные миноры (Inverted Minors)

**Заявки:**

- 1♣ - **2♣**
- 1♦ - **2♦**

**Тип:** Искусственная/натуральная, форсирующая

**Значение:** "У меня 10+hcp и фит 4+ в твоем миноре"

**Цель:** Эффективно исследовать 3NT или 5m, показывая держки

**Когда применять:**

- ✓ 10+hcp
- ✓ Фит 4+ в миноре открывающего
- ✗ Нет 4М (иначе называем мажор)

**Ответы открывающего (1♦-2♦):**

| Заявка  | Значение                                 |
| ------- | ---------------------------------------- |
| **2♥**  | Ценности в ♥, нет держки ♠               |
| **2♠**  | Ценности в ♠, нет держки ♥               |
| **2NT** | Держки в обоих мажорах, не минимум (13+) |
| **3♣**  | Держка в ♣                               |
| **3♦**  | НФ, минимум или нет держек               |
| **3М**  | GF, Сплинтер, краткость М, 6+♦           |
| **3NT** | G, есть все держки                       |

**Примеры:**

```
✅ Поиск 3NT:
Север: 1♦
Юг: 2♦ (♠ТКх ♥Дхх ♦КВхх ♣Дхх, 12 очков, фит ♦)
Север: 2♥ (♠хх ♥ТКх ♦ТДххх ♣Кхх, 14 очков, ценности ♥, нет держки ♠)
Юг: 3NT (держка ♠, к игре)

✅ Игра в миноре:
Север: 1♦
Юг: 2♦ (♠Кхх ♥Дхх ♦ТВхх ♣хх, 10 очков, фит ♦)
Север: 3♦ (♠хх ♥хх ♦ТКДххх ♣Кхх, 11 очков, минимум)
Юг: Пас (играем 3♦)

✅ Сплинтер к шлему:
Север: 1♦
Юг: 2♦ (♠Тхх ♥Дхх ♦ТКхх ♣Кхх, 14 очков)
Север: 3♥ (♠Кх ♥х ♦ТКВххх ♣ТДх, 16 очков, сплинтер ♥)
Юг: 4♦ (хорошая рука)
Север: 4NT (Блэквуд)
Юг: 5♦ (1 туз)
Север: 6♦
```

**Детали:** → [[2.5. Секвенция 1♦-2♦]]

## 1.7. Смолен (Smolen)

**Заявки:** 1NT - 2♣ - 2♦ - **3♥/3♠**

**Тип:** Искусственная, показ распределения

**Значение:**

- **3♥** = 4♥ и 5+♠ (показываем КОРОТКИЙ мажор!)
- **3♠** = 4♠ и 5+♥

**Когда применять:**

- ✓ Спросили Стейман
- ✓ Партнер ответил 2♦ (нет 4М)
- ✓ У вас 4М и 5+OM (другой мажор)
- ✓ GF сила (10+hcp)

**Цель:** Найти фит 4-4 ИЛИ фит 5-3, причем сильная рука будет разыгрывающей

**Примеры:**

```
✅ Смолен 3♥ (4♥5♠):
Север: 1NT
Юг: 2♣ (♠ТКххх ♥Дххх ♦хх ♣Дх, 11 очков, 5♠4♥)
Север: 2♦ (нет 4М)
Юг: 3♥ (Смолен: 4♥5+♠)
Север: 3♠ (♠Кхх ♥Дхх ♦ТВхх ♣ТКх, выбираю ♠)
Юг: 4♠

✅ Смолен 3♠ (4♠5♥):
Север: 1NT
Юг: 2♣ (♠Кххх ♥ТКххх ♦хх ♣Дх, 11 очков, 5♥4♠)
Север: 2♦
Юг: 3♠ (Смолен: 4♠5+♥)
Север: 4♥ (♠Дхх ♥Кхх ♦ТВхх ♣ТКх, выбираю ♥)
```

**Детали:** → [[5.1. Стейман]]

## 1.8. Сплинтеры (Splinters)

**Заявки:** Прыжковая заявка в новой масти после фита

**Тип:** Искусственная, показ краткости

**Значение:** "У меня фит в твоей масти + синглет/ренонс в заявленной масти + шлемовые амбиции"

**Варианты:**

- **Прямые:** 1♥ - 3♠/4♣/4♦ или 1♠ - 3♥/4♣/4♦
- **Автосплинтеры:** 1NT-2♦-2♥-3♠/4♣/4♦

**Требования:**

- ✓ Фит 4+ в масти партнера
- ✓ Синглет или ренонс в заявленной масти
- ✓ 11-15hcp (раскладные руки)

**Примеры:**

```
✅ Прямой сплинтер:
Север: 1♠
Юг: 4♣ (♠Дххх ♥Тххх ♦Кх ♣х, 11 очков, синглет ♣, фит 4♠)
Север: 4NT (♠ТКДхх ♥Кхх ♦ТВхх ♣х, 17 очков, у меня тоже синглет ♣!)
Юг: 5♦ (1 туз)
Север: 6♠

✅ Автосплинтер после трансфера:
Север: 1NT
Юг: 2♦ (♠х ♥КВхххх ♦Дх ♣ТКхх, 14 очков, 6♥, трансфер)
Север: 2♥
Юг: 3♠ (автосплинтер, краткость ♠, 6♥)
Север: 4♣ (♠Кхх ♥ТДхх ♦ТКх ♣Дхх, кюбид, интерес к шлему)
Юг: 4♦ (кюбид)
Север: 4NT (RKCB)
Юг: 5♠ (2 ключевых с Д♥)
Север: 6♥
```

**Детали:** → [[Шлемовая торговля]]

## 1.9. 2wcb (Double Check-Back)

**Заявка:** 1m - 1М - 1NT - **2♣**

**Тип:** Искусственная, вопрос

**Значение:** "Проверяю: есть ли у тебя фит в моем мажоре или 4 карты в другом мажоре?"

**Когда применять:**

- ✓ Открывающий ответил 1NT
- ✓ У вас 5М или 4-4 в мажорах
- ✓ Хотите проверить фит

**Ответы открывающего:**

| Заявка  | Значение                            |
| ------- | ----------------------------------- |
| **2♦**  | Нет 3М, нет 4OM                     |
| **2М**  | Есть 3М (фит в вашем мажоре)        |
| **2OM** | Есть 4OM (четверка в другом мажоре) |
| **2NT** | Нет фита, максимум                  |
| **3М**  | Фит 4М, максимум                    |

**Примеры:**

```
✅ Нашли фит 5-3:
Север: 1♦
Юг: 1♥ (♠Кхх ♥ТКххх ♦Дх ♣Вхх, 11 очков, 5♥)
Север: 1NT (♠Дхх ♥Кхх ♦ТВхх ♣Кхх, 13 очков, 3♥)
Юг: 2♣ (2wcb, проверяю фит)
Север: 2♥ (есть 3♥!)
Юг: 4♥ (гейм с фитом 5-3)

✅ Нашли второй мажор 4-4:
Север: 1♦
Юг: 1♥ (♠Кххх ♥ТКхх ♦Дх ♣Вхх, 11 очков, 4♥4♠)
Север: 1NT (♠ДВхх ♥хх ♦ТКхх ♣Кхх, 13 очков, 4♠)
Юг: 2♣ (2wcb)
Север: 2♠ (есть 4♠!)
Юг: 4♠ (гейм с фитом 4-4)

✅ Не нашли фит:
Север: 1♦
Юг: 1♥ (♠Кхх ♥ТКххх ♦Дх ♣Вхх, 11 очков)
Север: 1NT (♠Дхх ♥хх ♦ТКхх ♣Кххх, 13 очков, нет 3♥)
Юг: 2♣ (2wcb)
Север: 2♦ (нет фита)
Юг: 3NT (играем БК)
```

## 1.10. Fourth Suit Forcing (4SF)

**Заявка:** 1♣-1♥-1♠-**2♦** (четвертая масть)

**Тип:** Искусственная, форсирующая

**Значение:** "У меня нет поддержки твоих мастей, у меня нет 5♥, нужно время для описания руки"

**Когда применять:**

- ✓ Партнер открыл и ответил, показав 2 масти
- ✓ У вас нет фита в его мастях
- ✓ У вас нет 5+ карт в своей масти
- ✓ Нужно время для описания (GF или близко к GF)

**Цель:** Создать форсирующую ситуацию, чтобы описать свою руку

**Ответы открывающего:**

- **2NT** = нет 3М, равномерная, держка в 4-й масти
- **3♣** = 6+♣, минимум
- **3♥** = 3♥ (фит запаздывающий)
- **3♠** = 6+♠, минимум
- **4♥** = 4♥, хорошая рука

**Примеры:**

```
✅ Fourth Suit Forcing:
Север: 1♦
Юг: 1♥ (♠Дхх ♥ТКхх ♦хх ♣Кххх, 11 очков, 4♥)
Север: 1♠ (♠ТКхх ♥хх ♦ТКххх ♣Дх, 15 очков, 4♠5♦)
Юг: 2♣ (4SF! Нет фита ♠, нет 5♥, нужно время)
Север: 2NT (равномерная, держка ♣)
Юг: 3NT (играем БК)

✅ 4SF находит запаздывающий фит:
Север: 1♣
Юг: 1♥ (♠Кх ♥ТДххх ♦Дхх ♣хх, 10 очков)
Север: 1♠ (♠ДВхх ♥Кхх ♦хх ♣ТКДх, 15 очков, 3♥!)
Юг: 2♦ (4SF)
Север: 3♥ (есть 3♥, запаздывающий фит!)
Юг: 4♥ (отлично!)
```

## 1.11. New Minor Forcing (NMF)

**Заявка:** 1♦-1М-1NT-**2♣** или 1♣-1М-1NT-**2♦**

**Тип:** Искусственная, форсирующая

**Значение:** "Проверяю: есть ли у тебя 3М или 4OM?"

**Аналог:** Похож на 2wcb, но через ДРУГОЙ минор

**Когда применять:**

- ✓ Открывающий ответил 1NT
- ✓ У вас инвит или сильнее (9+hcp)
- ✓ Проверяете фит 5-3 или 4-4 в мажорах

**Ответы:**

| Заявка  | Значение          |
| ------- | ----------------- |
| **2М**  | Есть 3М (фит)     |
| **2OM** | Есть 4OM          |
| **2NT** | Нет фита, минимум |
| **3М**  | Есть 3М, максимум |

**Примеры:**

```
✅ NMF находит фит 5-3:
Север: 1♣
Юг: 1♥ (♠Кхх ♥ТКххх ♦Дх ♣Вхх, 11 очков, 5♥)
Север: 1NT (♠Дхх ♥Кхх ♦ТВхх ♣Кхх, 13 очков, 3♥)
Юг: 2♦ (NMF, проверяю фит)
Север: 2♥ (есть 3♥!)
Юг: 4♥ (гейм)

✅ NMF находит второй мажор:
Север: 1♦
Юг: 1♥ (♠Кххх ♥ТКхх ♦Дх ♣Вхх, 11 очков, 4♥4♠)
Север: 1NT (♠ДВхх ♥хх ♦ТКхх ♣Кхх, 13 очков, 4♠)
Юг: 2♣ (NMF)
Север: 2♠ (есть 4♠!)
Юг: 4♠
```

## 1.12. Реверс (Reverse)

**Заявка:** 1♣-1♥-**2♦** или 1♣-1♠-**2♥** (старшая масть на 2 уровне)

**Тип:** Натуральная, сильная

**Значение:** "У меня 16+hcp, две масти, первая масть длиннее"

**Определение:** Заявка старшей масти на 2 уровне (выше первой масти открывающего)

**Правила реверса:**

- ✓ 16+hcp (сильная рука)
- ✓ Первая масть длиннее или равна (5+)
- ✓ Вторая масть 4+
- ✓ Форсирует на 1 круг

**Примеры:**

```
✅ Реверс (сильная рука):
Север: 1♣ (♠Тх ♥ТКхх ♦Дх ♣КДВхх, 17 очков, 4♥5♣)
Юг: 1♠ (♠ДВххх ♥хх ♦Кхх ♣хх, 8 очков)
Север: 2♥ (РЕВЕРС! 16+hcp, 5♣4♥)
Юг: 2NT (минимальный ответ)
Север: 3NT

✅ Реверс с фитом:
Север: 1♦ (♠КДхх ♥Тх ♦ТКВхх ♣Дх, 18 очков, 5♦4♠)
Юг: 1♥ (♠хх ♥КДххх ♦Дх ♣Вххх, 9 очков)
Север: 2♠ (РЕВЕРС! 16+hcp)
Юг: 3♥ (ребид своей масти)
Север: 3NT

✗ НЕ реверс (слабая рука):
Север: 1♣ (♠Кх ♥Дххх ♦хх ♣ТКВхх, 12 очков, 4♥5♣)
Юг: 1♠
Север: 1NT (НЕ 2♥! Недостаточно силы для реверса)
```

> [!warning]- ВАЖНО про реверс
> Реверс ОБЕЩАЕТ 16+hcp! Не делай реверс с 12-15hcp - это переобещание силы.

## 1.16. Reverse Drury (Обратный Друри)

**Заявка:** пас-пас-1М-**2♣**

**Тип:** Искусственная, вопрос о силе

**Значение:** Как Друри, но ответы открывающего обратные

**Отличие от обычного Друри:**

**Обычный Друри:**

- 2♦ = слабая рука, нет гейма
- 2М+ = интерес к гейму

**Reverse Drury:**

- 2♦ = У МЕНЯ ЕСТЬ ГЕЙМ! (12+hcp)
- 2М = Слабая рука (10-11hcp), только если у партнера максимум

**Примеры:**

```
✅ Reverse Drury (сильная рука):
Торговля: пас-пас-1♠(Север)-2♣(Юг, Reverse Drury)
Север: 2♦ (♠ТКДхх ♥Кх ♦Тхх ♣Дхх, 15 очков, У МЕНЯ ГЕЙМ!)
Юг: 3♠ (♠Дххх ♥Дхх ♦Кхх ♣Кхх, 10 очков, принимаю)
Север: 4♠

✅ Reverse Drury (слабая рука):
Торговля: пас-пас-1♥(Север)-2♣(Юг)
Север: 2♥ (♠Кхх ♥ТКххх ♦Дхх ♣хх, 11 очков, слабое открытие)
Юг: Пас (♠Дх ♥Дххх ♦Кхх ♣Тхх, 10 очков)
```

## 1.17. Gerber (4♣)

**Заявка:** 1NT-**4♣** или 2NT-**4♣**

**Тип:** Искусственная, вопрос о тузах

**Значение:** "Сколько у тебя тузов?" (аналог Блэквуда, но через 4♣)

**Когда применять:**

- ✓ После открытия 1NT или 2NT
- ✓ Хотите спросить о тузах
- ✓ 4NT может быть количественным инвитом

**Ответы:**

| Заявка  | Количество тузов |
| ------- | ---------------- |
| **4♦**  | 0 или 4 туза     |
| **4♥**  | 1 туз            |
| **4♠**  | 2 туза           |
| **4NT** | 3 туза           |

**Продолжение:** **5♣** = вопрос о королях

**Примеры:**

```
✅ Gerber:
Север: 2NT (20-21hcp)
Юг: 4♣ (♠Кххх ♥Дххх ♦Тхх ♣Кх, 11 очков, Gerber)
Север: 4♥ (1 туз)
Юг: 5♣ (вопрос о королях)
Север: 5♠ (2 короля)
Юг: 6NT (у нас все тузы и короли!)

✗ НЕ Gerber (натуральная заявка):
Север: 1♣
Юг: 1♥
Север: 2♣
Юг: 4♣ (НЕ Gerber! Это натуральная поддержка ♣)
```

> [!note]- Gerber vs Блэквуд
> **Gerber (4♣)** - после 1NT/2NT, когда 4NT = количественный инвит
> **Блэквуд (4NT)** - после согласования масти

**Значение:** "У меня сила, есть карты в непоказанных мастях, предлагаю выбрать"

**Когда применять:**

- ✓ Партнер сделал контру для вывода
- ✓ Противник поддержал масть
- ✓ У вас 6+hcp
- ✓ Есть карты в обеих непоказанных мастях

**Примеры:**

```
✅ Responsive Double:
Противник: 1♥
Партнер: Контра (для вывода)
Противник: 2♥ (поддержка)
Вы: Контра (♠Кххх ♥хх ♦Дххх ♣Вхх, 8 очков, Responsive: есть ♠ и ♦)
Партнер: 2♠ (выбираю ♠)

✅ Responsive Double на высоком уровне:
Противник: 1♠
Партнер: Контра
Противник: 3♠ (блокирующая поддержка)
Вы: Контра (♠х ♥Кххх ♦Дххх ♣Тххх, 9 очков, Responsive: есть оба минора и ♥)
Партнер: 4♥ (выбираю ♥)
```

---

## 2.9. Maximal Double (Максимальная контра)

**Заявка:** 1♥-2♣-2♥-3♣-**Контра**

**Тип:** Искусственная, инвит

**Значение:** "У меня инвит в нашей масти, но нет места для инвита заявкой"

**Когда применять:**

- ✓ Конкуренция в двух мастях
- ✓ Хотите сделать инвит
- ✓ Следующая ступень = гейм (нет места для инвита)

**Примеры:**

```
✅ Maximal Double (инвит):
Север: 1♥
Противник: 2♣
Юг: 2♥ (♠Кх ♥Дххх ♦Тхх ♣Вххх, 9 очков)
Противник: 3♣
Север: Контра (♠Дхх ♥ТКххх ♦Кхх ♣Дх, 14 очков, Maximal Double = инвит!)
Юг: 4♥ (принимаю с максимумом)

✅ Прямой гейм (без Maximal):
Север: 1♠
Противник: 2♦
Юг: 2♠
Противник: 3♦
Север: 4♠ (♠ТКДхх ♥Кх ♦Дх ♣ТКхх, 18 очков, прямо в гейм)
```

## 2.10. Balancing (Балансирующие заявки) - ПОДРОБНО

**Заявка:** 1♥-Пас-Пас-**действие** (контра/масть/1NT)

**Тип:** Защитная, балансирующая

**Философия:** "Не даем противникам дешево играть на низком уровне в частичной"

---

### Что такое балансирующая позиция?

**Балансирующая позиция = 4-я позиция после двух пасов**

```
Позиции:
Открывающий - Партнер - Отвечающий - ВЫ
1♥          - Пас     - Пас        - ? (БАЛАНСИРУЮЩАЯ ПОЗИЦИЯ)
```

**Ключевой момент:** Если вы пасуете → торговля закончится и противники сыграют 1♥.

---

### Почему можно балансировать "легче"?

**Логика балансирования:**

1. Противник открыл 1♥ = ~12-19hcp
2. Отвечающий спасовал = 0-5hcp (слабая рука!)
3. **Значит оставшиеся очки (~18-25hcp) между вами и партнером**
4. Если у вас 8-11hcp → у партнера может быть 10-14hcp!

> [!success]- Правило балансировки
> **"Занимайте у партнера 3-4 очка!"**
>
> В балансирующей позиции действуйте как будто у вас на 3-4 очка больше, потому что партнер ОБЯЗАН иметь силу (он не смог вмешаться напрямую из-за отсутствия подходящей заявки).

---

### Таблица балансирующих заявок

| Заявка             | Прямо (2-я поз.) | Балансирующая (4-я поз.) | Разница |
| ------------------ | ---------------- | ------------------------ | ------- |
| **Контра**         | 12+hcp           | **8+hcp**                | -4 очка |
| **1NT**            | 15-18hcp         | **11-14hcp**             | -4 очка |
| **Масть на 1 ур.** | 8+hcp, 5+ карт   | **7+hcp**, 5+ карт       | -1 очко |
| **Масть на 2 ур.** | 10+hcp, 5+ карт  | **9+hcp**, 5+ карт       | -1 очко |
| **Прыжок**         | 16+hcp           | **13-15hcp**             | -3 очка |
| **2NT**            | 20-21hcp         | **18-19hcp**             | -2 очка |

---

### Виды балансирующих заявок

#### **1. Балансирующая КОНТРА**

**Требования:**

- ✓ 8+hcp (вместо обычных 12+)
- ✓ Поддержка в непоказанных мастях
- ✓ Раскладная рука или недостаток силы для прямой контры

**Примеры:**

```
✅ Балансирующая контра (классическая):
Противник: 1♥
Партнер: Пас
Противник: Пас
Вы: Контра (♠Кххх ♥хх ♦Дххх ♣Тхх, 10 очков)

Логика: У меня только 10 очков, но партнер должен иметь ~10-12,
поэтому безопасно балансировать.

✅ Балансирующая контра (нестандартное распределение):
Противник: 1♠
Партнер: Пас
Противник: Пас
Вы: Контра (♠хх ♥Кххх ♦Дхххх ♣Тх, 9 очков)

Хотя только 9 очков, но есть 5♦4♥ - можем найти фит.

✗ НЕ балансируйте с плоской слабой:
Противник: 1♦
Партнер: Пас
Противник: Пас
Вы: Пас! (♠Кхх ♥Вхх ♦Дхх ♣хххх, 8 очков, 4333)

Слишком плоская - партнер не сможет выбрать масть.
```

#### **2. Балансирующий 1NT**

**Требования:**

- ✓ 11-14hcp (вместо обычных 15-18hcp)
- ✓ Держка в их масти
- ✓ Сбалансированная рука

**Примеры:**

```
✅ Балансирующий 1NT:
Противник: 1♠
Партнер: Пас
Противник: Пас
Вы: 1NT (♠Кхх ♥Дхх ♦ТВхх ♣Вхх, 12 очков, балансирующий)

Партнер будет думать: "У партнера 11-14, у меня 10 → играем 1NT или 2NT"

✅ Балансирующий 1NT максимум:
Противник: 1♥
Партнер: Пас
Противник: Пас
Вы: 1NT (♠Кхх ♥Дх ♦ТКхх ♣Дххх, 14 очков, максимум баланса)

Партнер: 2NT (инвит, у него ~10-11)
Вы: 3NT (принимаю с максимумом)
```

#### **3. Балансирующая МАСТЬ**

**Требования:**

- ✓ 7-15hcp (зависит от уровня)
- ✓ 5+ карт в масти (хорошая масть!)
- ✓ Масть лучше, чем контра

**Примеры:**

```
✅ Балансирующая масть на 1 уровне:
Противник: 1♣
Партнер: Пас
Противник: Пас
Вы: 1♥ (♠Кх ♥ДВхххх ♦Тхх ♣хх, 9 очков, 6♥)

✅ Балансирующая масть на 2 уровне:
Противник: 1♠
Партнер: Пас
Противник: Пас
Вы: 2♣ (♠хх ♥Кх ♦Тхх ♣КДВххх, 10 очков, 6♣)

✅ Балансирующий прыжок (сильная):
Противник: 1♦
Партнер: Пас
Противник: Пас
Вы: 2♥ (♠Кх ♥ТКДВхх ♦хх ♣Дхх, 14 очков, прыжок = 13-15hcp!)
```

#### **4. Балансирующий 2NT**

**Требования:**

- ✓ 18-19hcp (вместо обычных 20-21hcp)
- ✓ Держка в их масти
- ✓ Сбалансированная

**Пример:**

```
✅ Балансирующий 2NT:
Противник: 1♥
Партнер: Пас
Противник: Пас
Вы: 2NT (♠ТКх ♥Дхх ♦ТКВх ♣Дхх, 19 очков)

Партнер: 3NT (у него 6-7 очков хватает для гейма)
```

---

### Ответы партнера на балансирующие заявки

**Правило для партнера:** "Вычти 3-4 очка из силы балансирующего"

#### На балансирующую КОНТРУ:

| Сила партнера | Действие                          |
| ------------- | --------------------------------- |
| **0-8hcp**    | Минимальная заявка в лучшей масти |
| **9-11hcp**   | Прыжковая заявка (инвит)          |
| **12+hcp**    | GF заявки или кюбид               |

```
Пример:
Противник: 1♥ - Пас - Пас
Партнер: Контра (БАЛАНС, 8-11hcp)
Противник: Пас
Вы: ?

С ♠Кххх ♥хх ♦Дххх ♣Вхх (9 очков) → 2♠ (прыжок-инвит)
С ♠Дххх ♥хх ♦Кхх ♣хххх (6 очков) → 1♠ (минимум)
```

#### На балансирующий 1NT:

| Сила партнера | Действие                      |
| ------------- | ----------------------------- |
| **0-7hcp**    | Пас или заявка масти (слабая) |
| **8-9hcp**    | 2NT (инвит)                   |
| **10+hcp**    | 3NT (гейм)                    |

```
Пример:
Противник: 1♠ - Пас - Пас
Партнер: 1NT (БАЛАНС, 11-14hcp)
Противник: Пас
Вы: ?

С 10 очками → 3NT (у нас 21-24 вместе, гейм!)
С 8 очками → 2NT (инвит, может быть гейм)
С 5 очками → Пас (играем 1NT)
```

---

### Когда НЕ балансировать?

❌ **НЕ балансируйте в этих случаях:**

**1. Слишком слабая рука (< 7hcp)**

```
Противник: 1♥ - Пас - Пас
Вы: Пас (♠Кхх ♥хх ♦Дхх ♣хххxx, 5 очков - слишком слабо!)
```

**2. Плоское распределение + минимум**

```
Противник: 1♦ - Пас - Пас
Вы: Пас (♠Кхх ♥Вхх ♦Дхх ♣хххх, 8 очков, 4333 - нет мастей)
```

**3. Длина в их масти (они фитованы)**

```
Противник: 1♥ - Пас - Пас
Вы: Пас (♠Кхх ♥Кхххх ♦Дхх ♣хх, 9 очков, но 5♥!)

Логика: У них слабый фит, поэтому отвечающий спасовал.
Если я балансирую - партнер будет переоценивать руку.
```

**4. Партнер пасовал ДВАЖДЫ**

```
Противник: 1♠ - Пас - 1NT
Партнер: Пас (уже показал слабую руку!)
Противник: Пас
Вы: Пас (партнер реально слаб, не балансируй)
```

---

### Балансирование после их 1NT

**Особый случай:** (1NT) - Пас - Пас - ?

Балансирование против 1NT ОПАСНЕЕ, потому что:

- У них 25-27 очков ИЗВЕСТНО (не 12-16 как после 1M)
- У вас остается меньше очков (~13-15)

**Требования для баланса против 1NT:**

- ✓ Хорошая раскладная рука (5-5 или 6-5)
- ✓ Минимум 10-12hcp
- ✓ НЕ балансируйте с сбалансированной рукой

```
✅ Балансируем (раскладная):
(1NT) - Пас - Пас
Вы: 2♦ (♠ТКххх ♥ДВххх ♦х ♣Кх, 11 очков, Cappelletti/Landy: 55 мажоры)

✗ НЕ балансируем (плоская):
(1NT) - Пас - Пас
Вы: Пас (♠Кхх ♥Дхх ♦Тххх ♣Вхх, 11 очков, плоская - пропустим)
```

---

### Балансирование на высоком уровне

**Правило:** Чем выше уровень, тем ОПАСНЕЕ балансировать

```
(2♥) - Пас - Пас - ?
Нужно ~10+hcp для балансирования контрой

(3♥) - Пас - Пас - ?
Нужно ~12+hcp для балансирования контрой

(4♥) - Пас - Пас - ?
Обычно НЕ балансируйте (слишком опасно)
```

---

### Защита от балансирования (для противников)

**Если вы противники и боитесь балансирования:**

1. **Открывающий:** Ребидируй с минимумом, чтобы отвечающий мог поддержать

   ```
   Вы: 1♥
   Партнер: 1NT (вынужденная, может быть 0-5!)
   Вы: 2♥ (показываю 6♥, чтобы партнер мог поддержать с 2♥)
   ```

2. **Отвечающий:** С 6-9hcp делай заявку, не пасуй
   ```
   Партнер: 1♥
   Вы: 1NT (♠Кхх ♥хх ♦Дххх ♣Вхх, 8 очков - не пасую, чтобы не давать балансировать!)
   ```

---

### Примеры полных раздач

#### **Пример 1: Успешное балансирование**

```
Север (противник): ♠ТДхх ♥Кхххх ♦Тх ♣Кх (13 очков)
Восток (вы): ♠Кхх ♥х ♦Дххххх ♣Дхх (9 очков)
Юг (противник): ♠Вхх ♥Вхх ♦Кхх ♣Вххх (4 очка)
Запад (партнер): ♠Дхх ♥ТДхх ♦Вхх ♣Тххх (14 очков!)

Торговля:
Север: 1♥
Восток: Пас (не могу вмешаться с 9 очками напрямую)
Юг: Пас (слабо)
Запад: Контра (БАЛАНС! Знаю, что у Востока есть очки)
Север: Пас
Восток: 2♦ (мой лучший минор)
Юг: Пас
Запад: Пас (достаточно)

Результат: 2♦ = +90 (у нас 23 очка, мы должны торговаться!)
Без баланса: они играли бы 1♥ = +140
```

#### **Пример 2: Опасное балансирование**

```
Север (противник): ♠ТДхххх ♥Кх ♦Тх ♣Дхх (14 очков)
Восток (вы): ♠Кх ♥Дххх ♦Дххх ♣Тхх (10 очков)
Юг (противник): ♠хх ♥Вхх ♦Кххх ♣Вххх (5 очков)
Запад (партнер): ♠Дхх ♥ТВхххх ♦Вх ♣Кх (11 очков)

Торговля:
Север: 1♠
Восток: Пас
Юг: Пас
Запад: 2♥ (БАЛАНС, 6♥, 11 очков)
Север: Контра (штрафная!)

Результат: 2♥X -1 = -100
Если бы не балансировали: 1♠ = +80

Вывод: Балансирование с 6-карточной мастью иногда опасно.
```

---

### Золотые правила балансирования

1. ✅ **Занимай у партнера 3-4 очка** - действуй смелее
2. ✅ **Раскладная рука лучше** - не балансируй с 4333
3. ✅ **Не балансируй с длиной в их масти** - они могут быть фитованы
4. ✅ **Чем выше уровень, тем опаснее** - осторожность на 3+
5. ✅ **Партнер вычитает 3-4 очка** - он должен помнить о балансе
6. ✅ **Против 1NT осторожнее** - нужна раскладная рука

## 2.11. Lightner Double (Контра Лайтнера)

**Заявка:** Контра на шлемовый контракт противников

**Тип:** Специальная, направляющая выход

**Значение:** "НЕ ходи стандартно! Нужен особый выход (обычно в масть болвана)"

**Когда применять:**

- ✓ Противники в шлеме
- ✓ У вас есть особая масть для выхода
- ✓ Обычно: пустая в их побочной масти, или ТКД в побочной

**Примеры:**

```
✅ Lightner Double (пустая в побочной):
Торговля противников: 1♠-3♠-4♣(кюбид)-4♦(кюбид)-6♠

Ваша рука: ♠хх ♥КДхххх ♦- ♣Тхххх

Вы: Контра (Lightner! У меня ренонс ♦!)
Партнер: Выходит ♦ (необычный выход)
Вы: Перебиваете ♥
Результат: Положили 6♠!

✅ Lightner Double (ТКД в побочной):
Торговля: 1♥-3♥-4♣-4♦-6♥

Ваша рука: ♠хх ♥хх ♦ТКДхх ♣хххх

Вы: Контра (Lightner! У меня ТКД♦!)
Партнер: Выходит ♦
Результат: Взяли 2 взятки в ♦, положили 6♥!
```

---

## 2.12. Lead-Directing Double (Направляющая выход контра)

**Заявка:** Контра на искусственную заявку противников

**Тип:** Специальная, направляющая выход

**Значение:** "У меня сила в этой масти, ходи сюда!"

**Когда применять:**

- ✓ Противники делают искусственную заявку (Стейман, трансфер, кюбид)
- ✓ У вас сила в этой масти (ТКВ, КДВ10 и т.п.)
- ✓ Хотите направить выход партнера

**Примеры:**

```
✅ Контра на Стейман:
Противник: 1NT
Противник: 2♣ (Стейман)
Вы: Контра (♠Кх ♥Дхх ♦Тхх ♣КДВ10х, у меня ♣!)
Результат: Партнер выйдет ♣

✅ Контра на трансфер:
Противник: 1NT
Противник: 2♦ (трансфер в ♥)
Вы: Контра (♠Кхх ♥хх ♦ТКДхх ♣Вхх, у меня ♦!)
Результат: Партнер выйдет ♦

✅ Контра на кюбид:
Противник: 1♠
Противник: 2♠
Противник: 3♥ (кюбид)
Вы: Контра (♠хх ♥ТКДхх ♦Вххх ♣хх, у меня ♥!)
Результат: Если они играют 4♠, партнер выйдет ♥
```

## 2.13. SOS Redouble (Реконтра SOS)

**Заявка:** (1♠)-Контра-(Пас)-1NT-(Контра)-**Реконтра**

**Тип:** Искусственная, просьба о помощи

**Значение:** "Мы не можем играть здесь! Переключись на другую масть!"

**Когда применять:**

- ✓ Партнер законтровал их контракт
- ✓ Они законтровали нас
- ✓ У вас очень слабая рука в этой масти
- ✓ Просите партнера назвать другую масть

**Примеры:**

```
✅ SOS Redouble:
Противник: 1♠
Партнер: Контра (для вывода)
Противник: Пас
Вы: 1NT (♠ххххх ♥хх ♦ххх ♣хх, 2 очка, вынужденная заявка)
Противник: Контра (штрафная!)
Вы: Реконтра (SOS! Давай уйдем, у меня ничего нет!)
Партнер: 2♣ (ухожу в ♣)

✅ SOS после нашей контры:
Партнер: Контра (на 2♠)
Противник: Реконтра
Вы: 3♣ (SOS, давай уйдем, мы не можем бить 2♠)
```

# 3. Конвенции шлемовой торговли

## 3.1. Блэквуд (Blackwood)

**Заявка:** **4NT** (после согласования масти)

**Тип:** Искусственная, вопрос о тузах

**Значение:** "Сколько у тебя тузов?"

**Когда применять:**

- ✓ Масть согласована
- ✓ Достаточно силы для шлема (33+ очков)
- ✓ Нужно проверить количество тузов
- ✗ НЕ применяйте если масть НЕ согласована (= количественный инвит)

**Ответы:**

| Заявка | Количество тузов |
| ------ | ---------------- |
| **5♣** | 0 или 4 туза     |
| **5♦** | 1 туз            |
| **5♥** | 2 туза           |
| **5♠** | 3 туза           |

**Продолжение: 5NT** = вопрос о королях (только если есть ВСЕ тузы!)

**Примеры:**

```
✅ Блэквуд, идем в шлем:
Север: 1♠
Юг: 2♠ (♠Дххх ♥Тххх ♦Кх ♣Кхх, 11 очков, фит)
```

## 3.2. RKCB (Roman Key Card Blackwood)

**Значение:** "Сколько у тебя ключевых карт (4 туза + К козыря)?"

**Ответы:**

| **5♣** | 0 или 3              |
| **5♦** | 1 или 4              |

**Продолжение:** Следующая ступень = вопрос о Д козыря

**Примеры:**

## 3.3. Кюбиды (Cue-bidding)

**Заявка:** Новая масть после согласования (выше 3 уровня)

**Тип:** Искусственная, показ контроля

**Значение:** "У меня контроль (Т или К) в этой масти"

**Когда применять:**

- ✓ Масть согласована
- ✓ Шлемовые амбиции
- ✓ Блэквуд не подходит (есть быстрые потери)

**Правила:**

1. Первый контроль = Т или К (не уточняется)
2. Повторный кюбид = К (туз уже показан)
3. Пропуск масти = НЕТ контроля
4. Возврат в козырную масть = стоп

**Примеры:**

```
✅ Кюбиды приводят к шлему:
Север: 1♠
Юг: 2♠ (♠Дххх ♥Кххх ♦Кх ♣Тхх, 12 очков, фит)
Север: 3♣ (♠ТКДхх ♥Тх ♦ТВхх ♣Кх, 19 очков, кюбид К♣)
Юг: 3♦ (кюбид К♦)
Север: 3♥ (кюбид Т♥)
Юг: 4♣ (кюбид Т♣)
Север: 4NT (RKCB)
Юг: 5♥ (2 ключевых)
Север: 6♠

✅ Кюбиды, стоп:
Север: 1♠
Юг: 2♠ (♠Дххх ♥Дххх ♦хх ♣Кхх, 10 очков, нет Т♥!)
Север: 3♣ (кюбид)
Юг: 3♦ (кюбид, пропустил ♥ - НЕТ контроля!)
Север: 4♠ (стоп, у нас быстрые потери в ♥)
```

## 3.5. DOPI / ROPI

- **DOPI** (Double = 0, Pass = 1) - после контры

- ✓ Партнер спросил Блэквуд (4NT)
- ✓ Противник вмешался (контра или заявка)

```
Север: 4NT (RKCB)

С 1 ключевой → Пас

С 2 ключевых с Д → 5♦
```

| **Пас**        | 1 (или 4 в RKCB)          |
| **Следующая**  | 2 без Д                   |
| **Через одну** | 2 с Д                     |

**Пример ROPI:**

```
Север: 4NT (RKCB)
Противник: 5♣
Юг: ?

С 0 ключевых → Реконтра
С 1 ключевой → Пас
С 2 ключевых без Д → 5♦
С 2 ключевых с Д → 5♥
```

# 4. Защитные конвенции

## 4.1. Сигнал отношения (Attitude Signal)

**Применение:** На ход партнера мастью

**Тип:** Информационная, показ желания

**Значение:**

- **Высокая карта** = ХОЧУ продолжения (есть фигура)
- **Мелкая карта** = НЕ ХОЧУ продолжения

**Когда применять:**

- ✓ Партнер вышел Т или К
- ✓ Партнер вышел длинной мастью
- ✓ Нужно сказать: хочу продолжения или нет

**Примеры:**

```
✅ Поощряем продолжение:
Партнер вышел: К♠
Болван: ♠742
Вы: ♠Д108
Сброс: 8♠ (высокая = ХОЧУ, есть Д!)

✅ Отказываем от продолжения:
Партнер вышел: К♠
Болван: ♠742
Вы: ♠1083
Сброс: 3♠ (мелкая = НЕ ХОЧУ, нет фигур)
```

**Детали:** → [[Сигналы в защите]]

## 4.2. Сигнал количества (Count Signal)

**Применение:** Когда разыгрывающий играет масть

**Тип:** Информационная, показ распределения

**Значение:**

- **Высокая→Мелкая (ВМ)** = ЧЕТНОЕ количество (2, 4, 6)
- **Мелкая→Высокая (МВ)** = НЕЧЕТНОЕ количество (3, 5, 7)

**Когда применять:**

- ✓ Разыгрывающий играет масть из болвана
- ✓ Партнеру нужно знать распределение
- ✓ Для определения, когда брать взятку

**Примеры:**

```
✅ Четное количество (дуплет):
Болван играет: ♦ТКД
Вы: ♦96
Сброс: 9♦ (высокая) → затем 6♦
Сигнал: ВМ = ЧЕТНОЕ (2 карты)

✅ Нечетное количество:
Болван играет: ♦ТКД
Вы: ♦964
Сброс: 4♦ (мелкая) → затем 9♦
Сигнал: МВ = НЕЧЕТНОЕ (3 карты)
```

**Детали:** → [[Сигналы в защите]]

## 4.3. Сигнал преференса (Suit Preference)

**Применение:** При перебитии или когда партнер дает ход

**Тип:** Информационная, указание масти

**Значение:**

- **Высокая карта** = хочу СТАРШУЮ из оставшихся мастей
- **Мелкая карта** = хочу МЛАДШУЮ из оставшихся мастей

**Когда применять:**

- ✓ Вы перебиваете (рафф)
- ✓ Партнер отдает вам ход тузом
- ✓ Отношение и количество неактуальны

**Ранг мастей:** ♠ (старшая) > ♥ > ♦ > ♣ (младшая)

**Примеры:**

```
✅ Преференс при перебитии (хочу старшую):
Контракт: 4♥
Партнер вышел: Т♦
Вы перебиваете: ♥К (♠Тхх ♥Кх ♦х ♣хххх)

Анализ:
- Исключаем: ♥ (козырь), ♦ (текущая)
- Остаются: ♠ (старшая), ♣ (младшая)
- У нас Т♠ - хотим ♠!

Сброс под свое перебитие: Д♦ (высокая = хочу ♠!)

✅ Преференс (хочу младшую):
Контракт: 4♥
Партнер вышел: Т♦
Вы перебиваете: ♥К (♠хх ♥Кх ♦х ♣ТКхххх)

Сброс: 2♦ (мелкая = хочу ♣!)
```

**Детали:** → [[Сигналы в защите]]

## 4.4. МВМ (Мелкая-Высокая-Мелкая)

**Тип:** Информационная, показ количества

```
Партнер понимает: МВМ = 3 карты

✅ Дуплет (ВМ):
Вы: ♥94
```

## 4.5. Правило 11 (Rule of 11)

**Применение:** Партнер вышел мелкой против БК

**Тип:** Математическая, расчет распределения

**Формула:** **11 - (карта выхода) = количество карт старше у других игроков**

**Когда применять:**

- ✓ Против БК контрактов
- ✓ Партнер вышел 4-й по старшинству
- ✓ Вы в третьей руке (видите болван)

**Цель:** Определить карты у разыгрывающего

**Примеры:**

```
✅ Правило 11 помогает экономить:
Партнер вышел: 5♠
Болван: ♠К82
Вы: ♠Д104

Расчет:
11 - 5 = 6 карт старше пятерки
Видим: К,8 (болван) + Д,10,8 (мы) = 5 карт
Значит у разыгрывающего: 6 - 5 = 1 карта (это Т!)

Решение: Кладем 4♠ из болвана (экономим К и Д)

✅ Правило 11 показывает дуплет:
Партнер вышел: 7♦
Болван: ♦К93
Вы: ♦Д1082

Расчет:
11 - 7 = 4 карты старше
Видим: К,9 + Д,10,8 = 5 карт (больше 4!)

Значит: Партнер вышел НЕ 4-й (возможно, дуплет или МВМ)

Решение: Берем К♦ из болвана
```

**Детали:** → [[Первый ход (Атака)]]

```
Север: 4NT (RKCB)
```

| ------ | ---------------------------------- |
| **3♦** | Нет 5М, может быть 4М (марионетка) |
| **3♥** | 5♥                                 |
| **3♠** | 5♠                                 |

**После 3♦ (марионетка):**

- **Пас** = 6+♦, слабая рука (редко)
- **3♥** = 4♠ (НЕ 4♥)
- **3♠** = 4♥ (НЕ 4♠)
- **3NT** = нет 4М

**Примеры:**

```
✅ Puppet Stayman находит 5-3:
Север: 2NT (♠Кх ♥ТКДхх ♦ТКх ♣Дхх, 21 очков, 5♥)
Юг: 3♣ (♠Дхх ♥Кхх ♦Дххх ♣Кхх, 10 очков, Puppet)
Север: 3♥ (5♥!)
Юг: 4♥ (отлично!)

✅ Puppet Stayman находит 4-4:
Север: 2NT (♠ТКхх ♥Дхх ♦ТКх ♣ТДх, 21 очков, 4♠)
Юг: 3♣ (♠Дххх ♥Кх ♦Дхх ♣Кххх, 10 очков)
Север: 3♦ (марионетка, нет 5М, но есть 4М)
Юг: 3♥ (у меня 4♠, НЕ 4♥)
Север: 3♠ (подтверждаю 4♠)
Юг: 6♠ (у нас 31 очко + фит!)

✅ Puppet Stayman, нет фита:
Север: 2NT (♠Кхх ♥Дхх ♦ТКх ♣ТДхх, 20 очков, нет 4-5М)
Юг: 3♣ (Puppet)
Север: 3♦ (марионетка)
Юг: 3NT (нет 4М)
```

## 6.3. OGUST (Ответы на слабую двойку)

**Заявка:** 2М-**2NT**

**Тип:** Искусственная, вопрос о качестве

**Значение:** "Какая у тебя рука по силе и качеству масти?"

**Когда применять:**

- ✓ Партнер открыл слабой двойкой (2♥/2♠)
- ✓ У вас сила, хотите узнать о его руке
- ✓ Решаете: гейм или нет

**Ответы (после 2♥-2NT):**

| Заявка  | Значение                          |
| ------- | --------------------------------- |
| **3♣**  | Плохая масть, минимум (5-7hcp)    |
| **3♦**  | Хорошая масть, минимум (5-7hcp)   |
| **3♥**  | Плохая масть, максимум (8-10hcp)  |
| **3♠**  | Хорошая масть, максимум (8-10hcp) |
| **3NT** | ТКД в масти + держки              |

**Примеры:**

```
✅ OGUST - хорошая масть, максимум:
Север: 2♥
Юг: 2NT (♠Тхх ♥Кх ♦ТКхх ♣Дххх, 13 очков, OGUST)
Север: 3♠ (♠Дх ♥ТКДВхх ♦Вхх ♣хх, 9 очков, хорошая ♥, максимум!)
Юг: 4♥ (отлично, идем в гейм!)

✅ OGUST - плохая масть, минимум:
Север: 2♠
Юг: 2NT (OGUST)
Север: 3♣ (♠Дххххх ♥Кх ♦Вхх ♣хх, 6 очков, плохая ♠, минимум)
Юг: Пас (нет гейма)
```

## 6.6. Good-Bad 2NT (Хорошая-Плохая 2NT)

**Заявка:** 1♥-(2♣)-Пас-(Пас)-2♦-**2NT**

**Тип:** Искусственная, показ силы

| Заявка отвечающего | Значение                          |
| ------------------ | --------------------------------- |

**После 2NT открывающий:**

- **3♣** = минимум, пас или корректируй
- **3♥** = принимаю (если у отвечающего было 8-9)

**Примеры:**

```
✅ Good-Bad 2NT (минимум):
Север: 1♥
Противник: 2♣
Юг: Пас (♠Кхх ♥Дххх ♦Вхх ♣хх, 6 очков)
Противник: Пас
Север: 2♦ (♠Дхх ♥ТКххх ♦ТДхх ♣х, 14 очков)
Юг: 2NT (Good-Bad: минимум, но могу дать 3♥)
Север: 3♥ (принимаю)

✅ Прямая 3♥ (максимум):
Север: 1♥
Противник: 2♣
Юг: Пас (♠Кхх ♥Дххх ♦Тхх ♣Вхх, 9 очков, максимум!)
Противник: Пас
Север: 2♦
Юг: 3♥ (инвит, 9-10hcp)
Север: 4♥ (принимаю)
```

- **3NT** = 5♥ с солидной мастью

```
✅ Kokish с сильной сбалансированной:
Север: 2♣ (♠ТКх ♥Дхх ♦ТКВ ♣ТКДх, 25 очков, сбалансированная)

Север: 2♥ (Kokish relay)

Север: 2NT (24+hcp, сбалансированная)

Север: 2♣ (♠Тх ♥ТКДВх ♦ТКД ♣Кхх, 23 очка, 5♥)
```

# Памятка: Приоритет конвенций

## Что важнее?

1. **Найти фит 4-4 в мажоре** → Стейман
2. **Показать 5М** → Трансфер
3. **Проверить фит после 1NT** → 2wcb или NMF
4. **Создать форсинг без фита** → 4SF (Fourth Suit Forcing)
5. **Показать очень сильную руку** → Jump Shift (18+) или Jacoby 2NT
6. **Различить силу руки** → Газилли или Реверс (16+)
7. **Показать максимум паса** → Друри или Reverse Drury
8. **Точно показать поддержку** → Bergen Raises
9. **Проверить тузов перед шлемом** → Блэквуд/RKCB
10. **Проверить качество козырей** → Grand Slam Force
11. **Показать краткость** → Сплинтер или Exclusion Blackwood
12. **Показать РОВНО 3М в конкуренции** → Support Double
13. **Вмешаться двухмастной** → Микаэл/Unusual 2NT
14. **Балансировать** → Balancing (8+hcp в позиции)

## Когда НЕ применять конвенции

❌ **НЕ применяй Стейман с 5М** → используй трансфер
❌ **НЕ применяй Блэквуд без фита** → это количественный инвит
❌ **НЕ применяй Друри на 1-2 руке** → это обычная GF заявка
❌ **НЕ применяй большой трансфер для шлема** → используй обычный
❌ **НЕ применяй сплинтер с 16+hcp** → используй Jump Shift или Jacoby 2NT
❌ **НЕ делай реверс с 12-15hcp** → это обещает 16+!
❌ **НЕ делай Jump Shift с 12-15hcp** → это обещает 18+!
❌ **НЕ применяй 4SF если есть фит** → покажи фит напрямую
❌ **НЕ применяй Support Double с 4+М** → подними масть напрямую
❌ **НЕ применяй Gerber после заявки масти** → это натуральная ♣
❌ **НЕ балансируй с плоской 8hcp** → нужна раскладная рука
❌ **НЕ применяй Lightner Double на частичную** → только на шлем

# 6A. ПОЛНЫЙ СПРАВОЧНИК КОНТР И РЕКОНТР

> [!info]+ О разделе
> Контра и реконтра - самые многозначные заявки в бридже. Их значение зависит от контекста, уровня, силы и предыдущих заявок. В этом разделе - ПОЛНЫЙ разбор всех видов контр и реконтр.

---

## 🎯 КЛАССИФИКАЦИЯ КОНТР

**По назначению:**

1. **Вызывная (Takeout)** - "Назови свою масть"
2. **Негативная (Negative)** - "У меня есть другие масти"
3. **Наказательная (Penalty)** - "Хочу их бить!"
4. **Информационная** - показ поддержки или контроля
5. **Специальная** - особые ситуации

---

## 1. ВЫЗЫВНАЯ КОНТРА (Takeout Double)

**Применение:** (1♣/♦/♥/♠) - **Контра**

**Тип:** Искусственная, вопрос

**Значение:** "Назови свою лучшую масть, у меня поддержка во всех непоказанных мастях"

### Требования

**Обязательные условия:**

- ✓ 12+hcp (обычно 12-17hcp)
- ✓ Поддержка во ВСЕХ непоказанных мастях (обычно 4+ карт)
- ✓ Короткая масть противника (0-2 карты)

**Типичные распределения:**

- **4-4-4-1** (идеальная контра)
- **4-4-3-2** (хорошая)
- **5-4-3-1** (допустимо)
- **5-4-4-0** (отлично с ренонсом)

### Сила контры

| HCP       | Рука                                |
| --------- | ----------------------------------- |
| **12-15** | Минимум, но партнер должен ответить |
| **16-18** | Хорошая, можем быть в гейме         |
| **19+**   | Очень сильная, форсируем до гейма   |

### Ответы партнера

| Сила        | Заявка                 | Форсинг? |
| ----------- | ---------------------- | -------- |
| **0-8hcp**  | Минимум в лучшей масти | НЕТ      |
| **9-11hcp** | Прыжок (инвит)         | Инвит    |
| **12+hcp**  | Кюбид или GF           | ДА       |

### Примеры

```
✅ Классическая вызывная контра:
Противник: 1♥
Вы: Контра (♠ТКхх ♥х ♦Дххх ♣КДхх, 14 очков, 4♠4♦4♣1♥)
Партнер: 1♠ (лучшая масть)
Противник: Пас
Вы: 2♠ (поддержка, 14 очков)

✅ Сильная вызывная контра:
Противник: 1♣
Вы: Контра (♠ТКхх ♥ТДхх ♦Кхх ♣хх, 17 очков, 4♠4♥3♦)
Партнер: 1♥ (6 очков)
Противник: Пас
Вы: 3♥ (прыжок = 17-18hcp, инвит к гейму)
Партнер: 4♥ (принимаю)

✅ Контра с 5-4:
Противник: 1♦
Вы: Контра (♠Кхххх ♥ТДхх ♦х ♣Кхх, 13 очков, 5♠4♥)
Партнер: 1♥
Вы: 2♥ (поддержка 4♥)

✗ Слишком слабо для контры:
Противник: 1♥
Вы: Пас (♠Кххх ♥хх ♦Дххх ♣Тхх, 10 очков - недостаточно!)
```

### Когда контра НЕ вызывная?

❌ **Контра ШТРАФНАЯ (penalty) в этих случаях:**

1. **Партнер уже заявлял** (кроме паса)

   ```
   Партнер: 1♠
   Противник: 2♥
   Вы: Контра = ШТРАФНАЯ (не вызывная!)
   ```

2. **Они в гейме/шлеме**

   ```
   Противник: 1♠ - 4♠
   Вы: Контра = ШТРАФНАЯ
   ```

3. **После нашей заявки БК**
   ```
   Вы: 1NT
   Противник: 2♣
   Партнер: Контра = ШТРАФНАЯ
   ```

## 2. НЕГАТИВНАЯ КОНТРА (Negative Double)

**Применение:** 1♣/♦ - (1♥/♠) - **Контра**

**Тип:** Искусственная, показ других мастей

**Значение:** "У меня 6+hcp, нет фита в твоем миноре, но есть карты в других мастях (особенно в непоказанном мажоре)"

### Требования

- ✓ 6+hcp (минимум)
- ✓ НЕТ фита в миноре партнера
- ✓ Обычно 4+ карты в непоказанном мажоре
- ✓ Противник вмешался на 1 уровне

### Типичные руки

**После 1♦-(1♥)-X:**

- 4+♠ (показываем пику)
- Нет 5+♠ (иначе заявили бы 1♠)
- 6+hcp

**После 1♣-(1♠)-X:**

- 4+♥ и/или 5+♦
- Нет поддержки ♣
- 6+hcp

### Примеры

```
✅ Негативная контра (показываем ♠):
Партнер: 1♦
Противник: 1♥
Вы: Контра (♠Кххх ♥хх ♦Дхх ♣Вххх, 8 очков, 4♠, негативная)
Партнер: 1♠ (есть 4♠)
Вы: 2♠ (поддержка)

✅ Негативная контра (оба мажора):
Партнер: 1♣
Противник: 1♦
Вы: Контра (♠Кххх ♥Дххх ♦хх ♣Вхх, 8 очков, 44 мажора)
Партнер: 1♥ (есть 4♥)
Вы: 2♥

✅ Негативная контра (сильная):
Партнер: 1♦
Противник: 1♠
Вы: Контра (♠Кх ♥ТДхх ♦Дхх ♣Вххх, 12 очков, 4♥)
Партнер: 2♥
Вы: 3♥ (инвит, 12 очков)
Партнер: 4♥

✗ НЕ негативная (есть 5М):
Партнер: 1♦
Противник: 1♥
Вы: 1♠ (♠ТКххх ♥хх ♦Дхх ♣Вхх, 10 очков - не контра, 5♠!)
```

### Негативная контра на разных уровнях

| Уровень   | Сила   | Особенности        |
| --------- | ------ | ------------------ |
| **1 ур.** | 6+hcp  | Показ 4М           |
| **2 ур.** | 8+hcp  | Показ 4+М или силы |
| **3 ур.** | 10+hcp | Сильная рука       |

## 3. НАКАЗАТЕЛЬНАЯ КОНТРА (Penalty Double)

**Применение:** Контра на ФИНАЛЬНЫЙ контракт противников

**Значение:** "Хочу их бить! У меня защитные взятки"

### Когда контра штрафная?

**Автоматически штрафная:**

1. ✅ Партнер УЖЕ заявлял (кроме паса)
2. ✅ Противники в гейме (4+)
3. ✅ После нашего 1NT/2NT/3NT
4. ✅ После форсирующего паса
5. ✅ На искусственные заявки (иногда lead-directing)

### Требования для штрафной контры

**Что нужно для контры:**

- ✓ Козырные взятки (длина в их масти)
- ✓ Побочные взятки (тузы, короли)
- ✓ Ожидание положить их минимум на 1 (лучше на 2)

### Подсчет взяток для контры

**Правило:** Нужно ВИДЕТЬ взятки для положения

```
Они играют 4♠:
Нужно положить минимум 4♠X -1 = +100/200

Что нужно:
- 2 козырные взятки (♠ТК или ♠Т + ♠К у партнера)
- 2+ побочные взятки (♥Т, ♦Т, ♣ТК)
```

### Примеры

```
✅ Штрафная контра (очевидная):
Противники: 1♠ - 4♠
Вы: Контра (♠ТКДх ♥Кх ♦Дххх ♣Вхх, 14 очков, 4♠!)
Логика: 3 козырные + ♥К = минимум 4 взятки → должны положить

✅ Штрафная контра (партнер заявлял):
Партнер: 1♥
Противник: 2♠
Вы: Контра (♠КДхх ♥хх ♦ТКх ♣Дххх, 13 очков, ШТРАФНАЯ!)
Логика: У партнера сила в ♥, у меня ♠КД + ♦ТК = должны положить

✗ НЕ контруй без взяток:
Противники: 1♠ - 4♠
Вы: Пас (♠хх ♥ТКДх ♦Дххх ♣Кхх, 14 очков, но НЕТ козырных!)
Логика: Хоть 14 очков, но нет взяток в ♠ → могут сыграть
```

## 4. КОНТРА ПОДДЕРЖКИ (Support Double)

**Применение:** 1♣-1♥-(1♠)-**Контра**

**Значение:** "У меня РОВНО 3 карты в твоей масти ♥"

### Ключевое правило

> **Support Double показывает РОВНО 3 карты!**
>
> С 4+ картами → поднимаем масть напрямую

### Примеры

```
✅ Support Double:
Вы: 1♦
Партнер: 1♥ (♠Кх ♥ТКххх ♦Дх ♣Вххх, 11 очков)
Противник: 1♠
Вы: Контра (♠Дхх ♥Кхх ♦ТКххх ♣Дх, 14 очков, Support Double: РОВНО 3♥)
Партнер: 2♥ (принимаю с 5♥)

✅ Прямая поддержка (4+ карты):
Вы: 1♦
Партнер: 1♥
Противник: 1♠
Вы: 2♥ (♠Дх ♥Кххх ♦ТКххх ♣Дх, 14 очков, 4♥ - НЕ контра!)

✅ Support Redouble:
Вы: 1♣
Партнер: 1♠
Противник: Контра
Вы: Реконтра (♠Кхх ♥Тх ♦Дхх ♣ТКДхх, 15 очков, Support XX: РОВНО 3♠)
Партнер: 2♠ (принимаю)
```

## 5. ОТВЕТНАЯ КОНТРА (Responsive Double)

**Применение:** (1♥)-Контра-(2♥)-**Контра**

**Значение:** "У меня есть обе другие масти, выбирай"

### Применение

- ✓ Партнер сделал контру для вывода
- ✓ Противник поддержал масть
- ✓ У вас 6+hcp
- ✓ Карты в обеих непоказанных мастях

### Примеры

```
✅ Responsive Double:
Противник: 1♥
Партнер: Контра (для вывода)
Противник: 2♥ (поддержка)

Партнер: 2♠ (выбираю ♠)

✅ Responsive на высоком уровне:
Противник: 1♠
Партнер: Контра
Противник: 3♠ (блокирующая)
Вы: Контра (♠х ♥Кххх ♦Дххх ♣Тххх, 9 очков, Responsive)
Партнер: 4♥
```

## 6. МАКСИМАЛЬНАЯ КОНТРА (Maximal Double)

**Применение:** 1♥-2♣-2♥-3♣-**Контра**

**Значение:** "У меня инвит, но нет места для инвита заявкой"

### Когда применять

- ✓ Конкуренция в двух мастях
- ✓ Хотите сделать инвит
- ✓ Следующая ступень = гейм (нет места)

### Примеры


## 7. КОНТРА ЛАЙТНЕРА (Lightner Double)

**Применение:** Контра на ШЛЕМОВЫЙ контракт противников

**Значение:** "НЕ ходи стандартно! Нужен особый выход"

### Что просит контра Лайтнера?

**Обычно:**

1. **Ренонс в побочной масти** (их масть из торговли)
2. **ТКД в побочной масти** (можем сразу взять 2 взятки)
3. **Козырь на перебитие** (имею короткого козыря)

### Примеры

```
✅ Lightner (ренонс):
Противники: 1♠-3♠-4♣(кюбид)-4♦(кюбид)-6♠
Вы: Контра (♠хх ♥КДхххх ♦- ♣Тхххх, ренонс ♦!)
Партнер: Выходит ♦ (необычный выход)
Вы: Перебиваете ♥
Результат: Положили 6♠!

✅ Lightner (ТКД):
Противники: 1♥-3♥-4♣-4♦-6♥
Вы: Контра (♠хх ♥хх ♦ТКДхх ♣хххх, ТКД♦!)
Партнер: Выходит ♦
Результат: Взяли 2 взятки ♦, положили 6♥!
```

## 9. РЕКОНТРА СИЛОВАЯ (Strong Redouble)

**Применение:** 1М-(Контра)-**Реконтра**

**Значение:** "У меня 10+hcp, мы сильнее!"

### Требования

- ✓ 10+hcp
- ✓ Партнер открыл
- ✓ Противник законтровал
- ✓ НЕТ длинной масти для заявки

### Система после реконтры

**После 1♥-(X)-XX:**

Мы играем **PENALTY**:

- Все заявки = натуральные
- Контра партнера = ШТРАФНАЯ
- Цель: наказать противников

### Примеры

```

Противник: Контра

Противник: 1♠
Партнер: Пас
Вы: 2♦ (натурально, у нас сила)

✅ Реконтра с фитом:
Партнер: 1♠
Противник: Контра
Вы: Реконтра (♠Кххх ♥Дх ♦ТДхх ♣Вхх, 11 очков, фит + сила)
Противник: 2♥
Партнер: 2♠
Вы: 4♠ (у нас сила для гейма)
```

## 7.7. Forcing Pass (Форсирующий пас) - ПОДРОБНО

**Применение:** На высоких уровнях когда у партнерства есть сила для гейма

**Тип:** Искусственная, передача решения партнеру

**Основная идея:** "У нас достаточно силы для гейма/шлема. Я передаю решение тебе: контровать их или торговаться дальше?"

---

### Когда ПАС становится ФОРСИРУЮЩИМ?

**Обязательные условия (ВСЕ должны выполняться):**

1. **У нашей пары есть сила для гейма** (обычно 25+hcp в сумме)
2. **Противники торгуются на ВЫСОКОМ уровне** (обычно 4+ уровень)
3. **Мы уже показали силу** (открылись, пригласили гейм, или в гейме)
4. **Противники ВМЕШИВАЮТСЯ** в нашу конструктивную торговлю

> [!important]- Золотое правило форсирующего паса
> Если у нас достаточно силы для гейма, и мы УЖЕ находимся на уровне гейма (или выше), то ПАС = ФОРСИРУЮЩИЙ

---

### Ситуации форсирующего паса

#### **Ситуация 1: Мы в гейме, они вмешиваются**

```
Торговля:
Мы:    1♠ - 3♠ - 4♠ - ?
Они:   -  - 2♥ -    - 5♥

Значение ПАСА:
Пас = ФОРСИРУЮЩИЙ! "У нас гейм, решай ты: контра или 5♠?"
```

**Логика паса:**

- **Защитные взятки** → Контра (хочу бить 5♥)
- **Раскладная рука, мало защиты** → 5♠ (играем сами)
- **Средняя рука** → Пас (передаю решение)

#### **Ситуация 2: Мы заявили шлем, они жертвуют**

```
Торговля:
Мы:    1♠ - 4♠ - 6♠ - ?
Они:   -  - 2♥ -    - 7♥

Пас = ФОРСИРУЮЩИЙ! "У нас шлем 6♠, решай: 7♠ или контра?"
```

#### **Ситуация 3: Шлемовая торговля на высоком уровне**

```
Торговля:
Мы:    1♥ - 2NT - 3♣ - 4♥ - 4NT - ?
Они:   -  -     - 4♠ -     -     - 5♠

Мы: форсирующая торговля (Jacoby 2NT + RKCB)
Пас = ФОРСИРУЮЩИЙ! "Решай: контра или 6♥?"
```

---

### Ответы партнера на форсирующий пас

После форсирующего паса партнер ОБЯЗАН действовать:

| Заявка партнера       | Значение                                       |
| --------------------- | ---------------------------------------------- |
| **Контра**            | Штрафная! У меня защитные взятки, хочу их бить |
| **Следующий уровень** | Играем сами, у меня раскладная рука            |
| **Пас**               | Средняя рука, жду твоего решения дальше        |

---

### Детальные примеры

#### **Пример 1: Forcing Pass → Контра**

```
Торговля:
Север: 1♠
Противник: 2♥
Юг: 3♠ (инвит, 10-11hcp, 4♠)
Противник: Пас
Север: 4♠ (принимаю)
Противник: 5♥ (жертва!)
Юг: Пас (FORCING! Решай ты, партнер)

Север: ♠ТКДхх ♥Кх ♦ТДхх ♣Дхх (18 очков, 2♥ → защитные взятки!)
Север: Контра (у меня К♥ + козыри, должны положить)

Результат: 5♥X -3 = +500 (лучше чем наш 4♠ = +420)
```

#### **Пример 2: Forcing Pass → Играем дальше**

```
Торговля:
Север: 1♥
Противник: 1♠
Юг: 3♥ (блок, 4♥, слабая)
Противник: Пас
Север: 4♥
Противник: 4♠
Юг: Пас (FORCING!)

Север: ♠х ♥ТКВххх ♦ТКхх ♣Кх (16 очков, раскладная рука!)
Север: 5♥ (у меня синглет ♠, слабая защита, играем сами)

Результат: 5♥ = +450 (лучше чем контра 4♠X = +300)
```

#### **Пример 3: Forcing Pass → Второй форсирующий пас**

```
Торговля:
Север: 1♠
Противник: 2♥
Юг: 4♠ (хорошая рука)
Противник: 5♥
Север: Пас (FORCING! Средняя рука, решай ты)

Юг: ♠Дххх ♥Кх ♦ТКхх ♣Дхх (13 очков, средняя рука)
Юг: Пас (ТОЖЕ FORCING! Я тоже средний, решай ты обратно)

Север: Контра (ок, я решаю - контрую)
```

---

### Когда ПАС НЕ форсирующий?

❌ **Пас НЕ форсирующий в этих случаях:**

1. **Низкий уровень** (1-2 уровень) - пас = слабая рука

   ```
   1♥ - (2♣) - Пас = слабая рука, нет 6+hcp
   ```

2. **Нет силы для гейма** - пас = хочу остановиться

   ```
   1♠ - (2♥) - 2♠ - (3♥) - Пас = минимум, нет гейма
   ```

3. **Партнер не показал силу**
   ```
   (1♥) - Пас - (2♥) - Пас = слабая рука, не могу вмешаться
   ```

### Сравнительная таблица

| Ситуация           | Пас = ?     | Почему?                          |
| ------------------ | ----------- | -------------------------------- |
| 1♥-(2♣)-?          | НЕ FORCING  | Низкий уровень, неизвестная сила |
| 1♠-3♠-4♠-(5♥)-?    | **FORCING** | ✅ У нас гейм (25+), решаем      |
| 1♥-(2♠)-2NT-(3♠)-? | НЕ FORCING  | Частичная, нет гейма             |
| 1♠-2♠-4♠-(5♣)-?    | **FORCING** | ✅ В гейме, противники жертвуют  |
| (3♥)-4♠-(5♥)-?     | **FORCING** | ✅ Мы в гейме                    |

---

### Правила применения (чек-лист)

**Перед тем как пасовать на высоком уровне, спроси себя:**

1. ☑ У нас есть 25+hcp (сила для гейма)?
2. ☑ Мы на уровне 4+ (высокий уровень)?
3. ☑ Партнер показал конструктивную заявку?

**Если ДА на все 3 вопроса** → Пас = FORCING

**Если НЕТ хотя бы на один** → Пас = обычный (слабая рука)

---

### Психология форсирующего паса

**Зачем нужен форсирующий пас?**

1. **Передача информации:** Кто лучше знает свою руку - делает решение
2. **Защита от жертв:** Противники не могут дешево уйти
3. **Оптимальное решение:** Игрок с лучшей информацией решает

**Кто контрует?**

- Тот, у кого **ЗАЩИТНЫЕ ВЗЯТКИ** в их масти
- Тот, у кого **КОРОТКАЯ** козырная масть (2-3 карты)
- Тот, у кого **ФИГУРЫ** в побочных мастях

**Кто торгуется дальше?**

- Тот, у кого **РАСКЛАДНАЯ** рука (6+М, синглеты)
- Тот, у кого **ДЛИННЫЕ КОЗЫРИ**
- Тот, у кого **МАЛО ЗАЩИТЫ** в их масти

---

### Типичные ошибки

❌ **Ошибка 1:** Пасую с защитой → партнер торгуется → они сыграли

```
Правильно: С защитой → КОНТРА напрямую
```

## 7.8. Trump Ask (Вопрос о козырях)

**Заявка:** После трансфера следующая ступень

**Тип:** Искусственная, вопрос

**Значение:** "Какие у тебя козыри в моей масти?"

**Схема (после 1NT-2♦-2♥):**

- **2♠** = Trump Ask: "Какие ♥?"

**Ответы:**

| Заявка  | Козыри                |
| ------- | --------------------- |
| **2NT** | 2 карты, нет 3 онеров |
| **3♣**  | 3 карты, нет 2 онеров |
| **3♦**  | 2 онера (из ТКД)      |
| **3♥**  | 3 онера (ТКД)         |

**Примеры:**

```
✅ Trump Ask для шлема:
Север: 1NT (♠Кхх ♥ТКхх ♦ТДх ♣Дхх, 17 очков, 4♥)
Юг: 2♦ (♠Тх ♥ДВхххх ♦Кх ♣Кхх, 12 очков, трансфер)
Север: 2♥ (принимаю)
Юг: 2♠ (Trump Ask)
Север: 3♥ (3 онера в ♥ - ТКхх!)
Юг: 4NT (RKCB)
Юг: 6♥ (отличные козыри!)
```

## 7.9. Weak Jump Overcall (Слабый прыжок)

**Заявка:** (1♣)-**2♠** или (1♦)-**3♣**

**Тип:** Натуральная, блокирующая

**Значение:** "У меня 5-10hcp и 6-карточная масть, блокирую"

**Когда применять:**

- ✓ Противник открыл
- ✓ У вас 5-10hcp
- ✓ 6-карточная масть (хорошая)
- ✓ Цель: блокировка

**Примеры:**

```
✅ Weak Jump Overcall:
Противник: 1♣
Вы: 2♠ (♠ДВхххх ♥хх ♦Кх ♣хх, 7 очков, блок)

✅ Weak Jump на 3 уровне:
Противник: 1♦
Вы: 3♣ (♠Кх ♥хх ♦х ♣ДВххххх, 8 очков, блок)
```

## 7.10. Soloway Jump Shift (Солоуэй)

**Заявка:** 1♣-**2♠** (прыжок в масть, которую пропустили)

**Тип:** Искусственная, показ фита

**Значение:** "У меня фит в твоей масти И силу в пропущенной масти"

**Примеры:**

```
✅ Soloway Jump Shift:
Север: 1♣
Юг: 2♠ (♠ТКДх ♥хх ♦Дхх ♣Тххх, Soloway: фит ♣ + сила ♠)
Север: 3♣
Юг: 3NT или 5♣
```

## 7.12. Weak Two (Слабая двойка)

**Заявки:** **2♦/2♥/2♠** (открытие)

**Тип:** Натуральная, блокирующая

**Значение:**

- **2♦** = 5-10hcp, 54М (мажоры) - НАША ВЕРСИЯ
- **2♥** = 5-10hcp, 6♥ (нет 4♠)
- **2♠** = 5-10hcp, 6♠ (нет 4♥)

**Когда применять:**

- ✓ 5-10hcp (широкий диапазон)
- ✓ 6-карточная масть (хорошая)
- ✓ Цель: блокировка торговли

**Ответы партнера:**

- **2NT** = вопрос о силе и краткости (OGUST)
- **3М** = пас или корректируй
- **4М** = к игре
- **Новая масть** = натурально, форсирующая

**Примеры:**

```
✅ Weak Two:
Вы: 2♥ (♠хх ♥ТКВххх ♦Кх ♣хх, 9 очков, 6♥)
Партнер: 4♥ (хорошая рука, идем в гейм)

✅ Multi 2♦ (наша версия):
Вы: 2♦ (♠ТКххх ♥ДВххх ♦х ♣хх, 9 очков, 54М)
Партнер: 2♥ (предпочтение, пас или корректируй)
Вы: 2♠ (корректирую в ♠)
```
