> [!info]+
> NF, 6-10hcp, BAL, <font color="#d83931">нет 4М</font> (<font color="#9bbb59">м.б 4М333</font>)

## Примеры рук отвечающего
```
♠Кхх ♥Дхх ♦хх ♣Вхххх (7 очков, нет 4М)
♠Вхх ♥Кхх ♦Дхх ♣хххх (6 очков, 4333 с ♥)
♠Дхх ♥Вхх ♦Кхх ♣Дхх (9 очков, равномер)
```

## Заявки открывающего (1♦-1NT-?)

### Пас = 11-14hcp, BAL, нет 5♦
```
Рука открывающего: ♠Кхх ♥Дхх ♦ТВхх ♣Кхх (12 очков)
```
**К игре:** 1NT (может быть опасный контракт, но безопаснее чем 2♦)

### 2♣ = NF, 11-14hcp, 5♦4♣
```
Рука открывающего: ♠Кх ♥хх ♦ТДххх ♣КДхх (12 очков)
```
**Развитие отвечающего:**
- **Пас** = к игре, нет интереса к ♦
- **2♦** = предпочтение, к игре
- **2NT** = INV → 3NT, 9-10hcp
- **3♣** = INV, фит в ♣, 9-10hcp
- **3♦** = INV, фит в ♦, 9-10hcp
- **3NT** = G

### 2♦ = NF, 11-14hcp, 6+♦
```
Рука открывающего: ♠Кх ♥Дх ♦ТКДххх ♣Вхх (12 очков)
```
**Развитие отвечающего:**
- **Пас** = к игре, слабая рука
- **2NT** = INV → 3NT, 9-10hcp
- **3♦** = INV, фит в ♦, 9-10hcp
- **3NT** = G

### 2♥/2♠ = F1, 16+hcp, 5♦4М (реверс)
```
Рука открывающего (2♥): ♠Кх ♥ТКхх ♦ТДВхх ♣Дх (17 очков)
```
**Развитие отвечающего:**
- **2♠** (после 2♥) = слабая рука, выход через 2NT→3♣
- **2NT** = слабая рука (6-8hcp)
- **3♣♦** = GF, натурально, фит
- **3♥/♠** = GF, держка в масти
- **3NT** = G
- Все остальное = GF

### 2NT = NF, 18-19hcp, BAL
```
Рука открывающего: ♠ТКх ♥Дхх ♦ТДхх ♣Кхх (18 очков)
```
**Развитие отвечающего:**
- **Пас** = 6-7hcp, минимум
- **3♣♦** = натурально, 5+ карт, поиск фита
- **3NT** = G, 8+hcp

### 3♣ = INV, 15-17hcp, 5♦4♣
```
Рука открывающего: ♠Кх ♥хх ♦ТКххх ♣ТДхх (16 очков)
```
**Развитие отвечающего:**
- **3♦** = предпочтение ♦
- **3NT** = G, нет фита
- **4♣** = фит, шлемовые амбиции (крайне редко)
- **Пас** = слабая рука с фитом ♣

### 3♦ = INV, 15-17hcp, 6+♦
```
Рука открывающего: ♠Кх ♥хх ♦ТКДВхх ♣Дхх (16 очков)
```
**Развитие отвечающего:**
- **Пас** = слабая рука, 6-7hcp
- **3NT** = G, 8+hcp, нет фита
- **4♦** = фит, шлемовые амбиции (редко)

### 3NT = G, 18-21hcp
```
Рука открывающего: ♠Кх ♥Дх ♦ТКДВххх ♣Кх (18 очков)
```
**К игре:** 3NT
