> [!info]+
> **НА 1-2 руке:** GF, 12+hcp, равномер или 5+♣ или фит в ♠
> **НА 3-4 руке (после паса):** Друри, максимум паса 6-10hcp с фитом 3+♠

> [!warning]- ВАЖНО: Две разные конвенции!
> - **1♠ (на 1-2 руке) - 2♣** = GF, сильная рука
> - **пас-пас-1♠ - 2♣** = Друри, максимум паса с фитом
> 
> Не путай эти две ситуации!

## Вариант А: Открытие на 1-2 руке (GF заявка)

### Примеры рук отвечающего (GF)
```
♠Тхх ♥Кх ♦Кхх ♣ДВххx (13 очков, 5♣)
♠КДх ♥Дхх ♦Тхх ♣Вххх (12 очков, равномер с фитом)
♠Кх ♥Дхх ♦Тхх ♣ТКххх (14 очков, 5♣, GF сила)
```

### Заявки открывающего (1♠-2♣-?)

**Развитие аналогично 1♥-2♣, но с заменой мастей:**
- **2♦** = NF, 11-15hcp, 5♠4+♦
- **2♥** = NF, 11-15hcp, 5♠4+♥
- **2♠** = NF, 11-14hcp, 6+♠
- **2NT** = GF, 15-17hcp, BAL
- **3♣** = NF, 11-14hcp, 4+♣
- **3♦** = INV, 15-17hcp, 5♠5+♦
- **3♥** = INV, 15-17hcp, 5♠5+♥
- **3♠** = INV, 15-17hcp, 6+♠
- **3NT** = G, 18-21hcp
- **4♣/4♦/4♥** = GF, 18+hcp, Сплинтер

## Вариант Б: Друри (после паса-паса-1♠)

> [!note]- Конвенция Друри
> **Применение:** ТОЛЬКО после паса-паса-1♠-2♣
> 
> **Цель:** Показать максимум паса (6-10 очков) с фитом в пике (3-4 карты)
> 
> **Критерии:**
> - ✓ Открытие на 3-4 руке
> - ✓ У нас максимум паса (6-10 очков)
> - ✓ Есть 3-4 карты в ♠

### Примеры рук отвечающего (Друри)
```
♠Кххх ♥хх ♦ххх ♣Вххх (7 очков, максимум паса, 3♠)
♠ДВхх ♥Кхх ♦хх ♣ххх (9 очков, максимум паса, хороший фит)
♠Тххх ♥Дх ♦хххх ♣Вхх (8 очков, максимум паса, 4♠)
```

### Заявки открывающего (пас-пас-1♠-2♣-?)

| Заявка | Значение |
|--------|----------|
| **2♦** | Нет гейма, пасуй на любую заявку |
| **2♥** | F1, ИНВ, 4+♥ 5♠ |
| **2♠** | F1, ИНВ, есть интерес к гейму |
| **2NT** | ИНВ, 13-14hcp, 5332 |
| **3♣** | F1, ИНВ, 4+♣ 5♠ |
| **3♦** | F1, ИНВ, 4+♦ 5♠ |
| **3♥** | GF, Сплинтер, краткость ♥ |
| **3♠** | ИНВ, 6+♠ |
| **3NT** | НФ, на выбор, 5332, есть гейм |
| **4♣/4♦** | GF, Сплинтер, краткость ♣/♦ |
| **4♠** | G, к игре |

## Примеры торговли (Друри)

**Пример 1: 2♦ (нет гейма)**
```
Север: ♠ДВххх ♥Кхх ♦Тхх ♣хх (10 очков)
Юг: ♠Кххх ♥хх ♦хххх ♣Вх (7 очков)

Торговля:
пас - пас - 1♠ (Север) - 2♣ (Друри, Юг)
2♦ (нет гейма) - Пас
```

**Пример 2: 2♠ (инвит принят)**
```
Север: ♠ТКххх ♥Дхх ♦Кхх ♣Вх (12 очков)
Юг: ♠Дххх ♥Тх ♦Вхх ♣Кхх (10 очков, максимум!)

Торговля:
пас - пас - 1♠ (Север) - 2♣ (Друри, Юг)
2♠ (инвит) - 4♠ (принимаем с максимумом)
```

**Пример 3: 3♥ (сплинтер)**
```
Север: ♠ТКДхх ♥х ♦Кхх ♣ТДхх (16 очков)
Юг: ♠Дххх ♥Кхх ♦Тхх ♣Вхх (10 очков)

Торговля:
пас - пас - 1♠ (Север) - 2♣ (Друри, Юг)
3♥ (сплинтер, хорошая рука) - 4♠ (идем в гейм)
```

**Пример 4: 2♥ (инвит с 4♥5♠)**
```
Север: ♠ТДххх ♥ТКхх ♦Кх ♦Дх (14 очков, 5♠4♥)
Юг: ♠Кхх ♥Дхх ♦Тхх ♣Вххх (9 очков)

Торговля:
пас - пас - 1♠ (Север) - 2♣ (Друри, Юг)
2♥ (инвит, 5♠4♥) - 3♥ (фит в ♥, хорошая рука)
Север: 4♥ (гейм)
```

## Примеры торговли (GF на 1-2 руке)

**Пример 1: Поиск фита**
```
Север: ♠ТКххх ♥хх ♦Кхх ♣Вх (12 очков)
Юг: ♠Дхх ♥Кх ♦Тхх ♣ТКххх (14 очков, 5♣, фит 3♠)

Торговля:
Север: 1♠
Юг: 2♣ (GF, 5+♣ или равномер с фитом)
Север: 2♠ (6♠, 11-14hcp)
Юг: 4♠ (гейм с фитом)
```

**Пример 2: Поиск 3NT**
```
Север: ♠ТКххх ♥Кх ♦Дхх ♣Вхх (13 очков)
Юг: ♠хх ♥Дхх ♦Тхх ♣ТКДхх (14 очков, 5♣, нет фита)

Торговля:
Север: 1♠
Юг: 2♣ (GF, 5+♣)
Север: 2♠ (6♠, 11-14hcp)
Юг: 2NT (задержка, поиск 3NT)
Север: 3NT (есть держки)
```
