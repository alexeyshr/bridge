> [!info]+
> Стейман, 0+hcp, вопрос о мажорных четверках

> [!tip]- Когда применять Стейман?
>
> - ✓ У вас есть 4 карты в мажоре (или 44 в мажорах)
> - ✓ Хотите найти фит 4-4
> - ✗ НЕ применяйте с мажором 5+! Используйте трансфер

## Примеры рук отвечающего

```
♠Кхх ♥Дххх ♦Тхх ♣Вхх (9 очков, ищем фит в ♥)
♠ТДхх ♥Кххх ♦хх ♣Дхх (11 очков, 44 в мажорах)
♠ДВхх ♥Кххх ♦хх ♣Тхх (10 очков, ищем фит 4-4)
♠Кххх ♥Дххх ♦хх ♣ххх (5 очков, даже слабые руки могут!)
```

## Ответы открывающего (1NT-2♣-?)

### 2♦ = Нет четверок в мажорах

```
Рука открывающего: ♠Кхх ♥Дхх ♦ТВхх ♣ТКх (16 очков, 4333)
```

**Развитие отвечающего:**

- **2♥** = НФ, 0-7hcp, 4-4 в мажорах (слабая рука, выбрали ♥)
- **2♠** = НФ, 0-7hcp, 4-4 в мажорах (слабая рука, выбрали ♠)
- **2NT** = INV, 8-9hcp
- **3♣♦** = GF, 5+♣/♦, поиск фита/3NT
- **3♥** = GF, 4♥5+♠ (Смолен)
- **3♠** = GF, 4♠5+♥ (Смолен)
- **3NT** = G, 10-15hcp
- **4♥** = GF, 4♥6+♠ (редко)
- **4♠** = GF, 4♠6+♥ (редко)

> [!note]- Что такое Смолен?
> **Смолен** - после Стеймана показ 4 карт в одном мажоре и 5+ в другом.
>
> - **3♥** = 4♥ и 5+♠ (показываем короткий мажор)
> - **3♠** = 4♠ и 5+♥ (показываем короткий мажор)
>
> Открывающий выбирает масть для гейма.

### 2♥ = Есть 4♥, может быть 4♠

```
Рука открывающего: ♠Кхх ♥ТДхх ♦ТВх ♣Кхх (16 очков)
ИЛИ: ♠ТКхх ♥ДВхх ♦Тхх ♣Кх (15 очков, 44 в мажорах)
```

**Развитие отвечающего:**

- **2♠** = вопрос: есть ли 4♠? (у меня 4♠, но может не быть 4♥)
  - Открывающий: **2NT** = нет 4♠, есть только 4♥
  - Открывающий: **3♥** = есть 4♠ тоже (44 мажоры)
- **2NT** = INV, 8-9hcp, нет 4♥
- **3♥** = INV, 8-9hcp, фит 4♥
- **3NT** = G, 10-15hcp, нет 4♥
- **4♥** = G, фит 4♥

### 2♠ = Есть 4♠, нет 4♥

```
Рука открывающего: ♠ТКхх ♥Дхх ♦ТВх ♣Кхх (16 очков)
```

**Развитие отвечающего:**

- **2NT** = INV, 8-9hcp, нет 4♠
- **3♠** = INV, 8-9hcp, фит 4♠
- **3NT** = G, 10-15hcp, нет 4♠
- **4♠** = G, фит 4♠

## Примеры торговли

**Пример 1: Нашли фит 4-4**

```
Север: ♠Кхх ♥ТДхх ♦ТВх ♣Кхх (16 очков)
Юг: ♠Дххх ♥Кххх ♦хх ♣Дхх (10 очков, 44 мажоры)

Торговля:
Север: 1NT
Юг: 2♣ (Стейман)
Север: 2♥ (есть 4♥, может быть 4♠)
Юг: 4♥ (гейм с фитом)
```

**Пример 2: Не нашли фит, играем 3NT**

```
Север: ♠Кхх ♥Дхх ♦ТВхх ♣ТКх (16 очков, нет 4М)
Юг: ♠Дххх ♥Кххх ♦хх ♣Дхх (10 очков, 44 мажоры)

Торговля:
Север: 1NT
Юг: 2♣ (Стейман)
Север: 2♦ (нет 4М)
Юг: 3NT (гейм без фита)
```

**Пример 3: Слабая рука со Стейманом**

```
Север: ♠Кхх ♥Дхх ♦ТВхх ♣ТКх (16 очков)
Юг: ♠Дххх ♥Кххх ♦хх ♣ххх (5 очков, 44 мажоры)

Торговля:
Север: 1NT
Юг: 2♣ (Стейман, даже с 5hcp!)
Север: 2♦ (нет 4М)
Юг: 2♥ (к игре, выбрали ♥)
```

**Пример 4: Смолен (4♥5+♠)**

```
Север: ♠Кхх ♥Дхх ♦ТВхх ♣ТКх (16 очков)
Юг: ♠ТКххх ♥Дххх ♦хх ♣Дх (11 очков, 5♠4♥)

Торговля:
Север: 1NT
Юг: 2♣ (Стейман)
Север: 2♦ (нет 4М)
Юг: 3♥ (Смолен: 4♥5+♠)
Север: 3♠ (выбираю ♠, у меня Кхх)
Юг: 4♠ (гейм)
```

**Пример 5: Уточнение 4♠ после 2♥**

```
Север: ♠ТКхх ♥ДВхх ♦Тхх ♣Кх (15 очков, 44 мажоры)
Юг: ♠Дххх ♥хх ♦ТДх ♣Вххх (10 очков, 4♠ нет 4♥)

Торговля:
Север: 1NT
Юг: 2♣ (Стейман)
Север: 2♥ (есть 4♥, может быть 4♠)
Юг: 2♠ (вопрос: есть ли 4♠?)
Север: 3♥ (есть 4♠ тоже!)
Юг: 4♠ (гейм с фитом в пике)
```

**Пример 6: Инвит с фитом**

```
Север: ♠Кхх ♥ТДхх ♦ТВх ♣Кхх (15 очков, минимум 1NT)
Юг: ♠Дххх ♥Кххх ♦хх ♣Дхх (9 очков, 44 мажоры)

Торговля:
Север: 1NT
Юг: 2♣ (Стейман)
Север: 2♥ (есть 4♥)
Юг: 3♥ (инвит с 9hcp)
Север: Пас (минимум 1NT)
```
