> [!info]+
> Шлемовая торговля - специальные конвенции и подходы для достижения контрактов 6 и 7 уровня (малый и большой шлем)

## Когда думать о шлеме?

### Условия для малого шлема (6 уровень):
- **Силовые:** 33+ очков в сумме у пары
- **Раскладные:** Фит 8+ карт + раскладные бонусы (синглеты, ренонсы)
- **Контроли:** Нет двух быстрых потерь в одной масти

### Условия для большого шлема (7 уровень):
- **Силовые:** 37+ очков в сумме у пары
- **Раскладные:** Отличный фит + длинные масти
- **Контроли:** Нужны ВСЕ тузы и почти все короли

## Базовые подходы к шлемовой торговле

### 1. Прямой путь (Количественный инвит)
```
Когда: Есть фит, сбалансированные руки, просто вопрос силы
Метод: 4NT как количественный инвит (не Блэквуд!)
```

**Пример:**
```
Рука Севера: ♠ТКхх ♥Дхх ♦ТВх ♣Кхх (16 очков)
Рука Юга: ♠Дххх ♥Кхх ♦Кхх ♣Дхх (13 очков)

Торговля:
Север: 1NT
Юг: 2♥ (трансфер в ♠)
Север: 2♠
Юг: 4NT (количественный инвит, у меня 13hcp, нужен максимум для шлема)
Север: 6♠ (принимаю с 16hcp)
```

### 2. Блэквуд (4NT = вопрос о тузах)
```
Когда: Согласована масть, достаточно силы, проверяем контроли
Метод: 4NT = вопрос о тузах
```

### 3. Кюбиды (Cue-bidding)
```
Когда: Серьезный шлемовый интерес, масть согласована
Метод: Показываем контроли по мастям (тузы и короли)
```

## Конвенция Блэквуд (4NT)

> [!tip]- Когда применять Блэквуд?
> **✓ ПРИМЕНЯЙТЕ:**
> - Масть согласована
> - У вас достаточно силы для шлема
> - Нужно проверить количество тузов
> 
> **✗ НЕ ПРИМЕНЯЙТЕ:**
> - Масть НЕ согласована
> - 4NT = количественный инвит (после БК торговли)
> - У вас есть быстрые потери в побочных мастях

### Ответы на Блэквуд (4NT-?)

| Заявка | Количество тузов |
|--------|------------------|
| **5♣** | 0 или 4 туза |
| **5♦** | 1 туз |
| **5♥** | 2 туза |
| **5♠** | 3 туза |

### Продолжение после ответа: 5NT = вопрос о королях

**Условие:** Можно спрашивать королей ТОЛЬКО если у пары есть ВСЕ тузы!

| Заявка | Количество королей |
|--------|-------------------|
| **6♣** | 0 или 4 короля |
| **6♦** | 1 король |
| **6♥** | 2 короля |
| **6♠** | 3 короля |

### Примеры с Блэквудом

**Пример 1: Есть все тузы, идем в шлем**
```
Север: ♠ТКДхх ♥Кхх ♦Тхх ♣Кх (18 очков)
Юг: ♠Дххх ♥Тххх ♦Кхх ♣Дх (11 очков, открыл 1♥)

Торговля:
Юг: 1♥
Север: 1♠
Юг: 2♠ (фит 4♠)
Север: 4NT (Блэквуд, у меня ~29-30 очков в сумме)
Юг: 5♦ (1 туз - ♥Т)
Север: 6♠ (отлично, у нас ТТ♠ + ТТ♥, идем в малый шлем)
```

**Пример 2: Не хватает туза, останавливаемся**
```
Север: ♠ТКДхх ♥Кхх ♦Тхх ♣Кх (18 очков)
Юг: ♠Дххх ♥Кххх ♦Кхх ♣Дх (10 очков, нет тузов!)

Торговля:
Юг: 1♥
Север: 1♠
Юг: 2♠ (фит 4♠)
Север: 4NT (Блэквуд)
Юг: 5♣ (0 тузов)
Север: 5♠ (стоп, не хватает 2 тузов, играем 5♠)
```

**Пример 3: Все тузы есть, спрашиваем королей**
```
Север: ♠ТКДхх ♥Тхх ♦Тхх ♣Кх (20 очков)
Юг: ♠Дххх ♥Кххх ♦Кхх ♣Тх (11 очков)

Торговля:
Юг: 1♥
Север: 1♠
Юг: 2♠ (фит 4♠)
Север: 4NT (Блэквуд)
Юг: 5♥ (2 туза - ♣Т и ♦Т)
Север: 5NT (спрашиваю королей, у нас все тузы!)
Юг: 6♦ (1 король - ♥К)
Север: 6♠ (идем в малый шлем, у нас 4 туза + 3 короля, но не все)
```

**Пример 4: Большой шлем!**
```
Север: ♠ТКДхх ♥Тхх ♦Тхх ♣Кх (20 очков)
Юг: ♠Дххх ♥КДхх ♦Кхх ♣Тх (13 очков, все короли!)

Торговля:
Юг: 1♥
Север: 1♠
Юг: 2♠ (фит 4♠)
Север: 4NT (Блэквуд)
Юг: 5♥ (2 туза)
Север: 5NT (спрашиваю королей)
Юг: 6♠ (3 короля!)
Север: 7♠ (БОЛЬШОЙ ШЛЕМ! У нас все тузы и все короли!)
```

## Кюбиды (Cue-bidding)

> [!tip]- Что такое кюбиды?
> **Кюбиды** - это способ показать контроли (тузы и короли) в конкретных мастях после согласования основной масти.
> 
> **Принцип:** Говорим масть, в которой есть контроль (туз или король), начиная с самой младшей.
> 
> **Когда применять:** Когда масть согласована и есть шлемовые амбиции, но Блэквуд не подходит (например, есть быстрые потери).

### Правила кюбидов

1. **Масть ДОЛЖНА быть согласована** до начала кюбидов
2. **Первый контроль = туз или король** (не уточняется)
3. **Повторный кюбид = король** (туз уже показан)
4. **Пропуск масти = НЕТ контроля** в этой масти
5. **Возврат в согласованную масть = стоп**, нет дальнейших контролей

### Примеры кюбидов

**Пример 1: Простые кюбиды**
```
Север: ♠ТКДхх ♥Кх ♦ТВхх ♣Дх (18 очков)
Юг: ♠Дххх ♥Дххх ♦Кх ♣Кхх (11 очков)

Торговля:
Север: 1♠
Юг: 2♠ (фит)
Север: 3♣ (кюбид, есть контроль в ♣ - Д♣)
Юг: 3♦ (кюбид, есть контроль в ♦ - К♦)
Север: 3♥ (кюбид, есть контроль в ♥ - К♥)
Юг: 4♠ (стоп, нет контроля в ♥, не хватает для шлема)
```

**Пример 2: Кюбиды приводят к шлему**
```
Север: ♠ТКДхх ♥Тх ♦ТВхх ♣Кх (19 очков)
Юг: ♠Дххх ♥Кххх ♦Кх ♣Тхх (12 очков)

Торговля:
Север: 1♠
Юг: 2♠ (фит)
Север: 3♣ (кюбид, К♣)
Юг: 3♦ (кюбид, К♦)
Север: 3♥ (кюбид, Т♥)
Юг: 4♣ (кюбид, Т♣)
Север: 4♥ (повторный кюбид ♥ - значит у меня был туз, теперь спрашиваю про короля)
Юг: 4♠ (нет К♥)
Север: 4NT (Блэквуд, проверяю тузов)
Юг: 5♥ (2 туза)
Север: 6♠ (отлично, идем в малый шлем!)
```

**Пример 3: Стоп на кюбидах**
```
Север: ♠ТКДхх ♥Кх ♦ТВхх ♣хх (16 очков, нет контроля ♣)
Юг: ♠Дххх ♥Дххх ♦хх ♣Кхх (10 очков)

Торговля:
Север: 1♠
Юг: 2♠ (фит)
Север: 3♦ (кюбид, пропустил ♣ - НЕТ контроля!)
Юг: 3♥ (кюбид, есть Д♥)
Север: 4♠ (стоп, нет контроля в ♣, быстрые потери)
```

## RKCB (Roman Key Card Blackwood)

> [!note]- Что такое RKCB?
> **RKCB** - продвинутая версия Блэквуда, где король козырной масти считается "пятым тузом".
> 
> **Преимущество:** Позволяет точнее оценить шлемовый потенциал при наличии фита.

### Ответы на RKCB (4NT-?)

| Заявка | Ключевые карты (4 туза + К козыря) |
|--------|-------------------------------------|
| **5♣** | 0 или 3 ключевые карты |
| **5♦** | 1 или 4 ключевые карты |
| **5♥** | 2 или 5 без Д козыря |
| **5♠** | 2 или 5 с Д козыря |

### Продолжение: Вопрос о Д козыря (если нужно)

**После 5♣/5♦:** Следующая ступень (кроме согласованной масти) = вопрос о Д козыря

**Ответы:**
- **1-я ступень** = НЕТ Д козыря
- **2-я ступень** = ЕСТЬ Д козыря

### Примеры RKCB

**Пример 1: RKCB с хорошим результатом**
```
Север: ♠ТКДхх ♥Кх ♦ТВхх ♣Дх (19 очков)
Юг: ♠Дххх ♥Тххх ♦Кх ♣Кхх (11 очков)

Торговля:
Север: 1♠
Юг: 2♠ (фит)
Север: 4NT (RKCB)
Юг: 5♠ (2 ключевые карты С Д♠ - это Т♥ и К♠)
Север: 6♠ (отлично, у нас все контроли!)
```

**Пример 2: RKCB, проверка Д козыря**
```
Север: ♠ТКххх ♥Кх ♦ТВхх ♣Дх (17 очков)
Юг: ♠Дххх ♥Тххх ♦Кх ♣Кхх (11 очков)

Торговля:
Север: 1♠
Юг: 2♠ (фит)
Север: 4NT (RKCB)
Юг: 5♦ (1 или 4 ключевые карты - у меня 1, это Т♥)
Север: 5♥ (вопрос о Д♠)
Юг: 5♠ (НЕТ Д♠)
Север: Пас (стоп, у нас только 3 ключевых карты)
```

## Количественный инвит (4NT без согласованной масти)

> [!warning]- ВАЖНО: 4NT - это НЕ всегда Блэквуд!
> Если масть НЕ согласована, 4NT = **количественный инвит** к малому шлему в БК.
> 
> **Значение:** "У меня примерно 16-17hcp, если у тебя максимум - идем в 6NT"

### Примеры количественного инвита

**Пример 1: Инвит принят**
```
Север: ♠Кхх ♥Дхх ♦ТВхх ♣ТКх (17 очков, максимум 1NT)
Юг: ♠Дхх ♥Кхх ♦Кхх ♣ДВхх (16 очков)

Торговля:
Север: 1NT
Юг: 4NT (количественный инвит, у меня 16hcp, нужен максимум)
Север: 6NT (принимаю с 17hcp, у нас 33 очка!)
```

**Пример 2: Инвит отклонен**
```
Север: ♠Кхх ♥Дхх ♦ТВхх ♣ТКх (15 очков, минимум 1NT)
Юг: ♠Дхх ♥Кхх ♦Кхх ♣ДВхх (16 очков)

Торговля:
Север: 1NT
Юг: 4NT (количественный инвит)
Север: Пас (отклоняю с 15hcp, у нас только 31 очко)
```

**Пример 3: Прямой 6NT с большой силой**
```
Север: ♠Кхх ♥Дхх ♦ТВхх ♣ТКх (16 очков)
Юг: ♠Дхх ♥Кхх ♦Кхх ♣ДВхх (18 очков)

Торговля:
Север: 1NT
Юг: 6NT (прямо в шлем, у меня 18hcp, даже с минимумом у нас 33+)
```

## Автосплинтеры (Splinters)

> [!tip]- Что такое сплинтер?
> **Сплинтер** - это прыжковая заявка, показывающая:
> - Фит в масти партнера (обычно 4+ карты)
> - Синглет или ренонс в заявленной масти
> - Шлемовые амбиции (обычно 11-15hcp)

### Когда применять сплинтеры

**✓ Применяйте:**
- Есть фит 4+ карты
- Есть синглет или ренонс в побочной масти
- Сила 11-15hcp (раскладные руки)

**✗ НЕ применяйте:**
- Если у вас слишком много очков (16+) - используйте другие подходы
- Если у вас 2 синглета - неясно, что показывать

### Виды сплинтеров

**1. Прямые сплинтеры (после открытия 1М)**
```
1♥ - 3♠/4♣/4♦ = сплинтер, краткость в заявленной масти, фит 4+♥
1♠ - 3♥/4♣/4♦ = сплинтер, краткость в заявленной масти, фит 4+♠
```

**2. Автосплинтеры (после трансфера из 1NT)**
```
1NT - 2♦ - 2♥ - 3♠/4♣/4♦ = автосплинтер, 6+♥, краткость
1NT - 2♥ - 2♠ - 3♥/4♣/4♦ = автосплинтер, 6+♠, краткость
```

### Примеры сплинтеров

**Пример 1: Прямой сплинтер приводит к шлему**
```
Север: ♠ТКДхх ♥Кхх ♦ТВхх ♣х (17 очков)
Юг: ♠Дххх ♥Тххх ♦Кх ♣Тхх (12 очков)

Торговля:
Север: 1♠
Юг: 4♣ (сплинтер, краткость ♣, фит 4♠)
Север: 4NT (RKCB, отлично, у меня синглет ♣ тоже!)
Юг: 5♦ (1 ключевая карта - Т♥)
Север: 6♠ (у нас 3 ключевых карты + отличный фит, идем!)
```

**Пример 2: Сплинтер, но потери в другой масти**
```
Север: ♠ТКДхх ♥хх ♦ТВхх ♣Кх (16 очков, 2 мелких ♥!)
Юг: ♠Дххх ♥Тххх ♦Кх ♣Тхх (12 очков)

Торговля:
Север: 1♠
Юг: 4♣ (сплинтер, краткость ♣)
Север: 4♠ (стоп, у меня хх♥ - быстрые потери)
```

**Пример 3: Автосплинтер после трансфера**
```
Север: ♠Кхх ♥ТДхх ♦ТКх ♣Дхх (17 очков, максимум 1NT)
Юг: ♠х ♥КВхххх ♦Дх ♣ТКхх (14 очков, 6♥, синглет ♠)

Торговля:
Север: 1NT
Юг: 2♦ (трансфер в ♥)
Север: 2♥ (принимаю)
Юг: 3♠ (автосплинтер, краткость ♠, 6♥)
Север: 4♣ (кюбид, согласован ♥, интерес к шлему)
Юг: 4♦ (кюбид)
Север: 4NT (RKCB)
Юг: 5♠ (2 ключевых карты с Д♥)
Север: 6♥ (шлем!)
```

## Чек-лист перед шлемом

✅ **Проверь силу:**
- [ ] У нас 33+ очков в сумме?
- [ ] Есть фит 8+ карт в козырной масти?

✅ **Проверь контроли:**
- [ ] Нет двух быстрых потерь в одной масти?
- [ ] Есть тузы и короли в критичных мастях?

✅ **Выбери метод:**
- [ ] Если нужно проверить тузы → Блэквуд/RKCB
- [ ] Если нужно показать контроли → Кюбиды
- [ ] Если БК без фита → Количественный инвит 4NT

✅ **Помни:**
- [ ] 4NT ≠ всегда Блэквуд
- [ ] После кюбидов можно использовать Блэквуд
- [ ] Сплинтеры помогают оценить раскладную силу
