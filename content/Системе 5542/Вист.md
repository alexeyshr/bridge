> [!info]+
> Вист - торговля в условиях помех со стороны противников. Включает интервенцию, контру, реконтру и защитные заявки.

> [!tip]- Смотри также
> - [[Первый ход (Атака)]] - Что выходить с разных рук
> - [[Сигналы в защите]] - Как общаться с партнером через сброс карт

## Базовые принципы виста

### 1. Закон общего количества взяток (ЗОВ)
```
ПРАВИЛО: Количество безопасных взяток = количество козырей у обеих пар

Пример: У нас 9♠, у них 9♥ → безопасно торговаться до 9 уровня (18 карт)
```

### 2. Правило 2-3-4
```
ПРАВИЛО: Блокируй противников на:
- 2 уровня с уязвимостью
- 3 уровня без уязвимости (они)
- 4 уровня без уязвимости (мы и они)
```

### 3. Быстро или медленно?
```
ПРАВИЛО:
- Быстрый выход на высокий уровень = слабая блокирующая рука
- Медленная торговля = сильная конструктивная рука
```

## Интервенция (вмешательство)

> [!tip]- Когда вмешиваться?
> **✓ ВМЕШИВАЙТЕСЬ:**
> - Хорошая 5-карточная масть (с фигурами)
> - Примерно 8-17hcp (но возможны исключения)
> - Готовы к последствиям (контре)
> 
> **✗ НЕ ВМЕШИВАЙТЕСЬ:**
> - Слабая масть без фигур
> - Сбалансированная рука без длинной масти
> - Когда торговля может вернуться к вам

### Виды интервенции на 1 уровне

#### 1. Простая интервенция (1♣ - 1♥)
**Требования:**
- 8-17hcp
- 5+ карт в масти (хорошая масть)
- Готовность к контре

**Примеры:**
```
✓ Хорошо: ♠Кх ♥ТКххх ♦Дхх ♣Вхх (11 очков, хорошая масть ♥)
✓ Хорошо: ♠ДВхххх ♥Кх ♦Тхх ♣Дх (10 очков, 5♠)
✗ Плохо: ♠Кхх ♥Дхххх ♦Кхх ♣хх (10 очков, но слабая масть)
```

#### 2. Интервенция 1NT
**Требования:**
- 15-18hcp (как открытие 1NT, но чуть больше)
- Держка в масти противника
- Сбалансированная рука

**Пример:**
```
Противник: 1♠
Вы: 1NT с рукой ♠Кхх ♥ТДх ♦Кхх ♣ДВхх (16 очков, держка ♠Кхх)
```

#### 3. Контра (такэаут дабл)
**Требования:**
- 12+hcp (обычно 12-17)
- Готовность к любому ответу партнера
- Обычно 4-4-4-1 или поддержка в непоказанных мастях

**Пример:**
```
Противник: 1♥
Вы: Контра с рукой ♠ТКхх ♥х ♦Дхх ♣КДххх (14 очков, 4♠4♦5♣1♥)
```

#### 4. Интервенция 2m (Микаэл кью-бид)
**Применение:** 1М от противника - 2m от вас
**Значение:** Минимум 5-5 в двух других мастях

**Пример:**
```
Противник: 1♠
Вы: 2♦ с рукой ♠х ♥ДВххх ♦Кх ♣ТДххх (11 очков, 5♥5♣)
```

### Виды интервенции на 2 уровне

#### 1. Простая интервенция на 2 уровне
**Требования:**
- 10-17hcp (чуть больше, чем на 1 уровне)
- Хорошая 5-карточная масть или 6+ карт
- Больший риск → нужна лучшая масть

**Примеры:**
```
Противник: 1♦
Вы: 2♥ с рукой ♠Кх ♥ТКДххх ♦хх ♣Вхх (11 очков, отличная масть)
```

#### 2. Слабый прыжок (Weak jump overcall)
**Применение:** 2-уровневая интервенция прыжком
**Требования:**
- 5-10hcp
- 6-карточная масть (хорошая)
- Цель: блокировка торговли противников

**Пример:**
```
Противник: 1♣
Вы: 2♠ с рукой ♠ДВхххх ♥хх ♦Кх ♣хх (7 очков, 6♠, блок)
```

#### 3. Интервенция 2NT (Unusual 2NT)
**Значение:** Минимум 5-5 в младших мастях (♣ и ♦)
**Требования:**
- 10-16hcp (или слабая блокирующая)
- 5-5+ в ♣ и ♦

**Пример:**
```
Противник: 1♠
Вы: 2NT с рукой ♠х ♥хх ♦ТДххх ♣КДххх (12 очков, 5♦5♣)
```

## Контра и реконтра

### Контра (Takeout Double)

> [!tip]- Контра для вывода (Takeout Double)
> **Цель:** Попросить партнера назвать свою лучшую масть
> 
> **Применение:** После открытия противника на низком уровне (1-го или 2-го)
> 
> **Условия:**
> - 12+hcp (обычно 12-17)
> - Поддержка во всех непоказанных мастях
> - Или очень сильная рука 18+

#### Ответы партнера на контру

| Сила руки | Действие |
|-----------|----------|
| **0-8hcp** | Минимальная заявка в лучшей масти |
| **9-11hcp** | Прыжковая заявка (инвит) |
| **12+hcp** | GF заявки или кюбид |

**Примеры:**

**Пример 1: Контра для вывода, слабый ответ**
```
Противник: 1♥
Партнер: Контра (♠ТКхх ♥х ♦Дхх ♣КДххх, 14 очков)
Противник: Пас
Вы: 1♠ (♠Дххх ♥хх ♦Кхх ♣Вххх, 6 очков, лучшая масть)
```

**Пример 2: Контра для вывода, сильный ответ**
```
Противник: 1♥
Партнер: Контра
Противник: Пас
Вы: 2♠ (♠ДВххх ♥хх ♦Тхх ♣Кхх, 10 очков, прыжок = инвит)
```

**Пример 3: Контра для вывода, GF ответ**
```
Противник: 1♥
Партнер: Контра
Противник: Пас
Вы: 2♥ (♠Кххх ♥хх ♦ТДхх ♣Дхх, 13 очков, кюбид = GF!)
```

### Контра штрафная (Penalty Double)

> [!warning]- Контра на штраф
> **Цель:** Получить штраф за контракт противников
> 
> **Применение:** На высоких уровнях (3+) или в особых ситуациях
> 
> **Условия:**
> - Вы уверены, что они не сыграют контракт
> - У вас много взяток в защите
> - Обычно 3+ козыря противника с фигурами

**Пример:**
```
Противник: 1♠
Вы: Пас
Партнер: 2♥
Противник: 4♠ (блок)
Вы: Контра (♠ТКДх ♥хх ♦Дххх ♣Вхх, у меня 4 взятки в ♠!)
```

### Реконтра (Redouble)

> [!tip]- Реконтра - что это?
> **Применение:** После того как противник законтровал открытие партнера
> 
> **Значение:** "У меня 10+ очков, мы сильнее!"

**Виды реконтры:**

#### 1. Силовая реконтра (10+hcp)
```
Партнер: 1♥
Противник: Контра
Вы: Реконтра (♠Кхх ♥Дх ♦ТДхх ♣Вххх, 12 очков)
Значение: У меня сила, решим что делать дальше
```

#### 2. SOS реконтра (слабая, просьба о помощи)
```
Применение: Когда партнер законтровал их контракт, но мы не можем его бить
Партнер: Контра (на 2♠)
Противник: Реконтра
Вы: 3♣ (SOS, давай уйдем!)
```

## Защитные заявки после интервенции

### После простой интервенции (1♥ - 1♠)

#### Открывающий (1♥ - 1♠ - ?)

| Заявка | Значение |
|--------|----------|
| **Пас** | Нет силы для дальнейших действий |
| **Контра** | 3-4 карты в ♠, штрафная контра |
| **1NT** | 18-21hcp, держка в ♠ |
| **2♣♦** | Натурально, 5+♥4+m, 11-15hcp |
| **2♥** | 6+♥, 11-15hcp |
| **2♠** | Кюбид, GF, без держки ♠ |
| **3♣♦** | GF, 5+♥5+m |
| **3♥** | Инвит, 6+♥, 16-17hcp |

#### Отвечающий (1♥ - 1♠ - ?)

| Заявка | Значение |
|--------|----------|
| **Пас** | Нет силы или желания торговаться |
| **Контра** | Негативная контра, 6+hcp, нет фита ♥, есть ♦ и/или ♣ |
| **1NT** | 6-10hcp, держка в ♠ |
| **2♣♦** | Натурально, 5+ карт, 6+hcp |
| **2♥** | Простая поддержка, 3+♥, 6-10hcp |
| **2♠** | Кюбид, GF, фит в ♥ |
| **3♥** | Прыжковая поддержка, 4+♥, 10-12hcp |
| **4♥** | Блокирующая, 5+♥, слабая рука |

### После контры (1♥ - Контра)

#### Открывающий (1♥ - Контра - ?)

| Заявка | Значение |
|--------|----------|
| **Пас** | Нейтрально, даем партнеру решать |
| **Реконтра** | 18+hcp, сильная рука |
| **1♠/2♣♦** | Натурально, как без контры |
| **2♥** | 6+♥, 11-14hcp |
| **3♥** | 6+♥, 15-17hcp |
| **4♥** | 6+♥, 18+hcp или блок |

#### Отвечающий (1♥ - Контра - ?)

| Заявка | Значение |
|--------|----------|
| **Пас** | Нет силы или желания помочь |
| **Реконтра** | 10+hcp, силовая |
| **1♠/2♣♦** | Натурально, 5+ карт, 6+hcp |
| **1NT** | 6-10hcp, нет 4М |
| **2♥** | Простая поддержка, 3+♥, до 9hcp |
| **2NT** | 10-12hcp, нет фита ♥ |
| **3♥** | Прыжковая поддержка, 4+♥, 10-12hcp |
| **4♥** | Блокирующая, 5+♥, слабая рука |

### Трансферы после контры (1NT - Контра)

> [!tip]- Система после контры на 1NT
> После 1NT - (Контра) играем трансферы:
> - **Пас** = к переконтровке или ловушка
> - **Реконтра** = 8+hcp, предложение играть
> - **2♣♦♥♠** = трансферы! (как обычно)

**Схема трансферов:**
- **2♣** = трансфер в ♦
- **2♦** = трансфер в ♥
- **2♥** = трансфер в ♠
- **2♠** = трансфер в ♣

## Лебензоль (Lebensohl)

> [!note]- Что такое Лебензоль?
> **Лебензоль** - конвенция для различения силы руки после интервенции противника на 2 уровне.
> 
> **Применение:** 1NT - (2♣/♦/♥/♠)
> 
> **Принцип:** Заявка 2NT = релей к 3♣, затем уточняем силу.

### Схема Лебензоля

**После 1NT - (2♥):**

| Заявка | Значение |
|--------|----------|
| **Контра** | Штрафная |
| **2♠** | Натурально, слабая рука (5+♠) |
| **2NT** | Лебензоль релей → 3♣, затем уточняем |
| **3♣♦** | GF, натурально |
| **3♥** | Кюбид, GF, держка в ♥ |
| **3♠** | GF, 4+♠ |
| **3NT** | G, держка в ♥ |

**После релея 2NT - 3♣:**

| Заявка | Значение |
|--------|----------|
| **Пас** | Слабая рука, 6+♣ |
| **3♦** | Слабая рука, 5+♦ |
| **3♥** | Стоппер-вопрос, нет держки ♥, ищем 3NT |
| **3♠** | Слабая рука, 4+♠ |
| **3NT** | G, нет держки ♥ |

## Блокирующие заявки

### Правило быстро-медленно

> [!tip]- Быстрый или медленный выход на высокий уровень?
> - **Быстро** (прямой прыжок) = слабая блокирующая рука
> - **Медленно** (через кюбиды) = сильная конструктивная рука

**Примеры:**

**Пример 1: Быстрый блок**
```
Партнер: 1♠
Противник: 2♥
Вы: 4♠ (♠Кхххх ♥х ♦хххх ♣хх, 5 очков, блокируем!)
Значение: Слабая рука с длинным фитом, блокируем торговлю
```

**Пример 2: Медленная конструктивная торговля**
```
Партнер: 1♠
Противник: 2♥
Вы: 3♥ (♠Кххх ♥Дх ♦ТКхх ♣Вхх, 14 очков, кюбид)
Партнер: 3♠
Вы: 4♠ (сильная рука, конструктивный подход к гейму)
```

### Закон позиции (Law of Total Tricks)

> [!tip]- Как использовать ЗОВ?
> **Правило:** Если у вас N козырей, можно безопасно торговаться до N-го уровня.
> 
> **Пример:**
> - У нас 9♠ → можем играть 3♠ (9 взяток)
> - У них 9♥ → могут играть 3♥ (9 взяток)
> - Итого: 18 козырей → безопасно торговаться до 9 уровня

**Применение:**
```
Партнер: 1♠
Противник: 2♥
Вы: 3♠ (♠Дххх ♥хх ♦Кххх ♣хх, 7 очков, но 4♠!)

Логика: У нас 9♠ (5+4), у них около 9♥ → итого 18 козырей
Значит 9-й уровень безопасен: 3♠ или 3♥ кто-то сыграет
```

## Жертвы (Sacrifices)

> [!warning]- Когда жертвовать контрактом?
> **Жертва** - осознанный выбор играть контракт, который, скорее всего, не сыграем, чтобы противники не сыграли свой гейм.
> 
> **Условия для жертвы:**
> - ✓ Противники близки к гейму (4М, 3NT)
> - ✓ Наша жертва дешевле их гейма
> - ✓ У нас длинный фит (9+ карт)
> - ✓ Учитываем уязвимость

### Таблица жертв (сколько можно положить)

| Их контракт | Мы не уязвимы | Мы уязвимы |
|-------------|---------------|------------|
| **4М (420)** | До -4 (-500) | До -2 (-500) |
| **3NT (400)** | До -4 (-500) | До -2 (-500) |
| **5m (400)** | До -4 (-500) | До -2 (-500) |
| **6М (980)** | До -7 (-1400) | До -4 (-1100) |

### Примеры жертв

**Пример 1: Выгодная жертва**
```
Противники торгуют гейм 4♠ (420 очков)
Мы не уязвимы, у нас 10♥
Мы: 5♥ (жертва)
Результат: -3 (не уязв) = -150 очков
Выигрыш: 420 - 150 = +270 очков!
```

**Пример 2: Невыгодная жертва**
```
Противники торгуют гейм 4♠ (420 очков)
Мы уязвимы, у нас 9♥
Мы: 5♥ (жертва)
Результат: -4 (уязв) = -1100 очков
Проигрыш: 1100 - 420 = -680 очков (катастрофа!)
```

## Конкурентная торговля: Чек-лист

✅ **Перед интервенцией:**
- [ ] У меня хорошая масть (5+ карт с фигурами)?
- [ ] У меня 8+hcp?
- [ ] Я готов к контре?

✅ **Перед контрой:**
- [ ] У меня 12+hcp?
- [ ] У меня поддержка во всех непоказанных мастях?
- [ ] Партнер может ответить в любой масти?

✅ **Перед блоком:**
- [ ] У нас длинный фит (9+ карт)?
- [ ] Применим Закон Общего Количества Взяток?
- [ ] Быстрый выход = слабая рука?

✅ **Перед жертвой:**
- [ ] Их гейм вероятен?
- [ ] Наша жертва выгоднее?
- [ ] Учел уязвимость?

✅ **Помни:**
- [ ] Контра на 1 уровне = для вывода (не штраф!)
- [ ] Реконтра = 10+hcp или SOS
- [ ] Быстро = слабо, медленно = сильно
- [ ] ЗОВ: N козырей → N-й уровень безопасен
