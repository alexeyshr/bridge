```
♠ДВ10х ♥Кхх ♦хх ♣Кхх (9 очков, 4♠)
♠ТКхххх ♥Дх ♦хх ♣Вхх (10 очков, 6♠ с 12hcp)
♠КДххх ♥хх ♦Тхх ♣Вхх (10 очков, 5♠)
```

## Заявки открывающего (1♥-1♠-?)

### 1NT = NF, 12-14hcp, BAL, 5♥332/5422 (м.б 4♠)

```
Рука открывающего: ♠Кхх ♥ТДххх ♦Дхх ♣Кх (13 очков)
```

**Развитие:**

- **Пас** = слабая рука без фита
- **2♣♦** = NF, натурально, 5+ карт
- **2♥** = NF, предпочтение, 6-9hcp, 2♥
- **2♠** = NF, 6+♠, к игре
- **2NT** = INV → 3NT, 10-11hcp
- **3♣♦** = GF, краткость в миноре, 5♠2♥
- **3♥** = GF, фит 3♥, шлемовые амбиции
- **3♠** = INV, 6+♠, 10-11hcp
- **3NT** = G

### 2♣ = ГАЗИЛЛИ! Искусственная заявка

**Значения:**

- Любая рука 16+hcp (все силы)
- ИЛИ 11-15hcp с 5♥4♣ (полунатуральная)

> [!note]- Что такое Газилли?
> **Газилли** - конвенция, позволяющая эффективно разделить руки открывающего по силе через систему **зон**:
>
> - **Зона 1 (11-15):** Масти на 2 уровне
> - **Зона 2 (15-17):** БК и масти на 3 уровне (инвит)
> - **Зона 3 (18+):** Прямые GF заявки
>
> После 1♥-1♠-2♣ отвечающий говорит **2♦** (waiting) = 8+hcp, и открывающий уточняет свою силу.

**Ответы отвечающего на 2♣:**

| Заявка    | Значение                            |
| --------- | ----------------------------------- |
| **2♦**    | 8+hcp, waiting (жду уточнения силы) |
| **2♥**    | <8hcp, 2-3♥ (к игре)                |
| **2♠**    | <8hcp, 5+♠ (к игре)                 |
| **2NT**   | <8hcp, 0-1♥, 4+4+ миноры            |
| **3♣/3♦** | <8hcp, 6+♣/♦                        |

**После 2♦ (waiting 8+hcp) открывающий уточняет:**

| Заявка    | Зона      | Значение              |
| --------- | --------- | --------------------- |
| **2♥**    | 1 (11-15) | 5♥, полунатуральная ♣ |
| **2♠**    | 3 (16+)   | 3+♠                   |
| **2NT**   | 2 (15-17) | 3♠5♥(32), инвит       |
| **3♣/3♦** | 3 (16+)   | 4+♣/♦, <3♠            |
| **3♥**    | 3 (18+)   | 6+♥, <3♠              |
| **3♠**    | 2 (18-19) | Ровно 4522            |
| **3NT**   | 2 (15-17) | 2♠5♥                  |

### 2♦ = NF, 11-14hcp, 5♥4+♦

```
Рука открывающего: ♠хх ♥ТКххх ♦КДхх ♣Кх (13 очков)
```

**Развитие:**

- **2♥** = NF, предпочтение, 6-9hcp
- **2♠** = NF, 6+♠, к игре
- **2NT** = INV → 3NT, 10-11hcp
- **3♣** = GF, задержка (поиск 3NT)
- **3♦** = INV, фит в ♦
- **3♥** = INV, фит 3♥
- **3♠** = GF, 6+♠
- **3NT** = G

### 2♥ = NF, 11-14hcp, 6+♥ или 5♥(332) минимум

```
Рука открывающего: ♠хх ♥ТКДххх ♦Кхх ♣Вх (12 очков)
```

**Развитие:**

- **Пас** = 6-9hcp, 2♥
- **2♠** = NF, 6+♠, 6-9hcp
- **2NT** = INV → 3NT, 10-11hcp
- **3♥** = INV, фит 3♥, 10-11hcp
- **3♠** = GF, 6+♠
- **3NT** = G
- **4♥** = G, фит 3♥

### 2♠ = NF, 11-14hcp, 4♠, обычно 5♥4♠

```
Рука открывающего: ♠ТКхх ♥ТДххх ♦Кх ♣хх (13 очков)
```

**Развитие:**

- **Пас** = 6-9hcp, фит 4♠
- **2NT** = INV → 3NT, 10-11hcp
- **3♣♦** = GF, краткость в миноре с фитом ♠
- **3♥** = GF, 6+♥ (редко), предпочтение ♥ над ♠
- **3♠** = INV, фит 4♠, 10-11hcp
- **4♠** = G, фит 4♠

### 2NT = INV, 18-19hcp, БК 2♠5♥ или 3♠5♥(32)

```
Рука открывающего: ♠Кхх ♥ТКДхх ♦ТДх ♣Кх (19 очков)
```

**Развитие:**

- **3♣♦** = натурально, 5+♠4+m
- **3♥** = фит 3♥ (если есть)
- **3♠** = 6+♠, нет фита ♥
- **3NT** = G
- **4♠** = фит 4♠

### 3♣ = INV, 15-17hcp, 5♥4+♣

```
Рука открывающего: ♠хх ♥ТКххх ♦Кх ♣ТДхх (16 очков)
```

**Развитие:**

- Все GF
- **3♦** = задержка (поиск 3NT)
- **3♥** = фит 3♥
- **3♠** = 6+♠
- **3NT** = G
- **4♣** = фит в ♣, шлем

### 3♦ = INV, 15-17hcp, 5♥4+♦

```
Рука открывающего: ♠хх ♥ТКххх ♦ТДхх ♣Кх (16 очков)
```

**Развитие:**

- Все GF (аналогично 3♣)

### 3♥ = INV, 15-17hcp, 6+♥

```
Рука открывающего: ♠хх ♥ТКДВхх ♦Тхх ♣Кх (16 очков)
```

**Развитие:**

- **3♠** = кюбид (согласован ♥)
- **3NT** = G, без фита ♥
- **4♣♦** = кюбид (согласован ♥)
- **4♥** = G, фит 3♥

### 3♠ = INV, 15-17hcp, 4♠, обычно 5♥4♠

```
Рука открывающего: ♠ТКхх ♥ТДВхх ♦Тх ♣Кх (17 очков)
```

**Развитие:**

- **3NT** = G, без фита ♠
- **4♣♦** = кюбид (согласован ♠)
- **4♥** = кюбид (согласован ♠)
- **4♠** = G, фит 4♠

### 3NT = G, 18-21hcp, остальные руки

```
Рука открывающего: ♠Кх ♥ТКДВххх ♦Тх ♣Кх (18 очков)
```

### 4♥ = GF, 18+hcp, Сплинтер, краткость ♥, 4♠ (редко)

## Примеры торговли с Газилли

**Пример 1: Зона 1 (11-15hcp)**

```
Север: ♠Кх ♥ТКххх ♦хх ♣КДхх (13 очков)
Юг: ♠ДВ10хх ♥Дх ♦Тхх ♣Вхх (11 очков)

Торговля:
Север: 1♥
Юг: 1♠
Север: 2♣ (Газилли)
Юг: 2♦ (waiting, 8+hcp)
Север: 2♥ (зона 1, 11-15hcp, 5♥4♣)
Юг: 2♠ (предпочтение пике)
Север: Пас
```

**Пример 2: Зона 3 (16+hcp) с фитом**

```
Север: ♠ТКхх ♥ТДВхх ♦Тх ♣Кх (18 очков)
Юг: ♠ДВхх ♥Кх ♦Дхх ♣Вххх (10 очков)

Торговля:
Север: 1♥
Юг: 1♠
Север: 2♣ (Газилли)
Юг: 2♦ (waiting, 8+hcp)
Север: 2♠ (зона 3, 16+hcp, 3+♠!)
Юг: 3♠ (фит, хорошая рука)
Север: 4♠ (гейм)
```

**Пример 3: Слабая рука отвечающего**

```
Север: ♠Кх ♥ТКххх ♦хх ♣ТДхх (15 очков)
Юг: ♠ДВххх ♥хх ♦Вххх ♣хх (5 очков)

Торговля:
Север: 1♥
Юг: 1♠
Север: 2♣ (Газилли)
Юг: 2♠ (<8hcp, 5+♠, к игре)
Север: Пас
```
