> F1, 10+hcp, 4+♦, <font color="#d83931">нет 4М</font> (инвертированные миноры)

## Инвертированные миноры

**Суть конвенции:** После 1♦-2♦ отвечающий показывает силу 10+ очков и фит в бубне (минимум 4 карты). Это форсирующая заявка на один круг.

**Цель:** Эффективно исследовать возможность игры в 3NT или 5♦, показывая держки в мастях.

## Примеры рук отвечающего
```
♠Кхх ♥Дхх ♦ТВхх ♣хх (10 очков, 4♦, нет 4М)
♠Вх ♥Кхх ♦ДВххх ♣Дхх (11 очков, 5♦, нет 4М)
♠Кх ♥Дхх ♦ТКххх ♣Вхх (13 очков, 5♦, GF сила)
```

## Заявки открывающего (1♦-2♦-?)

### 2♥ = Ценности в ♥, <font color="#d83931">нет держки в ♠</font>
```
Рука открывающего: ♠хх ♥ТКх ♦ТДххх ♣Кхх (14 очков)
```
**Значение:** У меня есть фигуры в черве, но нет держки в пике. Не могу гарантировать безопасность 3NT.

**Развитие отвечающего:**
- **2♠** = держка в ♠, поиск 3NT
- **2NT** = держка в ♠ и ♣, предлагаю 3NT
- **3♣** = натурально, 5+♣4♦, держка в ♣
- **3♦** = нет держки в ♠, играем в миноре
- **3♥** = фит в ♥ (редко)
- **3NT** = держка в ♠, к игре

### 2♠ = Ценности в ♠, <font color="#d83931">нет держки в ♥</font>
```
Рука открывающего: ♠ТКх ♥хх ♦ТДххх ♣Кхх (14 очков)
```
**Значение:** У меня есть фигуры в пике, но нет держки в черве.

**Развитие отвечающего:**
- **2NT** = держка в ♥ и ♣, предлагаю 3NT
- **3♣** = натурально, 5+♣4♦, держка в ♣
- **3♦** = нет держки в ♥, играем в миноре
- **3♥** = держка в ♥, поиск 3NT
- **3NT** = держка в ♥, к игре

### 2NT = Держки в обоих мажорах, не минимум (13+hcp)
```
Рука открывающего: ♠ТКх ♥Дхх ♦ТДххх ♣Кх (15 очков)
```
**Значение:** У меня держки и в ♠, и в ♥. 3NT безопасен.

**Развитие отвечающего:**
- **3♣** = держка в ♣, предлагаю 3NT
- **3♦** = не минимум, но нет держки ♣, осторожность
- **3NT** = G, держка в ♣ или хорошая рука
- **4♦** = фит, хорошая рука, шлемовые амбиции

### 3♣ = Держка в ♣, м.б держки в мажорах
```
Рука открывающего: ♠Кхх ♥Дх ♦ТВххх ♣ТДх (14 очков)
```
**Значение:** У меня держка в трефе, возможно и в мажорах.

**Развитие отвечающего:**
- **3♦** = нет всех держек, играем в миноре
- **3♥/3♠** = держка в масти, поиск 3NT
- **3NT** = есть держки в мажорах
- **4♦** = фит, шлемовые амбиции

### 3♦ = НФ, минимум (11-13hcp) ИЛИ нет держек в мажорах
```
Рука открывающего: ♠хх ♥хх ♦ТКДххх ♣Кхх (11 очков)
```
**Значение:** У меня либо слабая рука, либо нет держек для 3NT.

**Развитие отвечающего:**
- **Пас** = слабая рука (10-11hcp), играем 3♦
- **3♥/3♠** = GF, держка в масти, еще один шанс на 3NT
- **3NT** = GF, хорошая рука, есть все держки
- **4♦** = GF, фит, нет всех держек или шлем
- **5♦** = G, фит, нет шлемовых амбиций

### 3♥/3♠ = GF, Сплинтер, краткость в М, 6+♦
```
Рука открывающего (3♥): ♠Кх ♥х ♦ТКВххх ♣ТДх (16 очков)
```
**Значение:** У меня синглет/ренонс в масти и хорошая рука с длинной бубной.

**Развитие отвечающего:**
- **3NT** = нет шлемовых амбиций
- **4♣** = кюбид (согласована ♦)
- **4♦** = минимум из своего диапазона
- **4OM** = кюбид (согласована ♦)
- **4NT** = Блэквуд

### 3NT = G, есть все держки или хорошая рука
```
Рука открывающего: ♠ТКх ♥Дхх ♦ТКххх ♣Кх (16 очков)
```

### 4♦ = GF, фит 4+♦, хорошая рука, шлемовые амбиции
```
Рука открывающего: ♠Кх ♥хх ♦ТКВххх ♣ТКх (17 очков)
```

## Примеры торговли

**Пример 1: Поиск 3NT**
```
Север: ♠хх ♥ТКх ♦ТДххх ♣Кхх (14 очков)
Юг: ♠ТКх ♥Дхх ♦КВхх ♣Дхх (12 очков)

Торговля:
Север: 1♦
Юг: 2♦ (инвертированные, 10+hcp, фит ♦)
Север: 2♥ (ценности в ♥, нет держки ♠)
Юг: 3NT (держка в ♠, к игре)
```

**Пример 2: Игра в миноре**
```
Север: ♠хх ♥хх ♦ТКДххх ♣Кхх (11 очков)
Юг: ♠Кхх ♥Дхх ♦ТВхх ♣хх (10 очков)

Торговля:
Север: 1♦
Юг: 2♦ (инвертированные, 10hcp, фит ♦)
Север: 3♦ (минимум, нет держек)
Юг: Пас (слабая рука, играем 3♦)
```

**Пример 3: Шлемовые амбиции**
```
Север: ♠Кх ♥х ♦ТКВххх ♣ТДх (16 очков)
Юг: ♠Тхх ♥Дхх ♦ТКхх ♣Кхх (14 очков)

Торговля:
Север: 1♦
Юг: 2♦ (инвертированные, 14hcp, фит ♦)
Север: 3♥ (сплинтер, краткость ♥)
Юг: 4♦ (хорошая рука, но нет идеальных контролей)
Север: 4NT (Блэквуд)
Юг: 5♦ (0 или 3 туза)
Север: 6♦ (шлем)
```
