## УСЛОВНЫЕ ОБОЗНАЧЕНИЯ

### Масти и заявки:

- **♣** = Трефа (Clubs)
- **♦** = Бубна (Diamonds)
- **♥** = Черва (Hearts)
- **♠** = Пика (Spades)
- **БК** или **NT** = Без козыря (No Trump)
- **М** = Мажор (♥ или ♠)
- **м** = Минор (♣ или ♦)
- **ОМ** = Другой мажор (Other Major)

### Уровни форсирования:

- **НФ** = Не форсирует (можно пасовать)
- **Ф1** = Форсирует на одну заявку
- **ФГ** = Форсирует до гейма (ниже 3БК пасовать нельзя)
- **ИНВ** = Приглашает к гейму (можно пасовать с минимумом)

### Сокращения:

- **2wcb** = Двойной чек-бек (Double Checkback)
- **СМ** = Совместная масть (Combined Suit)
- **9СМ** = 9 карт в совместной масти
- **ТКД** = Туз, Король, Дама
- **ТКВ** = Туз, Король, Валет

## СВОД ПРАВИЛ

### Основные принципы:

1. **При розыгрыше полной масти** всегда играем короткой фигурой (к короткой фигуре)

2. **Приоритет мажоров:** Если есть фит в мажоре – стремимся найти гейм 4 в мажоре. Если есть фит в миноре – изучаем 3БК, проверяя держки в мастях

3. **Экономия заявок:** Если заявка партнера не форсирует, задаём себе вопрос – есть ли у меня в руке что-то важное, о чем партнёр ещё не знает. Если нет – пасуем

4. **В форсгейме не прыгаем:** Прыжок в гейм – это минимум и отсутствие интереса

5. **Правило 44/55:** При 44 называем всегда ближайшую масть, при 55 – старшую

6. **Преференции:** Играем преференцию мажоров на открытие 1♣/1♦ и преференцию силы на 1♥/1♠

7. **Контра "фит":** Играем контру и реконтру «фит» после ответа 1 в мажор на открытие

## ДВОЙНОЙ ЧЕК-БЕК (2wcb)

### Применение:

**Позиция:** 1x-1M-1NT (где x = ♣ или ♦, M = мажор)

### Цель:

Разделить слабые и сильные руки отвечающего после 1NT открывающего.

### Развитие после 1♣-1♥-1NT:

**Отвечающий говорит:**

- **2♣** = 2wcb, просьба сказать 2♦
- **2♦** = 2wcb, любой гейм-форсинг

**Если открывающий говорит 2♦ (слабый с ♦ или инвит):**

| Заявка  | Значение                                         |
| ------- | ------------------------------------------------ |
| **2♥**  | Играть (слабая рука)                             |
| **2♠**  | ФГ, 4♠5♥, проблема с бубной                      |
| **2БК** | Инвит в 3БК                                      |
| **3♣**  | Играть                                           |
| **3♥**  | Резкая рука, хорошая 6♥, инвит в 4♥ на контролях |
| **3БК** | Играть                                           |

**Если открывающий говорит 2♦ (гейм-форсинг):**

- Все заявки ФГ, изучение лучшего контракта

### Примеры:

**Пример 1:**

```
1♣ - 1♥ - 1NT - 2♣ (2wcb)
2♦ - 2♥ (играть)
```

**Рука отвечающего:** ♠Кхх ♥ДВ10хх ♦хх ♣ххх (6 очков, слабая рука с фитом)

**Пример 2:**

```
1♣ - 1♥ - 1NT - 2♣ (2wcb)
2♦ - 3♥ (инвит в 4♥ на контролях)
```

**Рука отвечающего:** ♠Тх ♥ТКДххх ♦Кх ♣хх (13 очков, хорошая черва, контроли)

## РЕЛЕ С ШЕСТИКАРТОМ

### Применение:

**Позиции:**

- 1♣-1♥-2♣
- 1♣-1♠-2♣
- 1♦-1♥-2♦
- 1♦-1♠-2♦
- 1♥-1♠-2♥

### Цель:

Эффективно описать руки с шестикартом в масти открытия.

### Развитие после 1♣-1♥-2♣:

**Отвечающий говорит:**

| Заявка  | Значение                                  |
| ------- | ----------------------------------------- |
| **2♦**  | Ф1, квази-натурально, не хочу занимать БК |
| **2♥**  | К игре (слабая рука)                      |
| **2♠**  | Ф1, квази-натурально, не хочу занимать БК |
| **2БК** | ФГ, Реле                                  |

**После 2БК (реле):**

| Заявка  | Значение      |
| ------- | ------------- |
| **3♣**  | 3 карты в ♥   |
| **3♦**  | Фит в ♣, 9СМ  |
| **3♥**  | Фит в ♥, 9СМ  |
| **3♠**  | Синглет в ♠   |
| **3БК** | Нет синглетов |

### Примеры:

**Пример 1:**

```
1♣ - 1♥ - 2♣ - 2БК (реле)
3♣ (3 карты в ♥) - 3♦ (фит в ♣, 9СМ)
3БК (6322) или 4♥ (краткости)
```

**Пример 2:**

```
1♣ - 1♠ - 2♣ - 2БК (реле)
3♣ (3 карты в ♠) - 3♦ (фит в ♣, 9СМ)
3БК (6322) или 3♥/4♠ (краткости)
```

## ДРУРИ

### Применение:

**Позиция:** пас-пас-1M (открытие на 3-4 руке)

### Цель:

Показать максимум паса с фитом в мажоре партнера.

### Критерии применения:

- ✓ Открытие на 3-4 руке
- ✓ У нас максимум паса (6-10 очков)
- ✓ Есть 3-4 карты в мажоре партнера

### Развитие после пас-пас-1♥-2♣ (Друри):

| Заявка    | Значение                    |
| --------- | --------------------------- |
| **2♦**    | Нет гейма, пасуй            |
| **2♥**    | Ф1, ИНВ, 4+♥                |
| **2♠**    | Ф1, ИНВ, 4+♠                |
| **2БК**   | ИНВ, 13-14, 5332            |
| **3♣**    | Ф1, ИНВ, 4+♣                |
| **3♥**    | ИНВ, 6+♥                    |
| **3♠**    | ФГ, Сплинтер, краткость ♠   |
| **3БК**   | НФ, на выбор, 5332          |
| **4♣/4♦** | ФГ, Сплинтер, краткость ♣/♦ |

### ⚠️ ВАЖНО:

**Если открытие на 1-2 руке, Друри НЕ применяется!**

### Примеры:

**Пример 1:**

```
пас - пас - 1♥ - 2♣ (Друри)
2♦ (нет гейма) - пас
```

**Рука отвечающего:** ♠ххх ♥КДхх ♦ххх ♣ххх (7 очков, максимум паса, 3 карты в черве)

**Пример 2:**

```
пас - пас - 1♥ - 2♣ (Друри)
2♥ (ИНВ, 4+♥) - 4♥ (принимаем инвит)
```

**Рука отвечающего:** ♠Кхх ♥ДВ10х ♦хх ♣ххх (9 очков, максимум паса, хороший фит)

## ВИСТ (ЗАЩИТА)

### Основные принципы:

- **Вист прямой, сигнал обратный**
- **На сносах московский Лавинталь** без цикла
- **Возвратный Лавинталь**
- **Следим за сносом фосок у партнера** - их порядком

### Атаки:

1. **4-й из-под фигуры**
2. **2-й с фосок**
3. **Старшей из дублета**
4. **Старшей из фигур подряд**

### На БК контрактах:

- **"10/9 = 0/2"** - ход с 10 означает либо 0, либо 2 фигуры старше
- **Ход с валета** - нет фигуры старше
- **Из обрезных ТК в козырных контрактах** - ходим с короля, потом туз

### Четность:

- **На туза** - поощрение/четность в козырных, если по столу не нужно иное
- **На ход королем** - четность
- **Четность сигналим изначальную**
- **При возвратной четности** 5 карт сигналим как 3

### Четность с 4 картами:

- **При даме на столе** сигналим четное количество только если есть возможность убить (до 5 уровня)
- **С 4-мя** сигналим нечет (исключение - высокий уровень контракта)

### Приоритеты сигналов:

1. **Козырное эхо** (если возможно убить)
2. **Поощрение/запрет**
3. **Четность**
4. **Лавинталь**

### Таблица сигналов:

| Ситуация      | Сигнал           | Значение        |
| ------------- | ---------------- | --------------- |
| Ход тузом     | Старшая фоска    | Поощрение       |
| Ход тузом     | Младшая фоска    | Запрет          |
| Ход королем   | Четность         | Количество карт |
| Снос          | Старшая четная   | Старшая масть   |
| Снос          | Старшая нечетная | Младшая масть   |
| Козыри (3 шт) | Старшая-младшая  | Просьба убить   |

## ГЛОССАРИЙ КОНВЕНЦИЙ

### Стейман (Stayman):

**Применение:** 1БК-2♣  
**Цель:** Найти фит в мажоре  
**Ответы:** 2♦ = нет четверок, 2♥ = 4♥, 2♠ = 4♠

### Трансферы (Transfers):

**Применение:** 1БК-2♦/2♥/2♠  
**Цель:** Безопасно показать длинную масть  
**Развитие:** Открывающий принимает трансфер заявкой в масть

### Газилли (Gazilli):

**Применение:** 1M-1NT-2♣ или 1M-1OM-2♣  
**Цель:** Эффективно описать руки открывающего  
**Развитие:** Сложная система с зонами силы

### Друри (Drury):

**Применение:** пас-пас-1M-2♣  
**Цель:** Показать максимум паса с фитом  
**Критерии:** Только на 3-4 руке

### Двойной чек-бек (2wcb):

**Применение:** 1x-1M-1NT-2♣  
**Цель:** Разделить слабые и сильные руки  
**Развитие:** Просьба сказать 2♦ или гейм-форсинг

### Реле с шестикартом:

**Применение:** 1x-1M-2x-2БК  
**Цель:** Описать руки с шестикартом  
**Развитие:** Показ фитов и синглетов

### Паппет Стейман:

**Применение:** 2БК-3♣  
**Цель:** Найти фит в мажоре после 2БК  
**Ответы:** 3♦ = одна или две четверки, 3♥/3♠ = конкретная четверка

### Лебензоль (Lebensohl):

**Применение:** 1БК-(2M)-?  
**Цель:** Разделить слабые и сильные руки  
**Принцип:** Чем ниже заявка, тем слабее рука

### Multy-Landy:

**Применение:** (1БК)-2♣/2♦  
**Цель:** Показать мажоры после входа на 1БК  
**Развитие:** Выбор мажора партнером

### Michaels:

**Применение:** (1x)-2M или (1x)-2БК  
**Цель:** Показать двуцветную руку  
**Развитие:** Вопрос о второй масти

### Leaping Michaels:

**Применение:** (3M)-4м  
**Цель:** Показать двуцветную руку после блока  
**Развитие:** Выбор масти для шлема

## ЧАСТЫЕ ОШИБКИ

### 1. Применение Друри на 1-2 руке

**ОШИБКА:** Использовать Друри после открытия на 1-2 руке  
**ПРАВИЛЬНО:** Друри только после паса-паса-1M (открытие на 3-4 руке)

### 2. Прыжок в гейм в форсгейме

**ОШИБКА:** Прыгать в гейм, когда партнер форсирует  
**ПРАВИЛЬНО:** В ФГ никуда не прыгаем, изучаем контракт

### 3. Забыли про реле с шестикартом

**ОШИБКА:** Не использовать реле после 1x-1M-2x  
**ПРАВИЛЬНО:** После 1x-1M-2x всегда помним про реле

### 4. Неправильное применение трансферов в интервенции

**ОШИБКА:** Играть натурально после контры оппонентов  
**ПРАВИЛЬНО:** Начиная с 1БК до 2M-1 играем трансферы

### 5. Блэквуд без согласованной масти

**ОШИБКА:** Использовать Блэквуд, когда масть не согласована  
**ПРАВИЛЬНО:** Блэквуд только при явно согласованной масти

### 6. Неправильное применение Газилли

**ОШИБКА:** Забыть про зоны силы в Газилли  
**ПРАВИЛЬНО:** Помнить про зоны: 11-15, 15-17, 18+

### 7. Ошибки в висте

**ОШИБКА:** Сигналить не то, что нужно партнеру  
**ПРАВИЛЬНО:** Сигналим то, что в данный момент нужно партнеру

### 8. Неправильное применение Стеймана

**ОШИБКА:** Использовать Стейман с мажором 5+ карт  
**ПРАВИЛЬНО:** С мажором 5+ карт используем трансфер
